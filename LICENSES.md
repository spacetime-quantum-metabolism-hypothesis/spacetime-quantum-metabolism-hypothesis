## Third-party / upstream data

The following datasets are referenced or (when redistributed) retained under
their upstream licenses. If a redistributed copy is added under `data/` in the
future, the upstream license below applies to that copy and must be preserved
verbatim alongside the data.

| Dataset | Upstream | License | Citation |
|---------|----------|---------|----------|
| SPARC galaxy rotation curves | http://astroweb.cwru.edu/SPARC/ | **CC-BY-4.0** | Lelli, McGaugh, Schombert 2016 (AJ 152, 157) |
| DESI BAO (DR1 / DR2) | https://github.com/CobayaSampler/bao_data | DESI Collaboration data release terms (CC-BY-4.0 effective for redistribution via Cobaya) | DESI Collaboration 2024 / 2025 |
| Pantheon+ SN Ia | https://github.com/PantheonPlusSH0ES/DataRelease | **CC-BY-4.0** | Scolnic et al. 2022 |
| DES-Y5 SN Ia | https://github.com/CobayaSampler/sn_data (`DESY5/`) | DES Collaboration release terms (CC-BY-4.0 redistribution) | DES Collaboration 2024 |
| Planck 2018 compressed CMB | ESA / Planck Legacy Archive | Planck data release license (CC-BY-4.0 effective) | Planck Collaboration 2020 |
| RSD `fσ_8` compilation | Various; see `paper/references.bib` | Per-source (mostly journal terms; statistics are facts) | See bib entries |

If any redistributed file's upstream license is more restrictive than CC-BY-4.0,
that file's license overrides the table entry above. Do not relicense upstream
data.

## Note

As of this commit, the repository does **not** ship a top-level `data/`
directory; cosmological data are pulled at run time from upstream sources
listed above (mostly via `bao_data` / `sn_data` Cobaya mirrors). The SPARC
entry is included for completeness because the project references SPARC
galaxy rotation curves; if SPARC files are later vendored under `data/sparc/`,
they remain under SPARC's CC-BY-4.0 with the Lelli et al. 2016 citation.
