> 원칙: 각 이론별 8인팀 상호토의. 방향만 주고, 경로(지도)는 주지 않는다. 수단과 방법 가리지 않음.

## L15에서 남긴 핵심 질문

L15는 "어떤 수식이 데이터와 잘 맞는가"에 답했다.

L16이 답해야 할 질문:

> **"A1+A2 공리로부터 이 수식이 왜 나와야 하는가?"**
> **"B 파라미터의 크기는 Planck 상수, H0, 물질 밀도로 어떻게 유도되는가?"**

이 질문에 답하지 못하면 L15는 "잘 맞는 경험식 발굴"에 그친다.

## 8인팀 검토 원칙

### "방향만 주고 지도는 주지 않는다 — 수단과 방법 가리지 않음"

각 검토자에게 제공하는 것:
1. A1+A2 공리 (물질 ↔ 시공간 양자 소멸/생성)
2. 목표 수식 (e.g., AX3: omega_de = OL0*(1+A*(1-cos(B*theta))))
3. 물리적 맥락 (우주론적 스케일, DESI 관측 범위)

제공하지 않는 것:
- 유도 경로 (어떤 물리 메커니즘을 통해 유도할지)
- 중간 단계 힌트

### 운영 방식: 상호토의 기반

검토자 8인은 **서로 토의하며** 유도 경로를 찾는다.

- 각자 자기 분야 관점에서 초안 제시
- 다른 검토자의 시도를 보고 비판/보완 허용
- 수단과 방법 가리지 않음: 수치 실험, 유추, 기존 문헌 참조, 역방향 추론 등 모두 허용
- 단, 유도 경로 힌트를 외부에서 주입받는 것은 금지 (검토팀 내부 토론만 허용)

**수렴 판정**: 8인 토의 후 도달한 유도 경로에 팀 과반수(5명 이상) 동의 → **이론 도출 성공**. 합의 실패 → **공리-수식 연결 실패 (경험식으로만 취급)**.

### 검토자 구성 (8인)

역할을 사전에 지정하지 않는다.

검토자 8인은 각자 자신의 배경과 직관에 따라 자유롭게 접근. 토의 중 자연스럽게 역할이 형성된다.

- 누군가 수식을 수치로 먼저 분석할 수 있고
- 누군가 물리적 유추로 시작할 수 있고
- 누군가 기존 문헌에서 유사 구조를 찾을 수 있고
- 누군가 논리 오류를 먼저 지적할 수 있다

사전 역할 배정 금지. 토론에서 자연 발생한 분업만 인정.

### 토의 흐름 (권고)

1. **라운드 1**: 8인 각자 자유롭게 초안 제시. 어떤 접근이든 허용.
2. **라운드 2**: 상호 비판 및 교차 검토. 유망한 경로에 집중.
3. **라운드 3**: 남은 경로의 논리 오류 집중 탐색. 버티는 경로만 남김.
4. **최종**: 남은 경로에 5인 이상 동의 → 성공.

## 대상 수식 (L15 생존자, 우선순위 순)

### T1: AX3 (F-C) — 최우선

```
omega_de = OL0 * (1 + A * (1 - cos(B * theta)))
theta = (Om*(1+z)^3 - Om) / OL0  =  Delta_rho_m / rho_de0
```

**L16 핵심 질문:**

1. cos(theta) 형태는 어디서 오는가?
   - 공리: 시공간 양자 밀도 n_sq(z) = n_sq0 * f(물질)
   - 힌트 없음. 검토자 스스로 "왜 진동/위상 구조인가" 탐색.

2. theta = Delta_rho_m / rho_de0의 물리적 의미는?
   - 물질 밀도 변화량을 현재 DE 밀도로 정규화한 값.
   - A1에서 "물질이 시공간 양자를 소멸"시킬 때 위상 변화로 해석 가능한가?

3. B ≈ 2.1 (Stage-3 MAP)의 기원은?
   - 현재 미해석. Planck 밀도 / H0² 스케일과 관계 있는가?
   - 검토자: B의 차원 분석부터 시작.

4. A ≈ 0.12의 기원은?
   - 진폭 계수. 시공간 양자 소멸 효율과 관계 있는가?

---

### T2: HB3 (F-L) — 2순위 (MCMC 최고 신뢰도)

```
omega_de = OL0 * (1 + A * (x_m^2/(B^2+x_m^2)^2 - x_m0^2/(B^2+x_m0^2)^2))
x_m = Om*(1+z)^3
```

**L16 핵심 질문:**

1. Lorentzian 제곱 구조 x²/(B²+x²)² 는 어디서 오는가?
   - 양자 공명(resonance) 형태. 산란 단면적의 Breit-Wigner형.
   - A1의 "소멸" 과정에서 공명 구조가 자연스럽게 나오는가?

2. 왜 x/(B²+x²) (1차 Lorentzian)이 아닌 x²/(B²+x²)² (2차)인가?
   - 데이터에서 2차가 1차보다 잘 맞음 (HB3 > Q3).
   - 이론적 근거: 2체 충돌 단면적? 2차 소멸 과정?

3. B ≈ 1.1 (Om과 비슷한 스케일)의 의미는?
   - B ≈ Om → 현재 물질 밀도와 같은 스케일에서 공명.
   - 우연인가, A1에서 예측 가능한가?

---

### T3: TF3 (F-G) — MAP chi² 최우수 (MCMC 신뢰도 낮음)

```
omega_de = OL0 * (1 + A * (E-1) * exp(-B * (E-1)^3))
```

**L16 핵심 질문:**

1. (E-1) × exp(-B*(E-1)³) 구조의 기원은?
   - (E-1): 허블 팽창에서의 "초과 에너지 밀도" 척도.
   - exp(-B*x³): 매우 강한 고에너지 억제.
   - A1에서 "고밀도 물질이 시공간 양자를 급격히 소멸"시킨다면 왜 3제곱인가?

2. B ≈ 4.5~5.2의 기원은?
   - 매우 큰 값. exp(-5*(E-1)³)은 z>1에서 거의 0.
   - 플랑크 스케일 억제인가, 열역학적 임계점인가?

3. MCMC acc=0.009: 파라미터 공간이 매우 좁고 뾰족함.
   - 이론에서 B가 강하게 고정된다면 이를 설명 가능한가?

---

### T4: EE2 (F-CL) — CMB 완벽 정합

```
omega_de = OL0 * (1 + A * (1 - cos(B * ln(E(z)))))
```

**L16 핵심 질문:**

1. ln(E(z))는 우주론적 시간의 로그 스케일.
   - ln(E) ≈ H·t (slow-roll 근사). 즉 우주 나이의 함수.
   - A1에서 "시간에 따른 위상 누적"으로 해석 가능한가?

2. AX3(물질 밀도 의존)과 EE2(E 의존)의 차이:
   - 물리: AX3는 물질 양에 반응, EE2는 팽창률에 반응.
   - A1+A2에서 어느 쪽이 더 자연스러운가?

3. B ≈ 9.1 (Stage-3), wa ≈ -0.36 (거의 ΛCDM).
   - CMB chi²=0.71이지만 wa가 거의 0 → DE 진화 없음 수준.
   - 이 수식이 A1+A2에서 나온다면 왜 진화가 거의 없어야 하는가?

---

### T5: ST1 (F-W) — 주의 대상 (검토 후 폐기 가능)

```
omega_de = OL0 * exp(-A*((1+z)^3-1)) * (1+B*(1+z)^3) / (1+B)
```

**L16 핵심 질문:**

1. 이 수식의 고z 거동: z→∞에서 omega_de→0 (지수 감쇠).
   - A1과 일치. 하지만 w0=-0.535로 현재 DE가 매우 약함.
   - A1+A2에서 현재 우주의 DE 강도를 예측하면 w0는 어떤 값이 나오는가?

2. BAO chi²=27.49 > ΛCDM(24.40): BAO를 희생.
   - 이론에서 이런 거래(trade-off)가 정당화되는가?

3. 검토 후 폐기 기준: A1+A2와 연결 경로가 없거나 BAO 악화가 이론적으로 설명 불가 → L16에서 탈락.

## L16 실행 절차 (설계)

### Phase 1: 이론 도출 시도 (8인 독립)

각 검토자에게 제공:
```
[L16 검토 패킷]
공리 A1: "물질은 시공간 양자를 소멸, 빈 공간은 생성"
공리 A2: "양자-고전 경계는 A1에서 유도"
목표 수식: [T1/T2/T3/T4 중 배정된 하나]
DESI best-fit: w0=?, wa=?, Om=?, B=?
질문: 왜 이 수식인가? A1+A2에서 어떻게 유도하는가?
제한: 경로 힌트 없음. 독립 시도 후 결과 보고.
```

### Phase 2: 수렴 판정

8명의 유도 결과 비교:
- 동일/유사 유도 경로 5명 이상 → **이론 도출 성공**
- 3-4명 일치 → **부분 성공 (추가 검토)**
- 2명 이하 → **실패 (경험식으로 분류)**

### Phase 3: 파라미터 예측 검증

성공한 수식에 대해:
- 이론에서 B, A를 Planck 단위로 계산
- L15 MAP 값과 비교
- 일치 → **진짜 이론 예측 성공**

### Phase 4: 문서화

- base.l16.result.md: 각 이론의 유도 성공/실패 기록
- 성공한 이론만 논문 "이론 섹션"에 포함

## 왜 "지도를 주지 않는가"

8인팀에게 유도 경로를 주면:
- 확증 편향: 주어진 경로를 따라가며 "맞다"고 결론
- 실제로 유도되는지 검증 불가

방향만 주면:
- 각자가 독립적으로 다른 경로 시도
- 5명 이상 같은 경로에 수렴 → 경로의 자연스러움 증명
- 수렴 실패 → 수식이 이론에서 자연스럽게 나오지 않는다는 증거

이것이 **이론의 힘**을 검증하는 유일한 방법.
