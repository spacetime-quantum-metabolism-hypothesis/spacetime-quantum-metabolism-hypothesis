# Figure 05: SQMH 시공간 양자 유입 벡터장 (2D)

## 내용
지구 질량 주위 시공간 양자 유입의 2D 벡터장. 중력 = 시공간 양자의 물질로의 유입.

## 주요 결과
- 모든 화살표가 중심(질량)을 향함 -> 반경방향 유입
- 색상: log10(속도) -- 중심에 가까울수록 빠름 (1/r^2 의존성)
- 구대칭: v(r) = sigma*M/(4*pi*r^2) = g(r)*t_P

## 검증 상태
- **base.md III**: 시공간 양자 유입 = 중력, 시각적 표현

## 데이터 출처
- M_Earth, R_Earth -- IAU 2015 공칭값
- sigma = 4*pi*G*t_P = 4.522e-53 m^3/kg/s (Issue #28 해결)

## 해석
"중력장" = 시공간 양자의 거시적 유입 패턴. 수학적으로 뉴턴 g = -GM/r^2 r_hat과 동일하나 물리적 해석이 다름: 시공간 자체가 물질로 흘러들어감. 유입속도는 극도로 아광속 (v/c ~ 1.8e-51, 지구표면).
