## L21 Phase 구조

### Phase 1: B = 2π/ln2 — 진정한 독립 기원 탐색 (8인, 10회)

**목표**: "λ=2, 즉 H 배증이 왜 기본 단위인가"를 공유 가정 없이 정당화하는
**단일 제1원리** 발굴. 또는 "발굴 불가" 솔직 기록.

**제공 정보 (최소)**:
- B = 2π/ln2의 수치 ≈ 9.065
- RP⁴ = S⁴/Z₂ (반대점 동일시)
- Z₂ = {e, σ}, |Z₂| = 2
- DESI DR2: B_free ≈ 8.76 (L19 Phase 3 시뮬레이션 결과)

**질문 (방향만)**:
"B = 2π/ln2는 어떤 단 하나의 수학적 또는 물리적 원리에서 나오는가?
 그 원리는 Z₂의 어떤 성질에서 직접 나오는가?
 λ=2가 아닌 다른 값(λ=3, λ=e)을 자연스럽게 배제할 수 있는가?"

**금지**:
- 열역학/스펙트럼/이중성/위상 "4경로" 재진술 금지
- "H 배증이 자연스럽다"는 직관적 주장 금지 (계산 근거 요구)

**수렴 기준**:
- 독립 제1원리 발굴: λ=2 이외 배제 논증 + 계산 → A등급
- 단일 독립 원리 발굴 (배제 미완): B등급
- 발굴 실패 + 미해결 명시: C 유지

### Phase 2: A₂ = 2 — Twisted Sector Convention 해소 (8인, 10회)

**목표**: "twisted sector 포함 vs 배제"라는 convention 의존성을 물리적
**필연성**으로 교체. 또는 "A₂는 관측 결정 매개변수" 솔직 선언.

**제공 정보 (최소)**:
- EE2: A = A₂ × e^{-S_cl} = A₂ × e^{-π}
- DESI DR2: A_free = 0.088 ± 0.037
- e^{-π} ≈ 0.04322 → A₂_obs ≈ 2.04
- Z₂ covering: π: S⁴ → RP⁴, deg(π) = 2

**질문 (방향만)**:
"A₂ = 2를 결정하는 것은 어떤 물리적 원리인가?
 'twisted sector를 포함한다'는 선택이 관측 가능한 물리량에서
 어떻게 convention-independent하게 확인되는가?
 A₂가 정수여야 하는 이유가 있는가?"

**금지**:
- "deg(π)=2이므로 A₂=2"라는 단계 비약 허용 금지 (중간 단계 계산 요구)
- 관측값 A_free≈2.04에서 역산하는 접근 금지

**수렴 기준**:
- Convention-free 도출 성공 + 단계별 계산: A등급
- 위상학적 동기 + 부분 계산: B등급
- 실패 + "A₂는 관측 결정 매개변수" 선언: D+로 하향 확정

### Phase 3: Euclidean → Lorentzian 연결 — 구체 계산 요구 (8인, 15회)

**목표**: L20에서 확인된 4개 장벽 중 **최소 1개**를 표준 교과서 수준으로
해소하는 구체적 계산 제시. 특히 cos(B·ln H) 항의 Lorentzian 기원.

**제공 정보 (최소)**:
- L20 장벽 요약:
  - 장벽1: Wick rotation → cos→cosh (방향 역전)
  - 장벽2: Euclidean instanton과 오늘날 H(t)의 시간적 단절
  - 장벽3: 이산 스케일 불변성 B=2π/ln2의 Lorentzian 기원 없음
  - 장벽4: S_cl, A₂, B 각각의 Euclidean 기원 개별 미확립
- Hartle-Hawking no-boundary 파동함수: Ψ[h_ij] = ∫_{no-boundary} Dg exp(-S_E)
- de Sitter Wick rotation: a(τ)=H⁻¹cos(Hτ) → a(t)=H⁻¹cosh(Ht)

**질문 (방향만)**:
"cos(B·ln H) 항이 Lorentzian 시공간에서 관측 가능한 ω_DE에 나타나는
 물리적 메커니즘은 무엇인가?
 Euclidean 위상수학의 불변량이 동적 H(t)에 어떻게 전달되는가?
 이 전달에 adiabatic 근사가 필요하다면 그 유효 조건은?"

**특별 허용 (Phase 3 한정)**:
- Picard-Lefschetz 방법 시도 허용 (복소 안장점 적분)
- dS/CFT 대응 가설 시도 허용 (단, 미완성 이론임을 명시)
- "초기 조건 고정" 해석 허용 (단, 동적 H(t) 문제 해결 조건 명시)

**수렴 기준**:
- 장벽 1-2 모두 해소 + 계산 명시: A등급 (현실적으로 어려움)
- 장벽 1 또는 2 중 하나 해소: B등급
- 부분 완화 (adiabatic 조건 명시): C등급
- 4개 장벽 모두 미해소: F 유지

### Phase 4: 통합 — 논문 수준 이론 논증 재구성 (4인 코드리뷰, 3회)

**목표**: Phase 1-3 결과를 반영하여 EE2 이론의 논증을 논문 수준으로 재구성.
"틀린 주장은 삭제, 입증된 주장만 남기기."

**작업**:
- B 도출: Phase 1 결과 반영 → 실제 논문에 넣을 수 있는 논증 초안
- A₂ 도출: Phase 2 결과 반영 → 실제 논문에 넣을 수 있는 논증 초안
- Euclidean→Lorentzian: Phase 3 결과 반영 → 실제 논문에 넣을 수 있는 논증 초안
- 각각에 대해: "논문에 쓸 수 있다 / 아직 쓸 수 없다" 판정

**수렴 기준**:
- 3개 항목 모두 "논문 수준" 판정: A
- 1-2개 항목 논문 수준: B
- 모두 미달: "이론 미완성 솔직 기록"
