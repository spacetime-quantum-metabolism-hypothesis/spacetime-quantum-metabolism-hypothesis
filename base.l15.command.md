## 핵심 문제: 이 파이프라인은 과적합인가?

### 답: 부분적으로 그렇다. 구체적으로:

**문제 1 — 순환 논리 (P3 지적)**

```
[잘못된 순서]   데이터 피팅 → 잘 맞는 수식 선택 → 이론 일관성 체크
[올바른 순서]   이론 공리 → 수식 유도 → 데이터 검증
```

L14는 180개 이론을 DESI BAO에 먼저 피팅한 뒤, 잘 맞는 것을 게임체인저로 불렀다.
그 다음 A1+A2와 얼마나 일관성이 있는지 점수를 주는 것은 **post-hoc rationalization**.

→ **개선**: A1+A2 필터를 Stage 0 직후로 이동. 이론적으로 의미 없는 수식은 데이터 보기 전에 제거.

**문제 2 — 선택편향 (P8 지적)**

L15 필터 기준 자체가 L14 결과를 보고 설계됨.
AX3가 chi²=9.10으로 잘 나오는 걸 알고 나서 "F-C 패밀리가 A1 정합 우수"라고 기준을 세우면
→ 그 기준은 AX3를 살리기 위해 만들어진 것.

→ **개선**: 독립 validation = DESI DR3 예측. L15 최종 생존자는 DR3 결과를 예측해야 함.
  L15가 살린 모델이 DR3에서도 유효하면 진짜, 아니면 L14 과적합.

**문제 3 — AICc 미사용 (P1 지적)**

n=7, k=3일 때 AIC는 과소 페널티. AICc 공식:
```
AICc = AIC + 2k(k+1)/(n-k-1) = chi2 + 2k + 2k(k+1)/(n-k-1)
n=7, k=3: AICc = chi2 + 6 + 24/3 = chi2 + 14   (AIC = chi2 + 6과 크게 다름)
```

7pt AIC 기반 판단은 파라미터 비용을 절반으로 과소평가.

## Stage 0T: A1+A2 이론 필터 (데이터 보기 전)

> **이 Stage가 먼저여야 하는 이유**: 이론에서 유도 불가능한 수식을 데이터 적합도와 무관하게 제거.
> 데이터 보고 나서 이론 일관성을 체크하면 순환 논리.

### A1 공리 적용 기준 (개선된 버전)

A1: "물질 → 시공간 양자 소멸, 빈 공간 → 시공간 양자 생성"

**수식이 A1에서 자연스럽게 유도되려면:**
1. omega_de가 물질 밀도(rho_m 또는 Om*(1+z)^3)의 **감소함수**여야 함 (고z에서 DE 적음)
2. 수식 구조가 "A1 소멸항"의 해석 가능한 형태여야 함 (단순 경험적 함수 금지)

**A2 공리 적용 기준 (강화):**

A2: "양자-고전 경계는 A1에서 유도된다."

기존 기준 "특성 스케일 파라미터 포함"은 너무 약함 (모든 3-파라미터 모델 통과).
강화된 기준:
- 수식의 특성 스케일이 **플랑크 밀도, H0, 또는 물질-복사 등가 스케일**과 연결 가능해야 함
- 단순히 B 파라미터를 피팅으로 결정하는 것은 A2 충족 불가

### 패밀리별 A1+A2 판정

| 번호 | 패밀리 | 수식 | A1 (0-2) | A2 (0-1) | 판정 |
|------|--------|------|-----------|-----------|------|
| 1 | F-G | OL0*(1+A*(E-1)*exp(-B*(E-1)^n)) | 1 (E 단조) | 0 (스케일 임의) | **유보** |
| 2 | F-C | OL0*(1+A*(1-cos(B*x_m))) | **2** (물질 직접) | 0 (B 임의) | **유보** |
| 3 | F-S2 | OL0*(1+A*sin^2(B*f(z))) | 0 (진동, 비단조) | 0 | **KILL** |
| 4 | F-L | OL0*(1+A*x_m/(B^2+x_m^2)) | **2** (물질 직접, 단조) | 0 (B 임의) | **유보** |
| 5 | F-W | OL0*exp(-A*vol)*(1+B*vol) | 1 (vol 단조) | 0 | **유보** |
| 6 | F-P | OL0*(1+A*x^2*exp(-B*x^2)) | 0 (피크형, 비단조) | 0 | **KILL** |
| 7 | F-H | 성장×감쇠 Page curve | 0 (A1+A2와 무관계) | 0 | **KILL** |
| 8 | F-CL | OL0*(1+A*(1-cos(B*ln(E)))) | 1 (로그 단조) | 0 | **유보** |
| 9 | F-CS | OL0*(1+A*(1-cos(B*sqrt(x)))) | 0 (진동 가능) | 0 | **KILL** |

**A1+A2 KILL 판정 (4개)**:
- F-S2 (AX2): sin² 진동 구조 → A1 단조성 위반
- F-P (RM3): x² × exp(-Bx²) 피크형 → 고z에서 DE 재증가, A1 위반
- F-H (H2): Page curve → SQMH 공리와 무관한 경험적 구조
- F-CS (AN3): cos(B*sqrt(x)) → 진동 가능

**Stage 0T 생존 (5개)**: F-G(TF3), F-C(AX3), F-L(HB3), F-W(ST1), F-CL(EE2)

> **주의**: A2 점수는 현재 모든 유보 모델이 0점 (B 파라미터 스케일 미해석).
> A2는 별도 이론 작업 필요. Stage 0T는 A1 기준만으로 필터링.

## Stage 1: 13포인트 공분산 chi² + AICc

### 목적

7pt diagonal chi²를 버리고, 진짜 DESI DR2 공식 통계로 재평가.
AIC 대신 소표본 교정 AICc 사용.

### 필요 데이터

- DESI DR2 공분산: `bao_data/desi_dr2/desi_dr2.cov` (13x13)
- ΛCDM baseline (13pt): chi²≈21.4 at Om=0.326 (CLAUDE.md 확인)

### AICc 공식

```
AICc = chi2 + 2k + 2k(k+1)/(n-k-1)

n=13 (13포인트), k=파라미터 수:
  k=1 (ΛCDM Om만): AICc = chi2 + 2 + 2*1*2/(13-1-1) = chi2 + 2 + 0.364 = chi2 + 2.364
  k=3 (Om,A,B):    AICc = chi2 + 6 + 2*3*4/(13-3-1) = chi2 + 6 + 2.667 = chi2 + 8.667
  k=4 (Om,A,B,C):  AICc = chi2 + 8 + 2*4*5/(13-4-1) = chi2 + 8 + 5.0   = chi2 + 13.0

ΛCDM AICc_ref = 21.4 + 2.364 = 23.764
```

### chi² 임계값 (dof 기반, p=0.05)

```
dof = n - k = 13 - 3 = 10 → chi²_crit(10, p=0.05) = 18.31
dof = 13 - 1 = 12 → chi²_crit(12, p=0.05) = 21.03
```

### Kill 기준 (Stage 1)

- `chi2_13pt > 21.4` (ΛCDM보다 나쁨) → **KILL**
- `Delta_AICc = AICc_model - AICc_LCDM > 0` → **KILL**
- 즉 chi²_13pt < 21.4 이고 AICc 개선이 있어야 통과

### 예상 결과 (5개 후보 기준)

실제 chi²_13pt는 실행 후 알 수 있음. 예비 예측:

| 패밀리 | 대표 | chi²_L14(7pt) | 예상 chi²_13pt | 예상 Delta_AICc |
|--------|------|--------------|----------------|----------------|
| F-G | TF3 | 10.492 | ~16-18 | ~-2 ~ +1 (경계) |
| F-C | AX3 | 9.103 | ~13-15 | ~-6 ~ -3 (유리) |
| F-L | HB3 | 10.370 | ~15-17 | ~-4 ~ -1 |
| F-W | ST1 | 10.638 | ~16-18 | ~-2 ~ +1 (경계) |
| F-CL | EE2 | 9.866 | ~14-16 | ~-5 ~ -2 (유리) |

> 7pt에서 chi²가 낮은 AX3, EE2가 13pt에서도 유리할 가능성 높음. 실제 결과는 실행 후 확인.

## 정직한 한계 기록

1. **이론 → 현상 연결이 자연스럽지 않음**: L14에서 데이터 피팅 먼저, 이론 점수 나중. Stage 0T를 앞으로 당겨도 A2 스코어는 여전히 약함 (B 파라미터 스케일 미해석).

2. **L15 자체가 L14를 보고 설계됨**: 필터 기준이 AX3를 살리는 방향으로 편향될 수 있음. 진짜 검증은 DR3.

3. **A2 이론 작업 미완성**: "B 파라미터가 Planck 스케일에서 어떻게 유도되는가"에 대한 이론적 계산이 없음. Stage 0T A2는 현재 형식적.

4. **JCAP/PRD Letter 조건 (CLAUDE.md)**: Q17 완전 달성 또는 (Q13+Q14) 동시 달성 전 제출 금지. L15 생존자도 이 조건 별도 확인 필요.
