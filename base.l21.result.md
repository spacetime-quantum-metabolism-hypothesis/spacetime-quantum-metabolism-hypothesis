## Phase 1: B = 2π/ln2 독립 기원 탐색 (5회)

### 시도한 경로 (11개)
1. ord(Z₂)=2 → λ=2 직접 대응: 정당화 없음
2. 피복 차수 2 → λ=2: 대척점과 "2배" 연결 불가
3. σ(H) = 1/H: Z₂ 불변이나 B 무결정
4. 가산 Z₂ 작용 on ln H: 직선에서 불가
5. Laplace 스펙트럼 (RP⁴): 수치적 불일치, π 아닌 값
6. H₂(RP⁴;Z)=Z₂ torsion 위수 → ln 2: 변환 근거 없음
7. Selberg 제타/측지선 길이: π ≠ ln 2
8. Mellin 공간 Z₂ 대칭: B 무결정
9. **L(1,χ_{Z₂}) = ln 2** (Dirichlet L-함수): 수학적으로 참, 물리 메커니즘 미완
10. Gal(ℂ/ℝ) = Z₂ → [ℂ:ℝ]=2: ln H에 비자명 작용 없음
11. 최소성 원리 (|Z₂|=2 최소 비자명 군): 순환 논리

### 핵심 발견
**L(1,χ_{Z₂}) = ln 2**: Z₂의 비자명 Dirichlet 지표의 L-함수값이 정확히 ln 2.
- Dirichlet eta 함수 η(s) = (1-2^{1-s})ζ(s), η(1) = ln 2 ✓
- χ₋(n) = (-1)^{n+1}: Z₂의 부호 표현으로 가중된 합
- 다른 군: Z₃ → B≈10.39, Z₄ → B=8.0 (관측값 8.76과 불일치)
- Z₂가 관측 데이터에 가장 가까운 B를 준다

**추가 발견 (예상 외)**:
- RP⁴ = S⁴/Z₂(반대점)은 **비가향 다양체** (w₁ ≠ 0)
- 비가향 시공간의 우주론적 전제 자체에 문제 제기됨

### 판정
- λ=2 배제 논증: 실패 (관측 배제는 가능, 제1원리 배제는 불가)
- 독립 경로 수: 완성 0개. L-함수 경로는 유망하나 미완.
- **등급: C** (유지)

## Phase 2: A₂ = 2 Convention 해소 (5회)

### 시도한 경로
1. **Orbifold formula**: Z_twisted(S⁴)=0 필요 → convention
2. **DW partition function**: cocycle 선택 의존 → convention
3. **BF theory on RP⁴**: linking#=0이면 Z=2, =1이면 Z=0
   - RP⁴의 H²∪H²=1(mod 2) → linking#=1 → Z=0 → A₂=0 또는 1
4. **Euler characteristic**: χ(S⁴)/χ(RP⁴)=2 → A₂와 동일시 근거 없음
5. **Index theorem**: involution 선택 의존
6. **String theory 유추**: oriented+unoriented 합산 → factor 2, 그러나 4D 직접 적용 아님

### 치명적 약점
- e^{-S_E(RP⁴)} = e^{-π/2} ≠ 2·e^{-π}: 기하학적 작용이 vol에 비례하면 A₂=2 구조 재현 불가
- BF theory에서 intersection form → Z(RP⁴)=0 (RP⁴ 기여 없음) 가능성
- 관측: A_free/e^{-π} ≈ 2.04 (1.1σ 일치, 확증 아님)

### 정직한 선언
> **A₂는 관측 결정 매개변수다.**
> Z₂ 구조가 A₂∈{0,1,2}를 시사하지만 어느 값인지는 관측으로만 결정된다.
> Convention-free 도출은 현재 이론 구조 내에서 불가능하다.

### 판정
- Convention-free 도출: **실패**
- **등급: D+** (L20과 동일, 하향 없음)

### Phase 3 추가 결과: Euclidean → Lorentzian

**장벽 1 (cos→cosh) 부분 완화**:

세 방향에서 독립적 접근:

1. **H^{iB} 해석성** (Run 13, 16, 17):
   - cos(B·ln H) = Re(H^{iB}) — H가 실수 양수인 한 항상 실수 실수함수
   - Wick 회전이 cos를 cosh로 바꾸지 않는 조건: H가 Wick 회전 후에도 실수 양수 유지
   - 물질 우세 시대에서 H→순허수로 되어 이 조건 깨짐 (Run 18에서 반례 확인)

2. **장벽 1의 정량적 심각성** (Run 15, 18에서 독립 확인):
   - 물질 우세 시대 완전 Wick 회전: cosh(Bπ/2) ≈ cosh(14.24) ≈ 750,000
   - 대규모 진폭 폭발 → 물리적 허용 불가
   - 장벽 1이 가장 치명적 장벽임을 정량화

3. **Jacobi theta 함수** (Run 18):
   - θ₃(0|τ)에서 cos는 Euclidean에서 자연스럽게 등장
   - Wick 회전 방향성 문제를 theta 함수의 해석적 연속으로 우회 가능
   - 단, τ = f(H) 동일시가 추가 전제

4. **RP⁴ antipodal↔Wick 비교환** (Run 17):
   - antipodal 대칭 = 시간 반전 T 포함 → T와 Wick rotation 비교환
   - 비교환성이 cos→cosh 전환을 막는 메커니즘 후보
   - 엄밀한 계산 미완

**장벽 2 (시간적 단절) 부분 진전**:
- RG 스케일 재해석 (Run 19): H(t)를 RG 에너지 스케일 μ로 해석 → Euclidean seed + Lorentzian RG 흐름
- dS/CFT + 복소 공형 차원 Δ=iB/2 (Run 16): H^{iB} = e^{iB·ln H} → 실부 = cos(B·ln H)
- 두 방향 모두 가설 수준, 엄밀화 미완

**장벽 3, 4**: 미해소

**등급: C** (최초 10회 F → 상승. 장벽 1, 2 부분 완화, 장벽 3, 4 미해소)

### Run 31: H 단위 정규화 및 장벽 3' 분석

**핵심 발견**:
- **Δ(B·ln H) ≈ 18.8 ≈ 3×2π — 단위 독립적** (z=0~3 구간)
  - H의 단위가 무엇이든, z=0→3 구간의 위상 변화량 ≈ 18.8은 불변
  - 이 진동 범위는 H/H_0 비율만 결정하므로 단위에 무관
- **절대 위상 φ_0 = B·ln(H_0/H_*)는 단위에 의존**: 이론 예측의 실제 값을 고정하려면 H* 기준이 필요
- **H/H_0 정규화**: z=0에서 cos=1 → ω_DE=OL0 → ΛCDM 초기조건 자연스러움. DESI DR2 피팅에서 암묵적으로 이 정규화 사용됐을 가능성 높음
- OL0가 복소수라면 단위 변환 = OL0 위상 재정의로 흡수 가능 (단, OL0 실수 조건이 단위를 암묵 고정)
- **미해결**: φ_0 고정을 위한 AdS₂ 고정점 H_AdS의 독립 유도

### Run 32: CPT 이중항 합산 규칙 엄밀화

**핵심 발견**:
- **전제 P4 (1-루프 실수성)**: 타원 연산자의 실스펙트럼 → Z[α] ∈ ℝ (1-루프 수준)
  - Z_total = Z[α] + Z[α]* = 2Z[α] → A₂=2 **1-루프에서 정확** (근사 없음)
  - 이전 cos(π/16)≈1 근사 문제가 1-루프 수준에서는 불필요했음을 확인
- **선형합 vs 제곱합 정당화**: CPT 이중항은 같은 힐베르트 공간에 공존 → 간섭 가능 → 선형합이 맞음
  - CP 위반 현상(입자-반입자 간섭)이 실제로 관측되는 선례
- **최후 장벽 P3**: dS₄/Z₂의 Z₂가 **게이지**인지 **글로벌**인지 미판정
  - Z₂가 게이지 대칭 → superselection → 제곱합 → A₂=2 붕괴
  - Z₂가 글로벌 대칭 → 코히어런스 → 선형합 → A₂=2 성립
- **등급**: A₂ = B+(P3 검증 조건부) — P3 해소 시 A등급 가능
