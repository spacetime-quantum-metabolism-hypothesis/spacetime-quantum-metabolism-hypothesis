# base.l17.command.md — L17 통합 수식 전면 검증

> 작성: 2026-04-12.
> 목적: EE2+AX3 통합 수식을 현상적·이론적으로 처음부터 검증.
> 원칙: 이론 도출 — 방향만 제공, 지도 극단적 금지. 역할 사전 지정 없는 8인 상호토의.
> 과적합 방지 명시. 시뮬레이션 실패 시 코딩 버그 우선 의심 → 4인 코드리뷰.

---

## 통합 수식 정의

```
omega_de = OL0 * (1 + A * (1 - cos(B1*theta + B2*ln(E))))

theta = (Om*(1+z)^3 - Om) / OL0     ← 물질 밀도 변화 (AX3 성분)
ln(E) = ln(H(z)/H0)                 ← 팽창률 로그 (EE2 성분)
```

파라미터: Om, A, B1, B2 (총 4개)

특수 경우:
- B1=0: 순수 EE2
- B2=0: 순수 AX3
- B1,B2 ≠ 0: 통합 (L17 대상)

물리적 의미:
- B1*theta: 물질 소멸이 구동하는 위상 (A1 소멸 측면)
- B2*ln(E): 팽창 창조가 구동하는 위상 (A1 창조 측면)
- 합산 위상: 두 A1 과정의 결합

## Phase 1: 현상적 검증 (시뮬레이션)

### Stage 0T: A1 이론 필터

기준: 고z에서 omega_de 단조 감소.
B1*theta + B2*ln(E) 모두 양수(z 증가 시) → 위상 증가 → cos 진동.
단조성 조건: B1, B2 > 0일 때 초기 거동 확인.

### Stage 1: 13포인트 chi² + AICc

ΛCDM 기준 (Om 자유): chi²≈17.46, AICc≈19.83
AX3 (k=3): chi²=4.996, AICc=13.663 (dAICc=-6.16)
EE2 (k=3): chi²=5.601, AICc=14.268 (dAICc=-5.56)

통합 수식 (k=4) AICc 패널티: +4.33 추가 (3파라미터 대비)
→ chi² < 1.27이어야 EE2보다 AICc 개선.
→ 엄격한 기준.

### Stage 2: 물리 합리성

- w0 ∈ [-1.5, -0.3]
- wa ∈ [-3, 1.5]
- Om ∈ [0.28, 0.36]
- 고z DE 감쇠 확인

### Stage 3: Joint BAO+SN+CMB+RSD

비교 기준:
- ΛCDM: joint=1687.76
- EE2: joint=1657.04 (delta=-30.73, 현재 최강)
- AX3: joint=1664.18 (delta=-23.58)

통합 수식이 EE2보다 개선되면 → 의미 있는 통합.
그 미만이면 → EE2 단독이 더 단순하므로 EE2 선택.

### Stage 5: MCMC

nwalkers=32, burn=500, prod=800.
4D 파라미터 공간 (Om, A, log(B1), log(B2)).
B1, B2 모두 log 공간 샘플링.
수용률 목표: > 0.05.

## Phase 2: 이론적 정당성 (8인 상호토의)

### 구성 원칙

- 역할 사전 지정 없음. 중복 없음.
- 8인이 각자 자유 배경으로 출발 → 토의에서 자연 분업.
- 방향만 제공. 지도 극단적 금지.
  - L16에서 발견된 경로(Berry, Josephson, e-fold 등) 힌트 일체 없음.
  - AX3, EE2가 성분이라는 힌트도 없음.

### 제공하는 것 (오직 이것만)

1. A1+A2 공리
2. 목표 수식: omega_de = OL0*(1+A*(1-cos(B1*theta+B2*ln(E))))
3. 물리적 맥락: 우주론 스케일, DESI 관측 범위
4. 질문: "이 수식이 A1+A2에서 왜 나와야 하는가?"

### 과적합 방지 체크

이론 유도 과정에서 금지:
- "데이터가 이렇게 나왔으니 이론도 이렇게 맞춰야 한다" 식 역공학
- 파라미터 의미를 사후에 짜맞추는 방식

허용:
- 수식 형태에서 구조를 역추적
- A1+A2의 논리적 귀결을 전방 추론

### 수렴 기준

8인 중 5인 이상 같은 유도 경로 동의 → 성공.
10회 이상 반복, 과반 성공 → 이론 수식 확정.

## Phase 3: B 파라미터 물리적 크기 분석

통합 수식의 B1, B2 best-fit 값을 Planck 단위로 해석.

분석 대상:
- B1 (물질 위상 계수): [Om, Planck 밀도, H0]의 함수인가?
- B2 (팽창 위상 계수): [c/H0, 곡률 스케일]의 함수인가?
- B1/B2 비율: "소멸 대 창조" 상대 강도. 이론 예측 가능한가?

## Phase 4: 결론 및 논문 섹션

| 결과 | 논문 취급 |
|------|---------|
| 통합이 EE2보다 개선 + 이론 도출 성공 | 논문 메인 수식 |
