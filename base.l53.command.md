# L53 시뮬 사양

## 목적

L48~L52 결과: 4 검증, 4 다른 σ_0 (단일 σ_0 사망).
- L50: σ_IR=3.04×10⁸, k_t=1e-2 (T17 H_0 적합)
- L52: σ_IR=5.36×10⁷, k_t=0.68 (T20 σ_8 KiDS 적합)
- L50 vs L52: σ_IR 0.75 dex, k_t 1.83 dex 모순

**L53 목적**: 단일 (σ_IR, σ_UV, k_t)이 4 검증 *동시* 만족 영역 *존재 여부* 결정.

## 핵심 검증 (4 조건)

```
T17 H_0 tension : k_eval=1e-4 Mpc^-1 → σ_eff_T17 → SQT ODE → H_0_CMB_apparent
                  χ²_T17 = ((H_0_CMB_app - 67.4) / 0.5)²
T17 자기일관    : τ_q = σ_eff_T17/(4πG), τ_q·3H_0=1 검증
                  χ²_sc = ((τ_q·3H_0 - 1) / 0.1)²
T20 σ_8         : k_eval=0.13 Mpc^-1 → σ_eff_T20 → 성장 ODE → σ_8
                  χ²_T20 = min(((σ_8-0.811)/0.006)², ((σ_8-0.76)/0.02)²)
T22 a_0         : k_eval=100 Mpc^-1 → σ_eff_T22≈σ_UV → a_0 예측
                  a_0_pred = 1.2e-10 · √(σ_eff_T22/3.31e9)
                  χ²_T22 = ((a_0_pred/1.2e-10 - 1) / 0.1)²
합산: χ²_total = χ²_T17 + χ²_sc + χ²_T20 + χ²_T22
```

## 모델

```
σ_0(k) = (σ_IR + σ_UV·(k/k_t)²) / (1 + (k/k_t)²)   (L50 Model IV)
σ_eff_T17 = σ_0(k=1e-4)  ≈ σ_IR (k_t >> 1e-4 이면)
σ_eff_T20 = σ_0(k=0.13)
σ_eff_T22 = σ_0(k=100)   ≈ σ_UV (k_t << 100 이면)
```

T17 ODE: L50과 동일 (σ_IR → σ_eff_T17 교체), Γ_0 = 1.0 고정
T20 성장 ODE: L52와 동일 (friction_factor = 1 + 0.1·log10(σ_eff_T20/σ_ref))

## 그리드

- σ_IR: [10⁷, 10^9.5], 15점 로그 등간격
- σ_UV: [10⁸, 10¹⁰], 15점 로그 등간격
- k_t: [10⁻⁴, 10¹] Mpc⁻¹, 15점 로그 등간격
- 총 3375 점, 8 병렬

## 통과 기준

```
PASS   : χ²_total < 16  (4 검증 동시 ~2σ)  → ★★★★☆ 격상
PARTIAL: 16 ≤ χ²_total < 50               → ★★★☆☆ 유지
FAIL   : χ²_total ≥ 50                    → ★★☆☆☆ 강등
```

## 시각화 (6 패널)

- (a) σ_IR × σ_UV 슬라이스 @ best k_t
- (b) σ_IR × k_t 슬라이스 @ best σ_UV
- (c) σ_UV × k_t 슬라이스 @ best σ_IR
- (d) best 점에서 검증별 χ² 막대
