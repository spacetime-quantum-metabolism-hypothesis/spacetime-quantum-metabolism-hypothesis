## Phase 1 시뮬레이션 결과

### 코드리뷰 이력 (4인팀, CLAUDE.md 규칙 준수)

**버그 1 (데이터 로딩)**: 잘못된 변수명 `DESI_Z`, `DESI_OBS`, `DESI_COV_INV`
→ 수정: `from desi_data import DESI_DR2, DESI_DR2_COV, DESI_DR2_COV_INV`

**버그 2 (chi² 계산)**: scipy.quad 내부루프 (L15 불일치)
→ 수정: L15 검증 패턴 (trapezoid) 그대로 복사

**버그 3 (거리 인자)**: `c_kms/(H0_KMS*ei(z))` 단위 오류
→ 수정: `fac = c_SI / (H0_SI * Mpc_m)` (L15 동일)

**버그 4 (Stage 0T)**: `arr[z_hi].mean() < OL0` — AX3/EE2 오탈락
→ 수정: A1 이론 필터 = 양수 유한성 검사 + A ≥ 0
  (A1 물리: 고z에서 DE > OL0 정상 — 더 많은 물질 소멸 → 더 많은 DE 생성)

**버그 5 (CMB 고z 외삽)**: cubic extrapolation at z>6 → chi2_joint 폭발
→ 수정: z > Z_CUT 구간 LCDM bridge (CLAUDE.md 규칙 적용)

주목: UNI-unified의 wa=+0.096 (양수) — DE가 과거에 더 약함. A1 이론과 방향 불일치.

### 과적합 판정 (AICc 기준)

**UNI AICc (18.347) - EE2 AICc (14.268) = +4.079**

기준: chi² < 1.27이어야 EE2보다 AICc 개선 가능 (4파라미터 패널티 +4.33)
실제 chi²_13pt(UNI) = 5.347 >> 1.27

**결론: EE2 단독 선택. 통합 수식은 AICc 개선 없음.**
