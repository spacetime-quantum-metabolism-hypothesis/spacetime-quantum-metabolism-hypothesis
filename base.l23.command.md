### Phase 1: 수치 완성 — ee2_fit.py + DESI χ² (8인, 8회)

**목표**: ee2_ode.py 결과를 DESI DR2 BAO 13pt 데이터와 연결. χ² 피팅 완성.

**현재 상태**:
- ee2_ode.py: E(z) ODE 수렴 완료. H(z), ω_DE(z) 플롯 3종 완성.
- 미완: DESI 13pt 공분산 포함 χ² 계산. H_ref=H₀ 가정 명시.

**제공 정보 (최소)**:
- ee2_ode.py 기존 코드 (simulations/l19/)
- DESI DR2 BAO 13pt: CobayaSampler bao_data 공식 파일
- 비교 기준: AICc(ΛCDM) ≈ 21.4 (Ω_m 자유, best-fit), AICc(EE2 고정) = 10.8

**질문 (방향만)**:
"ODE로 계산한 H(z)가 DESI 13pt 데이터 포인트를 얼마나 잘 설명하는가?
 χ²(EE2 고정) 대 χ²(ΛCDM best-fit)의 차이는?
 H_ref=H₀ 가정이 χ²에 어떤 영향을 주는가?
 논문에 쓸 수 있는 수준의 H(z), ω_DE(z) 그래프는 어떻게 생겨야 하는가?"

**과제 항목**:
- [#1] ee2_ode.py 기존 코드 → H(z), ω_DE(z) 그래프 논문 품질로 개선
- [#8] ee2_ode.py → DESI 13pt χ² 피팅 구현
- [#9] H_ref 처리 방식 명시: "H_ref=H₀ 가정 (외부 관측 의존, 이론 내부 고정 불가)"를 논문 본문에 정확히 기술
- [#12] ee2_fit.py 완성 → 현상론 논문 데이터 자료 확보

**요구 결과물**:
- ee2_fit.py (새 파일): DESI χ² 계산 포함
- 논문 품질 그래프: H(z)/H_ΛCDM(z), ω_DE(z), ΔH/H_ΛCDM (z=0~3)
- χ² 수치: EE2(고정), EE2(자유), ΛCDM 비교표

**코드 요구사항**:
- CLAUDE.md: ODE solver odeint, CobayaSampler 공식 BAO 데이터, 공분산 처리
- 4인 코드리뷰 필수
- H_ref=H₀ 가정을 코드 주석과 논문에 명시

**수렴 기준**:
- χ² 수치 완성 + 논문 그래프 생성: Phase 1 완료 → 현상론 논문 최소 요건 충족
- χ²(EE2 고정) < χ²(ΛCDM): 논문 주장 가능
- χ²(EE2 고정) > χ²(ΛCDM): "AICc 이점이 χ²에서 확인 안됨" 솔직 기재

### Phase 2: MCMC — B=9.065 재현 여부 (8인, 8회)

**목표**: A, B를 자유 파라미터로 DESI DR2 BAO 13pt에 피팅. 이론 생사 판정.

**항목**: [#2]

**제공 정보 (최소)**:
- Phase 1에서 완성된 ee2_fit.py (ODE + DESI χ²)
- 이론 예측값: A_theory = 0.0864, B_theory = 9.065
- 비교: AICc(EE2 고정) = 10.8, AICc(ΛCDM) = 21.4
- L19 결과: A_free = 0.088 ± 0.037, B_free ≈ 8.76 (불확실도 미확인)

**질문 (방향만)**:
"A, B 자유 MCMC에서 사후 분포 피크는 어디인가?
 B의 68% 신뢰구간에 9.065가 포함되는가?
 A의 posterior가 이론값 0.0864를 몇 σ에서 포함하는가?
 Bayesian log-evidence log K: EE2(자유) vs EE2(고정) vs ΛCDM 비교?"

**금지**:
- B=9.065를 prior에 강하게 반영 금지
- 의도적으로 좁은 prior 설정 금지
- 결과를 이론에 유리하게 보고하는 시각화 금지

**코드 요구사항**:
- MCMC: emcee 또는 dynesty
- Prior: A ∈ [0, 0.5] uniform, B ∈ [1, 20] uniform (flat, 이론값 강조 금지)
- Bayesian log-evidence: dynesty 권장
- 결과: corner plot + 1D marginal + log K 수치
- 4인 코드리뷰 필수

**수렴 기준**:
- B posterior 피크가 9.065±1 이내 + 68% CI 포함: "B 이론 지지" → B 등급 C+→B
- B posterior 피크가 9.065와 2σ 이상 불일치: "B는 관측 결정 파라미터" → D로 하향
- Posterior 평탄: 데이터 변별력 부족 → C+ 유지

### Phase 3: B의 ln2 동역학적 기원 — RG 고정점 탐색 (8인, 8회)

**목표**: ln2가 어디서 오는가? 이진 마스터 방정식 Mellin 구조로부터 RG 흐름에서 이진성이 동역학적으로 선호되는가?

**항목**: [#5] [#6]

**배경**:
- L22에서 확인: 이진 Mellin 변환에서 ln2 자연 발생 (C+). 그러나 "왜 이진인가"의 동역학 없음.
- 사용자 방향: Wilson-Fisher 유형 이진 고정점. RG 흐름에서 λ=2가 동역학적으로 선택되는가?

**제공 정보 (최소)**:
- 이진 마스터 방정식: W_+/W_- = λ에서 Mellin 스펙트럼 극점 s_n = 2πin/lnλ
- Wilson-Fisher 고정점: ε-전개에서 λ*는 어떻게 고정되는가?
- 스케일링 장: 임계 지수 η, ν와 λ의 관계

**질문 (방향만)**:
"마스터 방정식의 전이율 비율 λ = W_+/W_-를 고정하는 RG 고정점이 존재하는가?
 Wilson-Fisher 유형 이진 고정점(λ=2)이 동역학적으로 우선되는가, 아니면 임의 λ가 고정되는가?
 λ=2가 선택되는 조건은 무엇인가? (정보 최대화? Z₂ 대칭? 엔트로피 극대화?)
 만약 λ=2가 동역학적으로 강제된다면, AICc 패널티 없이 ln2가 도출되는가?"

**금지**:
- λ=2를 가정으로 출발하는 것은 순환. 동역학에서 λ=2가 나와야 함.
- 정보이론적 이유(이진 엔트로피 최대화)를 물리 원리로 선언 금지.

**수렴 기준**:
- RG 고정점에서 λ=2 동역학 도출 + 논증: B 등급 C+→B+
- 동역학적 선택 원리 없음 확인: "ln2는 약정적" 확정 → 논문 한계 섹션에 기재
- λ가 [1, ∞)의 임의값으로 고정: B 등급 D로 하향

### Phase 4: A의 e^{-π} — Chern-Simons 또는 모듈러 형식 동역학 (8인, 8회)

**목표**: A = 2e^{-π}의 물리적 동역학 유도. Chern-Simons 위상 작용 또는 Dedekind η(i), Ramanujan 특수값에서 동역학적 도출 가능성 탐색.

**항목**: [#7]

**배경**:
- L22 확립: A = 2e^{-π}의 수식 수준 기원은 B+ (θ₃(τ=i), Hopf CS, dS 분배함수).
- 공통 약점: 수식 수준 동형이지, 왜 이 값이 선택되어야 하는지 동역학 없음.
- 사용자 방향: CS 위상 작용 (S_CS = π → A = 2e^{-S_CS}) 또는 Dedekind η(i) 특수값.

**제공 정보 (최소)**:
- L22 확립: S_CS = (1/4π)·∫_{S³} A∧dA = (1/4π)·4π² = π (Hopf, k=1)
- Dedekind η(τ) = e^{πiτ/12} Π(1-e^{2πinτ}), η(i) = Γ(1/4)/(2π^{3/4})
- Ramanujan: π^{1/4}/Γ(3/4) = θ₃(0|i) = 1.0864...

**질문 (방향만)**:
"Chern-Simons 작용 S_CS가 EE2의 A 파라미터를 고정하는 경우, 이 연결이 어떤 물리 원리에서 나오는가?
 k=1 Hopf 연결에서 S_CS=π가 나오는 조건이 우주론에서 어떻게 실현되는가?
 Dedekind η(i) 또는 Ramanujan 특수값이 암흑에너지 파라미터를 동역학적으로 고정하는 메커니즘이 있는가?
 만약 S_CS=π가 단순 경로 적분에서 안장점 조건으로 나온다면, A는 추가 공리 없이 도출되는가?"

**수렴 기준**:
- CS 안장점 → A = 2e^{-π} 완전 도출 + 논증: A 등급 B+→A
- 수식 수준 연결만 재확인: B+ 유지
- 모든 경로 실패: "A는 수식 수준 우연" 선언 → 논문 한계 섹션

### Phase 5: 팬텀 횡단 해결 — CPT 이중항 스칼라 구성 (8인, 6회)

**목표**: ω_DE,min = -1.17 < -1 팬텀 구간을 물리적으로 처리. CPT 이중항에서 Quintom 구조 자동 생성 검토.

**항목**: [#3]

**배경**:
- L22 확립: CPT 이중항 (Z₂ 글로벌) → ρ_DE = ρ_1 + ρ_2 (Quintom 구조 자동). L_eff = +(1/2)(∂φ₊)² - (1/2)(∂φ₋)²
- L22 미완: Quintom의 구체적 스칼라 장 V(φ) 형태. 유령 안정성 조건 c_s² > 0.
- ω_DE,min = -1.17은 팬텀 구간이므로 표준 단일 스칼라로는 불가.

**제공 정보 (최소)**:
- EE2 ω_DE(H) 공식 (H/H₀ 정규화)
- CPT 이중항 구조: Z[α] + Z[β] = 2Re[Z[α]] → A₂=2. L_eff에 ghost 항 포함.
- Hamilton-Jacobi 역변환: φ(H) = (2M_pl/B√(3A))·ln|tan(B/4·ln(H/H₀))| + φ*
- ω_DE,min = -1.1729 (ODE 수치 결과)

**질문 (방향만)**:
"CPT 이중항에서 φ₊ (정규 스칼라)와 φ₋ (ghost) 두 필드가 EE2의 ω_DE(H)를 어떻게 재현하는가?
 c_s² > 0 (유령 안정성) 조건이 팬텀 구간에서 충족되는가?
 HJ 역변환 φ(H)에서 Quintom 퍼텐셜 V(φ₊, φ₋)를 명시적으로 쓸 수 있는가?
 팬텀 횡단이 실제로 발생하는가, 아니면 ω → -1^- 점근인가?"

**수렴 기준**:
- Quintom V(φ₊, φ₋) 명시적 구성 + c_s² > 0 확인: 유령 안정성 B+→A-
- 팬텀 횡단이 점근임 확인 (ω → -1^-로만): 단일 스칼라로 충분 → 구조 단순화
- c_s² < 0 구간 존재: 유령 불안정 → "팬텀 구간 이론적 불안정" 논문 기재 필수

### Phase 6: EE2 방정식 유도 + τ=H/H₀ 정당화 (8인, 8회)

**목표**: EE2가 어떤 작용(action)에서 유도되는가? 모듈러 매개변수 τ와 H의 동일시 이론적 근거.

**항목**: [#10] [#11]

**제공 정보 (최소)**:
- EE2 ω_DE = OL0·(1 + A·(1 - cos(B·ln(H/H₀))))
- L22 확립: A = 2·nome(τ=i). B = 2π/ln2. S_cl = π.
- SL(2,R)/Z₂ = PSL(2,R): AdS₂ 등거리 변환군 (Run 26)
- dS/AdS 대응: near-horizon de Sitter ↔ AdS₂

**질문 (방향만)**:
"ω_DE = OL0·(1 + A·(1 - cos(B·ln H)))를 EOM(운동방정식)으로 갖는 작용은 어떤 형태인가?
 cos(B·ln H) 구조는 비최소 결합(non-minimal coupling), k-essence, ghost condensate 중 어느 것과 가장 가까운가?
 τ = H/H₀ 동일시가 가능하려면 τ 공간과 H 공간이 어떻게 동형이어야 하는가?
 AdS₂/dS 대응에서 τ(AdS₂)가 H에 어떻게 매핑되는가?"

**수렴 기준**:
- 작용 S[φ] 명시 + EOM에서 EE2 재현: "EE2는 유도된 이론" → 논문화 가능성 A로 상향
- 작용 미발견: "EE2는 가정" 명시 → 논문 한계 섹션
- τ=H/H₀ 정당화 성공: A 등급 전체 B+→A
