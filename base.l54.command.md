# L54 시뮬 사양

## 목적

L53 결과: σ_0(k) Model IV에서 T17 H0 tension과 T17 SC를 동시에 만족하는 영역 부재 (구조적 불가능성 확인).
- T17 SC 요구: σ_eff_T17 = σ_sc = 1.17×10⁸ (k=1e-4에서)
- T17 H0 요구: σ_eff_T17 ≈ 2.34×10⁸ (같은 k=1e-4에서)
- 단일 k 스케일에서 두 조건 동시 만족 불가

**L54 목적**: σ_0(z) Model V 채택 — z=0 (현재)과 고z를 분리하여 4 검증 동시 만족 가능 여부 결정.
- σ_now (z=0): T17 SC 조건 ≈ σ_sc = 1.17×10⁸ 담당
- σ_late (z>>z_t): T17 H0 tension 담당
- z_t: 전환 적색이동 (0.3~5.0)

**이 실험이 이론의 마지막 시도**: PASS → 이론 생존, FAIL → 이론 사망.

## 모델 (Model V)

```
σ_0(z) = σ_now + (σ_late - σ_now) × 0.5 × (1 + tanh((z - z_t) / Δz))
Δz = 0.5 (고정)
```

- z=0: σ_0(0) ≈ σ_now (T17 SC 조건)
- z >> z_t: σ_0 ≈ σ_late (T17 H0 tension 담당)

σ_UV = 3.31×10⁹ (L49 고정), k_t = 0.68 Mpc⁻¹ (L52 고정)

## 핵심 검증 (4 조건)

```
T17 SC          : τ_q_now = σ_now/(4πG), τ_q_now·3H₀=1 검증
                  χ²_sc = ((τ_q_now·3H₀ - 1) / 0.1)²

T17 H0 tension  : SQT ODE (σ(z) 시간 의존) → H(a) → 음향지평선/시선거리 비율
                  H₀_CMB_apparent = H₀_true × (r_s_SQT/r_s_LCDM) × (χ_LCDM/χ_SQT)
                  χ²_T17 = ((H₀_CMB_app - 67.4) / 0.5)²

T20 σ_8         : 성장 ODE — friction_factor(z) = 1 + 0.1×log10(σ(z)/σ_ref) 시간 의존
                  χ²_T20 = min(((σ_8 - 0.811)/0.006)², ((σ_8 - 0.76)/0.02)²)

T22 a_0 (z=0)  : σ_eff_T22 = σ_UV (고정=3.31×10⁹) → χ²_T22 ≈ 0 (항상 통과)
                  χ²_T22 = ((a₀_pred/a₀_obs - 1) / 0.1)²

합산: χ²_total = χ²_T17 + χ²_sc + χ²_T20 + χ²_T22
```

## SQT ODE (T17 H0)

```
dn/da = (Γ₀ - 3Hn - σ(z)·n·ρ_m) / (a·H)
dρ_m/da = (-3H·ρ_m + σ(z)·n·ρ_m·ε/c²) / (a·H)
H = √(8πG·ρ_tot/3),  ρ_tot = ρ_m + ρ_n,  ρ_n = n·ε/c²
ε = 3H₀²·Ω_Λ·c² / (8πG·n₀)   [z=0 자기무모순]
Γ₀ = 1.0 고정
```

## 성장 ODE (T20 σ_8)

```
δ'' + ff(z)·(3/a + H'/H)·δ' - (3Ω_m)/(2a⁵E²)·δ = 0
ff(z) = 1 + 0.1·log10(σ_0(z)/σ_ref),  σ_ref = 1.17×10⁸
E² = Ω_m/a³ + Ω_Λ
```

## 그리드

- σ_late: [10^7.5, 10^10],  15점 로그 등간격  (H0 tension 탐색 범위)

## 통과 기준

```
PASS   : χ²_total < 16  (4 검증 동시 ~2σ)  → 이론 생존
PARTIAL: 16 ≤ χ²_total < 50               → 조건부
FAIL   : χ²_total ≥ 50                    → 이론 사망
```
