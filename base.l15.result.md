## 최종 후보 분류

### 1순위: 이론+데이터 균형

**AX3 (F-C)**: `omega_de = OL0*(1+A*(1-cos(B*(Om*(1+z)^3-Om)/OL0)))`
- A1 공리 직결 (물질 밀도 직접 의존)
- chi2_bao=7.55 (BAO 최우수)
- MCMC acc=0.067 (신뢰 가능)
- w0=-0.763, wa=-0.673

**HB3 (F-L)**: `omega_de = OL0*(1+A*(x_m^2/(B^2+x_m^2)^2 - norm))`
- chi2_MAP=-33.68 (2위)
- MCMC acc=0.089 (최우수)
- w0=-0.814, wa=-0.558

### 2순위: 데이터 fit 우수, MCMC 재검증 필요

**TF3 (F-G)**: `omega_de = OL0*(1+A*(E-1)*exp(-B*(E-1)^3))`
- chi2_MAP=-34.10 (1위) — 하지만 acc=0.009
- w0=-0.831, wa=-0.488

**EE2 (F-CL)**: `omega_de = OL0*(1+A*(1-cos(B*ln(E))))`
- chi2_cmb=0.71 (CMB 완벽 정합)
- w0=-0.858, wa=-0.360 (거의 ΛCDM wa)

### 주의 대상

**ST1 (F-W)**: `omega_de = OL0*exp(-A*((1+z)^3-1))*(1+B*(1+z)^3)/(1+B)`
- w0=-0.535 (약한 DE), wa=-1.624 (비표준적)
- BAO chi²=27.49 (ΛCDM 24.4보다 높음 — BAO를 희생하고 SN을 이김)
- 물리적 해석 어려움. 논문에서 별도 취급 권고.

## 한계 및 주의사항

1. **MCMC 공식 미통과**: TF3(acc=0.009), ST1(acc=0.005) — posterior CI 신뢰 불가. MAP만 참고.

2. **이론-현상 연결 미완성**: A1+A2 공리로부터 AX3 수식이 자연스럽게 유도되는 계산이 없음. B 파라미터의 Planck 스케일 기원 미해석.

3. **DR3 독립 검증 대기**: L15 결과는 DESI DR3 공개 후에야 真 독립 검증 가능.

4. **SN chi² 구조 확인 필요**: chi2_sn이 모델 간 거의 동일 (~1640-1646). BAO가 모델 간 차이를 주도함.

5. **ST1 BAO 악화**: ST1은 BAO chi²=27.49로 ΛCDM(24.40)보다 나쁨. SN 개선이 BAO 악화를 상쇄한 것. 물리적 의미 약함.
