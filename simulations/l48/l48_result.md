## 1. 실험 개요

### 목적
SQT 재구축 III 이론의 3변수 연성 ODE 시스템을 사용하여 σ₀ × Γ₀ 파라미터 그리드를 탐색하고,
- **T17_full**: H₀ tension (H₀_local=73.8 vs H₀_CMB=67.4 km/s/Mpc) 동시 만족 영역 탐색
- **T20**: sigma8 성장인자 예측
- 자기일관성 조건 (τ_q·3H₀=1) 검증

### 이론 (SQT 재구축 III)
**ODE 시스템** (스케일인자 a를 독립변수로 변환):
```
dn/da    = [Γ₀ - 3H·n - σ₀·n·ρ_m] / (a·H)
dρ_m/da  = [-3H·ρ_m + σ₀·n·ρ_m·ε/c²] / (a·H)
H²       = (8πG/3)·(ρ_m + n·ε/c²)
```

**도출 관계식**:
```
τ_q = σ₀ / (4π·G_N)          [매질 고유 시간]
n_∞ = Γ₀ · τ_q               [평형 양자 밀도]
ε   = 3H₀²·Ω_Λ·c² / (8πG·n_∞) [양자 에너지, Λ_eff 일치]
자기일관성: τ_q · 3H₀ = 1
```

**H₀ tension 메커니즘**: SQT가 H(z)를 수정 → D_A(z_*) 변화 → CMB-apparent H₀ 변화  
`H₀_CMB_apparent = H₀_true × (r_s_SQT/r_s_LCDM) × (D_A_LCDM/D_A_SQT)`
