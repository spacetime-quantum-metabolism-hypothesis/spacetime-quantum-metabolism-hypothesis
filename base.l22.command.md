| 항목 | L21 최종 등급 | 핵심 미해결 |
|------|-------------|-----------|
| B = 2π/ln2 | C+ | s=π/ln2 순환. H_ref 미정의 (가장 치명). |

## L22 Phase 구조

### Phase 1: MCMC — 이론 생사 판정 (8인, 10회)

**목표**: A, B를 자유 파라미터로 DESI DR2 BAO 13pt에 피팅.
데이터가 B=9.065를 독립적으로 선택하는가?

**제공 정보 (최소)**:
- EE2 공식: ω_DE = OL0·(1 + A·(1 - cos(B·ln(H/H₀))))
  (H/H₀ 정규화 명시 — Run 31 결론 반영)
- DESI DR2 BAO 13pt: 공식 CobayaSampler bao_data 사용
- 비교 기준: AICc(ΛCDM) ≈ 21.4, AICc(EE2 고정) = 10.8
- 이론 예측값: A_theory = 2e^{-π} ≈ 0.0864, B_theory = 2π/ln2 ≈ 9.065

**질문 (방향만)**:
"A, B를 자유 파라미터로 풀었을 때 사후 분포(posterior)의 피크는 어디인가?
 B의 68% 신뢰구간에 9.065가 포함되는가?
 데이터가 B를 독립적으로 고른다면 이것이 이론에 무엇을 의미하는가?
 고르지 않는다면?"

**금지**:
- B=9.065을 사전 분포(prior)에 강하게 반영하는 피팅 금지
- B에 대해 의도적으로 좁은 prior 설정 금지
- 결과를 이론에 유리하게 보고하는 시각화 금지

**수렴 기준**:
- B 사후 피크가 9.065±0.5 이내: 이론 지지 → B 등급 C+→B 상향
- B 사후 피크가 9.065와 2σ 이상 불일치: "B는 관측 결정 파라미터" 선언 → 등급 D로 하향
- 사후 분포 평탄(uninformative): 데이터 변별력 부족 → C+ 유지

**코드 요구사항**:
- MCMC: emcee 또는 dynesty
- BAO 데이터: CobayaSampler 공식 파일 (CLAUDE.md 규칙 준수)
- 4인 코드리뷰 필수 (데이터 로딩, 수식 구현, 공분산 처리, 최적화 루틴)
- 결과: corner plot + marginal 1D 분포 + AICc/Bayesian log-evidence 수치

### Phase 2: H_ref 고정 — AdS₂ 고정점 독립 유도 (8인, 10회)

**목표**: cos(B·ln(H/H*))에서 H*를 이론 내부에서 결정하는 물리적 원리 발굴.

**제공 정보 (최소)**:
- Run 31 발견: z=0~3 위상 변화량 Δ(B·ln H) ≈ 18.8 ≈ 3×2π — 단위 독립적
- Run 31 미해결: 절대 위상 φ₀ = B·ln(H₀/H*)는 H* 없으면 미결정
- SL(2,ℝ)/Z₂ = PSL(2,ℝ): AdS₂ 등거리 변환군 (Run 26)
- AdS₂ 반지름: ℓ_AdS = 1/H (현재 물리적 동일시)
- dS/AdS 대응: near-horizon de Sitter ↔ AdS₂

**질문 (방향만)**:
"H*를 이론이 내부적으로 고정할 수 있는 원리는 무엇인가?
 AdS₂의 어떤 불변량이 H의 기준 스케일을 결정하는가?
 H*가 CMB 또는 BAO 스케일과 독립적으로 연결될 수 있는가?
 H*를 고정하지 못한다면, 이 이론은 어떤 관측 가능 예측을 여전히 할 수 있는가?"

**금지**:
- H* = H₀ (오늘날 허블 상수)를 기준으로 단순 정규화하는 것은 외부 관측 의존이므로, 이것이 이론적으로 필연적임을 증명하는 논증 요구 (단순 선언 금지)

**수렴 기준**:
- H*의 이론 내부 유도 성공 + 계산 제시: B 등급 → B+ (H_ref 해소)
- 유도 실패 + H/H₀ 정규화의 이론적 필연성 논증: C+ 유지 (부분 해소)
- 유도 실패 + 필연성 논증도 실패: "H_ref는 외부 입력 — EE2는 예측 이론 아님" 선언

### Phase 3: P3 판정 — Z₂ 게이지/글로벌 구분 (8인, 8회)

**목표**: dS₄/Z₂의 Z₂가 게이지 대칭인지 글로벌 대칭인지 판정.
이것이 A₂=2의 최후 장벽이다.

**제공 정보 (최소)**:
- Run 32 확립:
  - Z₂ 글로벌 → 선형합 Z[α]+Z[β]=2Re[Z[α]] → A₂=2
  - Z₂ 게이지 → superselection → 제곱합 2|Z[α]|² → A₂=2 붕괴
- CPT는 반유니타리 연산자. CP 위반 현상에서 입자-반입자 간섭이 실제 관측됨.
- dS₄/Z₂의 Z₂ 작용: 반대점 동일시 (antipodal map)

**질문 (방향만)**:
"dS₄의 antipodal Z₂는 게이지 대칭인가, 글로벌 대칭인가?
 두 스핀 구조 α, β 사이를 잇는 물리적 과정(경로)이 존재하는가?
 CPT가 이 Z₂를 구현한다면, CPT의 반유니타리성이 superselection을 막는가?
 관측 가능한 물리량이 두 스핀 구조의 간섭항에 의존하는 예가 있는가?"

**수렴 기준**:
- Z₂ 글로벌 판정 + 논증: A₂ B→A (논문 수록 가능)
- Z₂ 게이지 판정 + 논증: A₂=2 붕괴 → D등급으로 하향, 대안 경로 탐색 시작
- 판정 불가 (원리적 애매성): B 유지, "P3 미결" 명시

### Phase 4: 자기일관 ODE 수치해 + Bayesian 증거 (8인, 7회)

**목표**: EE2가 수학적으로 자기일관한 우주 모델임을 수치로 확인하고, ΛCDM 대비 통계 우위를 엄밀하게 정량화.

**제공 정보 (최소)**:
- Friedmann 방정식: H² = (8πG/3)(ρ_m + ρ_DE)
- EE2 연속 방정식: ρ̇_DE + 3H(1 + ω_DE(H))ρ_DE = 0
  여기서 ω_DE(H) = OL0·(1 + A·(1 - cos(B·ln(H/H₀))))
- Run 35 확립: 물리적 H 영역에서 Picard-Lindelöf 해 존재성 증명 완료
- Run 35 미완: 수치해 미생성, 안정성 임계 확정 못함
- 수치 파라미터: Ω_m=0.315, Ω_DE=0.685, H₀=67.4 km/s/Mpc

**질문 (방향만)**:
"H(z) 곡선은 어떻게 생겼는가? ΛCDM H(z)와 편차가 언제 어디서 최대인가?
 ω_DE(z)의 실제 진동 진폭과 위상은 얼마인가? 관측 오차 내에 있는가?
 Bayesian log-evidence log K에서 EE2 vs ΛCDM의 차이가 AICc보다 크거나 작은가?
 안정성: 섭동이 성장하는가 감쇠하는가?"

**요구 결과물**:
- H(z), ω_DE(z) 수치 그래프 (z=0~3)
- Bayesian log-evidence (dynesty 또는 MultiNest 권장)
- 유령 안정성 c_s² 확인

**코드 요구사항**:
- CLAUDE.md E(z) 계산: 반드시 coupled ODE solver (odeint) 사용
- 수치 진동 제어: B·ln(H/H₀) 빠른 변화 구간 보폭 제어
- 4인 코드리뷰 필수

**수렴 기준**:
- H(z) 자기일관 수렴 + ω_DE(z) 그래프 + log K 계산: R4 완료 → 논문화 필수 요건 충족
- 수렴 실패 (ODE 폭주): 버그 먼저 의심, 4인 코드리뷰 후 재실행 (CLAUDE.md 규칙)

**목표**: Phase 1-4 결과를 반영하여 논문 초안의 각 주장에 "논문 수록 가능 / 불가" 판정.

**판정 항목**:

| 주장 | 판정 기준 |
|------|---------|
| B = 2π/ln2 이론 도출 | MCMC에서 B=9.065 재현 시 가능 |

| S_cl = π | 현재 B-등급 → 가능 (n=0 정당화 추가 필요) |

| Bayesian 증거 | Phase 4 완료 시 가능 |

**결과물**:
