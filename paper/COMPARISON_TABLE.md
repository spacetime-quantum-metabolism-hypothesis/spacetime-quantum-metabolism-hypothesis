# L450 — Modified Gravity Head-to-Head Comparison (Paper Sec X)

Six modified-gravity / dark-sector candidates vs SQT (Spacetime Quantum Metabolism), evaluated on six independent observational / theoretical criteria. Each cell: PASS / PARTIAL / FAIL with one-sentence honest justification.

Criteria
- **Λ origin**: does the framework derive (not insert) a non-zero late-time vacuum-like component?
- **MOND a_0**: does it reproduce the empirical galactic acceleration scale a_0 ≈ 1.2×10⁻¹⁰ m/s² without per-galaxy tuning?
- **Bullet cluster**: consistent with collisionless lensing offset from baryonic gas peak (1E 0657-558)?
- **BBN**: preserves standard primordial abundances (no |ΔN_eff| > 0.3, no H(z~10⁹) shift)?
- **Cassini**: PPN |γ−1| < 2.3×10⁻⁵?
- **S_8**: alleviates KiDS/DES low-z S_8 tension (≥1σ pull toward data) without breaking CMB?

Verdict legend: PASS = quantitatively consistent; PARTIAL = consistent in restricted regime or with extra assumption; FAIL = structural conflict or requires fine-tuning beyond ε~10⁻³.
