## 1. Padmanabhan emergent gravity (root §5.1)

- root claim: `dV/dt = L_P²(N_sur − N_bulk)` 가 SQT 대사 연속방정식과 동형 → Friedmann 재현.
- paper/base.md 의 어느 axiom/foundation 과 동형?
  - 후보: axiom 1+3 (흡수−생성 균형) + Holographic pillar (σ₀=4πG·t_P).
  - paper/base.md 자체에는 Padmanabhan, holographic *equipartition*, surface/bulk DOF, CosMIn 어느 것도 **명시되지 않음**. Holographic pillar 는 σ₀ 차원 scaling 만 다루며 Padmanabhan 의 N_sur−N_bulk 식은 도출 경로에 존재하지 않음.
- 동형의 성격: **trivial 가능**. dV/dt = (sources − sinks) 형태는 모든 dynamical-DOF 우주론 모델에서 발생. SQT 의 (axiom 1, axiom 3) 균형식이 Padmanabhan 형태로 *재배열* 가능한지는 paper/base.md 에서 보이지 않았음.
- 명시 도출 vs 사후 매칭: **사후 매칭**. paper §2.2 derived 2 (n_∞ 정상상태) 가 Padmanabhan 식의 zero-mode 와 닮았다는 것은 가능하나, paper/base.md 어디에도 N_sur, N_bulk, surface equipartition 으로 환원되는 단계가 없음.
- 판정: **PARTIAL** (motivational analogy 수준; paper/base.md framework 에서 *구조적 동형* 이 입증된 적 없음).

## 2. Volovik condensed matter analogue (root §5.2)

- root claim: 드 시터 = 열적 배스 T=H/π, 2-fluid 제2음파 = 무질량 spin-2 graviton, λ=0 평형.
- paper/base.md 의 어느 axiom/foundation 과 동형?
  - 후보: axiom 4 (emergent metric) + axiom 6 (선형) + Z₂ SSB pillar (BEC-like 응축).
  - paper/base.md 어디에도 "superfluid", "second sound", "two-fluid", "Volovik", "He-3" 명시 없음. Z₂ SSB pillar 는 η ≲ 10 MeV 제약만 진술; BEC 와 condensed matter analogue 로 연결되는 단계 부재.
- 동형의 성격: **trivial**. SSB → Goldstone/Higgs 모드 → 집단여기 가 spin-2 transverse mode 에 닮는다는 관찰은 모든 SSB-기반 emergent gravity (Volovik, Wen, Xu) 공통이며 SQT 특이 예측 아님.
- 명시 도출 vs 사후 매칭: **사후 매칭**. paper/base.md 의 axiom 4 자체가 "OPEN" 으로 표지되어 있고 (§2.5), Volovik analogue 는 5번째 축 후보 목록에 *포함되어 있지 않다* — paper 가 채택한 후보는 Causet meso 단 하나.
- 판정: **NOT_INHERITED** — paper/base.md framework 가 Volovik program 을 명시적으로 상속받지 않음.

## 3. Causal Set theory (Sorkin, Dowker) (root §5.3)

- root claim: 시공간 원자의 순차 성장; Sorkin 1990 의 δΛ ∼ 1/√N 사전예측.
- paper/base.md 의 어느 axiom/foundation 과 동형?
  - paper §2.5 5번째 축 후보 **Causet meso** (4/5 조건부 PASS). Q10 (line 762–763) 에서도 axiom 4 의 미시 후보 중 하나로 causal set 명시.
- 동형의 성격: **부분적으로 unique**. Causet meso 를 paper/base.md 가 axiom 4 의 *유일한 명시 후보* 로 채택했다는 점에서 5 claim 중 유일하게 framework 내부 link 가 살아있음.
- 명시 도출 vs 사후 매칭: 대부분 **사후 매칭**. paper/base.md 는 Causet 을 *후보로 거론* 할 뿐, axiom 1–6 → causal-set growth → δΛ∼1/√N 을 단계별로 도출하지 않음. derived 2 (n_∞ 정상상태) 의 ρ_Λ 도출은 §5.2 에서 *circular* 임을 자체 인정 (line 528, 655).
- 판정: **PARTIAL** — 5 program 중 paper framework 와 가장 가까운 link. 그러나 "구조적 동형 입증" 까지는 미달.

## 4. Jacobson thermodynamic gravity (1995) (root §5.4)

- root claim: δQ=TdS 에서 Einstein 방정식 도출 (parameter 0개).
- paper/base.md 의 어느 axiom/foundation 과 동형?
  - 후보: Holographic pillar + Schwinger-Keldysh pillar (KMS = thermal).
  - paper/base.md 의 SK pillar 는 retarded propagator 의 상반평면 해석성 (causality) + KMS 검증 (PASS) 로만 진술됨. **Clausius/horizon entropy/area thermodynamics 도출 단계 부재**.
- 동형의 성격: **trivial**. KMS 는 모든 finite-T QFT 의 정의 조건이며 Jacobson 의 *horizon* thermodynamic argument 와 동치가 아님. SQT 의 KMS PASS 는 *propagator 수준* 일관성 체크일 뿐 Einstein eq 도출 경로 아님.
- 명시 도출 vs 사후 매칭: **사후 매칭**. paper/base.md derived 1 (Newton G 회복) 은 depletion-zone gradient 로 진행하지 (axiom 1+axiom 4), Clausius 적분으로 가지 않음.
- 판정: **NOT_INHERITED** — Jacobson 도출 경로는 paper framework 안에 존재하지 않음.

1. paper/base.md 본문 1192 줄에서 Padmanabhan / Volovik / Jacobson / GFT 4 키워드는 **단 한 번도 등장하지 않는다**. Causal set 만 axiom 4 5번째 축 후보로 1 회 인용.
