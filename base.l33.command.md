## ■ SQT 이론 구조

### 공리 2개

- **A1**: 물질은 시공간 양자(SQ)를 소멸시킨다.
- **A2**: 빈 공간에서 SQ가 생성된다. 생성 = 공간 팽창의 미시적 원인.

### 제약 3개

- **C1 (로렌츠 불변성)**: 소멸은 에너지-운동량 텐서 대각합에 비례. 복사 자동 탈결합.
- **C2 (공변성)**: 생성은 좌표계 무선호, 공변 부피당 평균률.
- **C3 (에너지-운동량 보존)**: 총 에너지-운동량 국소 보존.

### 장 변수

ψ: SQ 밀도의 정규화 스칼라장 (무차원)
- ψ=1: 평형 진공
- ψ 감소: 물질 밀도 증가 (고적색이동) 방향

### 핵심 SQT 구조

- 소멸률 ∝ 물질밀도 × SQ밀도
- 생성률 ∝ (최대밀도 - SQ밀도)
- 평형 ψ*(z): 물질 분율과 암흑에너지 분율의 비로 결정
- Γ_norm: 정규화 소멸률 (소멸-생성 불균형 척도)

## ■ L33 핵심 구조: 8인 독립 팀

### 팀 구성 원칙

- 인원: 8명
- 역할 사전 지정 금지. 중복 탐색 방지만 합의.
- 매 실행마다 각 팀원이 독립적으로 이론 형태 제안.
- 팀원 간 결과 공유 후 중복 제거 → 최종 32개 이론 (팀원당 4개).

### 8인 탐색 방향 (방향만 제시, 수식 절대 금지)

| 팀원 | 탐색 방향 |
|------|-----------|
| A | ψ 감소에 따른 팽창압 증폭 — 거듭제곱 법칙 계열 |
| B | 로그 엔트로피 기반 — SQ 밀도 로그 편차 계열 |
| C | 포화 구조 — 고z에서 자연 포화되는 함수 계열 |
| D | 선형 하이브리드 — A·B·C 결합의 가중 선형 합 |
| E | 적색이동 의존 결합 — 결합 강도가 z에 따라 변하는 구조 |
| F | 지수 감쇠 접근 — 지수함수 기반 포화 계열 |
| G | 분수 거듭제곱 — 비정수 지수 탐색 계열 |
| H | 이중 채널 — ψ 감소 + Γ 보정 이중 구조 결합 |

### 토의 규칙

- 각 팀원은 자신의 방향에서 4개 이론 독립 제안.
- 제안 후 팀 전체 비교 → 동일 함수형 제거.
- 최종: 32개 (팀원당 최소 3개 보장).
- **매 실행마다 새로운 파라미터 샘플링** → 실행마다 다른 이론 집합.

## ■ 암흑에너지 상태방정식 추출

w(z) = -1 - (1/3)·d ln ρ_DE / d ln(1+z)

CPL 매개화: w(z) ≈ w₀ + wa·z/(1+z)

z ∈ [0.01, 2] 최소자승으로 w₀, wa 산출.
