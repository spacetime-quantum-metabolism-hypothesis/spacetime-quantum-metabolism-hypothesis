## 결론 (Conclusions)

1. **데이터**: DESI+Planck+SN은 wₐ<0를 강하게 지지. A12가 최우수 0-파라미터 프록시.

2. **스케일 분리**: σ = 4πGt_P는 배경 우주론에서 62자리 무시가능. SQMH 배경 ≡ ΛCDM. 이것은 이론 결함이 아니라 SQMH가 국소(플랑크 스케일) 메커니즘임을 의미한다.

3. **역유도 실패**: 세 후보 모두 SQMH 역유도 불가. 근본 이유: 고전 다크에너지에는 t_P가 없다. 역유도 실패 ≠ 이론 기각.

4. **논문 전략**: JCAP 타깃. "QG-동기 현상론 프레임워크"로 포지셔닝. A12 = 현상론 프록시, C11D/C28 = 독립 이론으로 병기.

5. **미래**: SKAO 21cm (2030년대)에서 후보 간 구별 가능 (SNR 12~14). 섭동 레벨 SQMH 연결은 L9+ 과제.
