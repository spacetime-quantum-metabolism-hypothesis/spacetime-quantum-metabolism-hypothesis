# L356 NEXT_STEP — Λ_UV = 18 MeV 후속 작업

## 정직 한국어 한 줄

## 즉시 후속 (priority 순)

### N1. SQT β 함수 명시 도출
- 자유도: 양자 number density n, transfer 결합 μ (또는 결합 곱 n₀μ).
- 방향만: Wilsonian effective action 으로부터 1-loop β-functional 도출.
- 산출: β_n(μ_RG), β_μ(μ_RG) — IR (cosmological) → UV (18 MeV) 흐름이 폭주 / fixed point / Landau pole 중 어느 것인지.
- 통과 기준: 18 MeV 근처에서 결합이 *발산하지 않고* 자연스럽게 끝나는지.

### N2. 1-loop 자기에너지 명시 계산
- Λ_UV cutoff 의존성을 ln(Λ_UV/μ_IR) 로 흡수 가능한지 (renormalizable EFT 조건).
- 위반 시 (power-law divergence): cutoff-sensitivity → I3 PASS 가 사실 fragile 할 수 있음.

### N3. 핵물리 / 저에너지 정밀 실험 비교
- 18 MeV 근방 영역의 표준모형 정밀 검증 데이터:
  - 핵 γ-ray spectroscopy (수 MeV 영역 nuclear transition)
  - 저에너지 e+e− 산란 (Mainz MAMI, Jefferson Lab)
  - 5th force / fifth-force 한계 (sub-fm 영역)
- SQT dark-only 결합이 이들 baryonic 실험에 *얼마나 누설* 되는지 정량 한계.

### N4. UV completion 후보 매칭
- LQG / asymptotic safety / pre-geometric 후보가 *Planck → 18 MeV* dimensional reduction 으로 자연 cutoff 를 만드는 메커니즘 제안 / 거부.
- 18 MeV 의 *수치적 자연성* 정당화 가능 여부.

### N5. 논문 표기 정정
- 논문 본문에서 "Λ_UV = 18 MeV" 인용 시 다음 단서 명시 의무:
  - "Definitional cutoff from ℏc/d_inter-quantum, not RG-derived"
  - "Cosmological observables shown cutoff-insensitive (I3)"
  - "Microscopic justification (β function, UV completion) currently open"
- 현재 표기 "physical sub-fm cutoff" 는 *부분적으로 misleading* — 정정 권고.
