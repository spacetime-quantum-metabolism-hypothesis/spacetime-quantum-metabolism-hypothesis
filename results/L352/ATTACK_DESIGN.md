# L352 ATTACK DESIGN — RG β(σ) 의 b 계수 1-loop first-principle 도출 시도

## 배경 (L334 결과 인용만)
- L334 pillar 4 평가: ★★ — RG 형태 β(σ) = a σ − b σ² + c σ³ 자체는 phenomenologically motivated, 그러나 b, c 계수의 a priori (실측 fit 이전) 도출 경로가 비어 있다는 약점 기록.
- L352 의 임무: 그 빈 칸 중 **b 계수만** 1-loop 수준에서 first-principle 로 채울 수 있는지 *탐색*.
- 본 문서는 [최우선-1] 에 따라 어떤 수식·계수 값·유도 경로 힌트도 포함하지 않는다. 8인 팀이 처음부터 자유롭게 유도해야 한다.

## 공격 대상
- **계수 b**: β(σ) 전개의 σ² 차수 계수. 부호·크기·다른 양과의 관계를 미시이론에서 결정 가능한가.

## 입력 방향 (이름만 — 지도 아님)
1. **n 필드 self-coupling**: SQMH n 장의 자기상호작용 항이 RG 흐름에 기여한다는 사실만 인정. 형태·차수·계수는 팀이 자유 도출.
2. **G_N 결합 채널**: 중력적 결합 (Newton 상수 G_N) 이 σ 의 RG 흐름에 들어오는 *한 가지* 채널이라는 점만 명시. 결합 형태 미지정.
3. **1-loop 수준**: 다이어그램 차수는 1-loop 로 한정. higher-loop 는 본 세션 범위 밖.

## 금지 사항 (최우선-1 직접 적용)
- L14, L22, L334 등 과거 세션의 β 함수 명시적 형태를 가져와 "개선" 하는 행위 금지.
- 어떤 수치 ("b ≈ ...", "b 의 부호는 +") 도 Command 단계에서 미리 적시 금지.
- 특정 regularisation scheme (MS, dim-reg, cutoff 등) 을 사전 선택 금지 — 팀이 정당화 후 선택.
- "이 다이어그램을 계산하라" 식의 다이어그램 지정 금지.
- universality / scheme-independence 여부도 팀이 판정.

## 팀 구성 (역할 사전 지정 금지)
- **8인 이론팀**: 인원수만 지정. 분업은 토의 중 자연 발생. 서로 중복되지 않게 자유 접근.
- **4인 코드리뷰팀**: 만일 1-loop 적분이 수치/symbolic 코드로 검증되면, 별도 4인이 자율 분담 검토. 역할 사전 배정 없음.

## 산출 기준
- **PASS (b 도출 성공)**: 1-loop 다이어그램 집합으로 b 가 다른 (이미 알려진) 상수의 함수로 표현되며, scheme 종속성·universal 부분 분리가 명확.
- **PARTIAL**: b 의 부호 또는 크기 한계 (bound) 만 도출.
- **FAIL**: 1-loop 수준에서 first-principle 도출 불가능함이 구조적으로 드러남 (예: 추가 sector 필요).
- 결과는 정직하게 NEXT_STEP.md, REVIEW.md 에 기록. base.md 와 충돌하면 base.fix.md 에 별도 기록 (CLAUDE.md 재발방지 항).

## 과적합 방지
- 만약 b 도출이 추가 보조 파라미터 (예: 새 cutoff, 새 결합) 도입을 요구하면, AICc 패널티 관점에서 "단순 fit (b 자유) vs first-principle (b 고정 + 보조파라미터)" 비교를 명시.
