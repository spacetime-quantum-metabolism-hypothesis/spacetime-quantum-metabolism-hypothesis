# L207 — 8-team attack: T^{μν}_n stress-energy tensor explicit form

## Attack
A1: "n field 의 T^{μν} 명시 부재. fluid approximation 회피."
A2: "n 은 scalar? vector? tensor? Lagrangian L[n] 에서 직접 도출 필요."
A4: "Cosmological perturbation 에 어떻게 들어가는지 — δT^{μν}_n 형식 미제시."
A5: "Bianchi identity ∇_μ T^{μν}_n = source(matter) — explicit covariant form 필요."

## 가장 강력한 vector
A1 + A5: **T^{μν}_n 명시 + ∇_μ T^{μν} = source 검증**

## 권고
스칼라 fluid approx: T^{μν}_n = (ρ_q + p_q) u^μ u^ν + p_q g^{μν}, ρ_q = nε/c², p_q = -ρ_q (Λ-like)
- pressure → -ρ 이면 dark energy 행동
- 다른 regime 에서 p_q ≠ -ρ_q 이면 dark matter-like
- σ_0 ρ_m n source term: continuity 검증

## 다음
시뮬: T^{μν}_n 의 ∇_μ T^{μν}_n = - ∇_μ T^{μν}_m (energy exchange) 정량 검증.
