## §1 4-pillar 별 시간 의존도 표

| Pillar | 시간이 등장하는 위치 (정성) | 시간 의존도 등급 | 비고 |
|---|---|---|---|
| **SK (Schwinger-Keldysh)** | closed-time-path 의 정의 자체, retarded/advanced 분기, time order 명시 | **High** | "time order" 가 구조의 본체 |
| **Wetterich RG flow** | RG scale 변수 (=ln k) 를 RG "time" 으로 호명 | **High (명목)** / Low (물리적) | 명목상 time 이지만 abstract scale parameter |
| **Holographic (σ₀=4πG·t_P 형태)** | Planck time 이 length scale 의 짝으로 등장 | **Medium** | 시간 자체가 동역학 변수 아님, scale 상수 |
| **Z_2 SSB** | thermal phase transition, domain wall dynamics | **Medium** | 시간 흐름은 phase transition 이력에서만 등장 |

## §2 Time emergent 충돌 등급

| Pillar | 1차 충돌 강도 | 재해석 후 충돌 강도 | 핵심 충돌 지점 |
|---|---|---|---|
| SK | High | Medium | "time order" 를 잃으면 retarded vs advanced 구분이 모호해짐 |
| Wetterich | Low | Low | RG "time" 은 이미 물리적 시간이 아님 |
| Holographic | Low | Low | t_P 는 length 의 동반 scale 로 흡수 가능 |
| Z_2 | Medium | Low | dynamics 가 빠지지만 topology 는 남음 |

→ 가장 큰 충돌은 SK pillar. 나머지 3 은 거의 자연 정합.

## §3 각 pillar 의 재해석 가능성

### 3.1 SK
- 방향: time order → causal/count order 로 재해석.
- 즉, "시간 순서" 대신 "상호작용 횟수 순서" 또는 "정보 흐름 부분순서" 로 SK 구조의 두 분기를 재정의.
- 재해석이 가능하면 SK 의 핵심 (분기 + 인과 분리) 은 보존.

### 3.2 Wetterich
- 방향: RG scale 을 "관측 거칠기" 로 그대로 둠.
- Time emergent 시에도 RG flow 의 abstract 성격은 변하지 않음.
- 재해석 거의 불필요.

### 3.3 Holographic
- 방향: t_P 를 l_P 와의 짝 scale 로 축약, 시간 변수 의존을 length 변수로 흡수.
- 재해석 가능, 영향 작음.

### 3.4 Z_2
- 방향: thermal phase transition → topological transition (static).
- domain wall dynamics → static topological defect 분류.
- 재해석 가능.

## §4 결합 가능성 (L610 A3 + Time emergent + 4-pillar)

- L610 A3 paradigm shift 가 time emergent 를 핵심 축으로 채택할 경우:
  - SK pillar: count/causal order 재정의 채널로 흡수.
  - Wetterich pillar: 손대지 않아도 정합.
  - Holographic pillar: scale 짝으로 흡수, 정합.
  - Z_2 pillar: static topology 채널로 재배치, 정합.
- 결론 방향: 4-pillar 모두 time emergent 와 *구조적 비양립* 은 아님. 단, SK 재해석은 명시적 작업이 필요.

---

## §5 Cross-cutting check (L601 A2 SK central angle 과)

- L601 A2 가정: SK 가 가장 일반적, 나머지 3 pillar 가 SK 의 특수 한계.
- L611 결과: SK 가 time 에 가장 강하게 의존.
- 결합 시 위험: time emergent 채택 → SK 가 재정의 부담을 단독으로 떠안음. 그런데 A2 angle 은 SK 를 "중심" 에 둠 → 재정의 실패 시 4-pillar 전체가 흔들리는 단일 실패점 형성.
- 정성적 권고 방향: A2 + time emergent 결합은 SK 재해석 (count/causal order) 의 성공 여부에 *전적으로* 의존.

---

## §6 핵심 발견

- 사전 직관: 4-pillar 가 시간을 명시적으로 호명하므로 time emergent 와 큰 충돌 예상.
- 실제 audit: SK 만 강한 충돌, 나머지 3 은 약하거나 명목적 충돌.
- 즉 4-pillar 와 time emergent 의 충돌은 *예상보다 약함*.
- 단, A2 angle (SK central) 과 결합되면 SK 단일 실패점 위험이 증폭.
