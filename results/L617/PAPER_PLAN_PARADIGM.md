## §1. §3.1 axiom 재진술 *방향*

목적: A3+Time emergent paradigm shift 가 채택될 경우, 기존 6 axiom 어휘 체계가 paradigm 과 충돌하지 않도록 재진술 *방향* 을 정리. 실제 axiom 텍스트 변경은 본 문서 범위 외.

- **A3 재정의 방향**:
  - 현행 A3 (Γ₀ emission balance) 의 시간 parameter 의존 어휘 적출
  - "balance" 의 의미를 *dynamical equilibrium* 에서 *boundary-condition state* 로 재진술
  - "emission" 어휘는 유지하되, *temporal flux* 가 아닌 *count-state attribute* 로 해석
  - axiom statement 에서 시간 미분 *언급 자체* 회피 — 단, 미분 형식은 *유도 결과* 로 §3.4 에 등장 가능

- **A5 재정의 방향**:
  - 현행 A5 (Hubble pacing) 의 "pacing" 어휘는 dynamical 어감 — paradigm 과 충돌
  - H₀ 의 역할을 *anchor input* 으로 재진술 (외부 관측 수치 = boundary 조건)
  - "rate" 어휘 retain 시 반드시 *count-rate* 로 재해석 명시
  - count-rate 정의는 axiom layer 가 아닌 derived layer 에 배치 — axiom 에서는 *anchor* 라는 추상적 역할만

- **나머지 4 axiom (A1, A2, A4, A6) 어휘 변경 *방향***:
  - A1, A2, A4, A6 은 paradigm 과 직접 충돌 어휘 미보유 — 변경 약함 (minor wording sync)
  - A4 (locality) 는 paradigm 의 boundary topology 어휘와 정합성 점검만 필요
  - A6 (universality) 는 commonsense 적출 disclaimer 와 cross-link

- **axiom layer 의 정직 disclosure 방향**:
  - A3, A5 재진술이 *post-hoc* 임을 §3.1 footnote 에 명시 *방향*
  - paradigm shift 시도가 axiom 부터 시작된 것이 아니라, phenomenology 결과 누적 후 paradigm 정합화 시도임을 정직 기록

  - 현행: closed-time-path 형식

  - 변경 강도: **중간** (어휘 다수 변경, 수식 inline 어휘만 sync)

  - 현행: RG flow as scale flow (이미 abstract)
  - 재해석 방향: scale 어휘 유지, "flow" 의 dynamical 어감만 정직 disclosure
  - 변경 강도: **약함** (paradigm 과 충돌 적음)

- **Holographic pillar**:
  - 현행: t_P = Planck length scale (이미 length scale)
  - 재해석 방향: time 어휘 잔존 여부 점검만 — 대부분 length 어휘로 이미 정합
  - 변경 강도: **약함**

- **Z_2 (boundary-topology) pillar**:

  - 재해석 방향: dynamics → static topology 어휘 전환
  - "evolution" 어휘 적출, "configuration" 으로 대체 *방향*
