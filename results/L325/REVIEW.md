# L325 — 4인 review (theory-prior vs data-driven anchor)

## 핵심 검증
RG FP 위치 σ*_2 = b/(2c), σ*_3 = b/c. 비율만 의미.
σ_0 = 4πG·t_P 는 dim 단일 스케일. 3 regime 비율과 무관.

## 4인 정직 평가
- P (이론): SQT 미시 Lagrangian 에서 β-function 1-loop 명시 도출 부재. (b, c) 는 effective phenomenology 계수, priori 미확정.
- N (시뮬): L272 false-rate 100% 는 *unconstrained tertile split* 결과. Gaussian prior 적용 시 재측정 필요하나, prior 자체 가 데이터 정보 사용하면 의미 없음.
- O (관측): σ_cosmic=10^8.37, σ_galactic=10^9.56 의 *절대값* 은 BAO/galaxy data 에서 backward 추출. 진짜 priori 아님.
- H (통합): L294 + L301 결합해도 σ_0 (1 자유도) + b/c 비율 (1 자유도, 미정) = 최대 2 자유도 priori. BB 3 regime 는 최소 3 anchor 필요.

## 대수적 핵심
- 미지수: σ_cosmic, σ_cluster, σ_galactic (3개)
- 이론 제약: σ_0 ∝ G·t_P (1식) + RG FP 비율 b/c (1식, 미확정)
- 자유도 부족: 최소 1 anchor 는 데이터로 결정해야 함 → 부분적 post-hoc 불가피

## 결론 (정직)
**현재 상태에서 BB anchors 는 theory-prior 로 완전 고정 불가.**
- L294 + L301 = 부분 prior (σ_0 단일 스케일 + 3-FP 구조)
- 절대 위치 (numerical anchors) 는 data-fit 단계 필요
- L272 100% false-detection 의 mitigation 부분적, 완전 해소 아님
