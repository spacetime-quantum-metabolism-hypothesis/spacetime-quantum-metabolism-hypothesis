## §2 — Top-2 protocol (가장 실현 가능)

**우선순위 1 — Protocol 2 (Negative-control)**
- 본 세션 내부에서 가장 직접 실행 가능
- 다른 결과값 후보군에 *동일 도출 protocol* 을 적용하여 paradigm shift 가 결과 부정확 일반 절차인지 점검
- 회의적 압박: "다른 결과값" 선정이 본 세션 내 의도적 — 회피하려면 결과값 후보군을 사전 무작위 / 망라적으로 고정해야 함
- 통과 기준 *방향*: 다른 결과값에서 paradigm 이 도출되지 *않거나* 다른 paradigm 으로 갈라지면 진정성, 동일 paradigm 이 모든 후보에서 도출되면 cherry-pick 입증

**우선순위 2 — Protocol 1 (Counterfactual axiom)**
- 본 세션 내부 실행 가능하나 axiom 변경 자체가 결과 정보 보유 상태에서 이뤄지므로 meta-postdiction 위험
- 회의적 압박 회피 *방향*: axiom 변경 후보군도 결과 노출 전 사전 등록 시 의미 있음
- 통과 기준 *방향*: axiom 변경 시 paradigm 이 *자동* 변화하면 axiom 의존 진정성, 변화 없으면 axiom 무관 cherry-pick

**Protocol 3, 4 는 위임 / 검증 불가 사유로 본 L613 단독 진행 불가.**

## §3 — 8인 Rule-A 진입 조건 (R3 protocol 통과)

A3+time emergent paradigm shift 가 8인 Rule-A 정식 리뷰에 진입하기 위한 *방향* 조건:

1. **Protocol 2 통과** — negative-control 결과값 후보군 (사전 정의) 에서 동일 paradigm 이 도출되지 않음을 확인
2. **Protocol 1 부분 통과** — counterfactual axiom 변경 시 paradigm 이 함께 변화 (axiom 의존성) 확인
3. **외부 위임 후보 동결** — Protocol 3 위임 대상 / 절차 방향 사전 동결 (실제 실행은 별도 세션)
4. **Blind reformulation 한계 명시** — Protocol 4 D등급 사유 정직 기록

위 1+2 동시 충족 시에만 8인 Rule-A 진입 가능. 단 1·2 중 하나만 충족 시 "조건부 진입" — 8인 리뷰 자체가 해당 protocol 결과를 R3 안전성으로 가산 금지.

## §4 — 미통과 시 박탈 (L549 P3a 동형)

- Protocol 2 에서 paradigm 이 다른 결과값에서도 동일하게 도출되면 → A3+time emergent 는 결과 부정확 일반 도출 절차 → cherry-pick 외형 확정
- Protocol 1 에서 axiom 변경에 paradigm 이 무반응이면 → axiom 무관 결과 후방생성 → cherry-pick 외형 확정
- 두 사유 중 하나라도 발생 시 L549 P3a 박탈 사유 동형 적용 — paradigm shift 자격 박탈, ★★★★ 등급 회복 불가, L568/L593 trajectory 의 ★★★★+ 가능성 소멸
- 박탈 결과는 본 세션 단독 결정 금지. 8인 Rule-A 합의 또는 외부 third-party 결과 인용 필요

## §5 — L590 §4 (본 세션 단독 0/9) 와의 관계

- L590 §4 "본 세션 단독 9 protocol 중 0 통과" 는 R3 회피의 본 세션 단독 한계 명시
- 본 L613 protocol 1, 2 는 본 세션 내부 실행 가능하나 L590 §4 와 모순되지 않음 — 본 세션 내부 실행 가능성 ≠ R3 통과 보장
- Protocol 3 (third-party) 만이 L590 §4 한계의 *외부* 회피 경로
- 따라서 본 L613 의 우선순위 1, 2 는 "본 세션 내 부분 검증" 위치이며 R3 최종 통과 선언은 Protocol 3 결과 도착 후에만 가능
- L590 §4 결과를 L613 protocol 통과 주장으로 우회 금지
