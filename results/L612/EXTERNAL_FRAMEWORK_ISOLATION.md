## §2. Top-2 Protocol *방향*

본 문서는 단일 에이전트 결정 금지 원칙에 따라 *top-2 후보 방향* 만 제시. 최종 선택은 8인 Rule-A 합의.

- **후보 A — Protocol 1 (Pure SQT internal)**: 가장 엄격, 외부 import 회피 가장 확실. "interaction count" 어휘조차 SQT axiom 으로 환원되는지 8인 검증 필수.
- **후보 B — Protocol 4 (WDW problem-recognition only)**: 문제 인식만 차용하므로 해법 도출 책임이 SQT 내부에 남음 → Protocol 1 로 자연 합류 가능. 진입 장벽 낮춤.

Protocol 3, 5 는 자기기만 위험으로 *2차 후보* 로 격하. Protocol 2 는 pillar 별 분기 필요 → top-2 진입 시 후보 A/B 와 병행 보조.

## §3. 8인 Rule-A 진입 조건

다음 *모든* 조건 충족 시에만 8인 Rule-A 이론 도출 세션 진입 허용:

1. **어휘 isolation 명시**: 차용 어휘와 차용 수학의 경계가 글로 명시. "어휘만" 자기기만 차단.
2. **외부 framework 의존성 0 선언**: 도출 path 가 외부 framework 의 정의/공리/기법에 의존하지 않음을 8인 중 *최소 6인* 독립 확인.
3. **SQT axiom 추적성**: 도출의 모든 단계가 a1-a6 + B1 중 하나로 직접 환원 가능. 환원 불가 단계가 1개라도 있으면 외부 import 로 간주.
4. **회의적 압박 통과**: §1 표의 회의적 위험 항목 각각에 대해 8인 중 회의 담당 최소 2인이 명시 반박.
5. **postdiction 차단**: 외부 framework 와 수렴이 발견될 경우, 도출 *시점* 이 수렴 *발견* 보다 앞섬을 commit history 등 timestamp 로 증명.

## §4. 미통과 시 Fallback *방향*

§3 진입 조건 미통과 시:

- **격하 1차**: A3+time emergent 를 "paradigm shift 후보" 에서 "phenomenological 어휘 정렬" 로 격하. L605 Bootstrap 명목 단계 (0.8/4) 패턴 회귀 명시.
- **격하 2차**: A3 자체를 "SQT 내부 공리 후보" 에서 "외부 framework 와의 비교 좌표" 로 재정의. paradigm shift 주장 철회.
- **정직 기록**: L610 §7 회의적 압박 #1 (외부 import 위험) 미해소 명시. GR 4축 보편성 점수 +0.2 회수.
- **재시도 조건**: Protocol 1 단독 또는 Protocol 1+4 결합 으로 재시도 1회 허용. 2회 연속 미통과 시 A3+time emergent 노선 폐기, 4-pillar 내부 노선으로 회귀.
