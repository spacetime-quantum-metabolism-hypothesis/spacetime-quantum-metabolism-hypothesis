## §4. Top-3 paradigm shift 후보

축 합산 + GR-shape (commonsense 갈아엎음 + empirical 양립 + 단순 + 보편) 4축에서 가장 큰 후보 셋:

### Top-1: **C-K10a Anchor as constitutive** (L603 measurement axiom)
- **Newton→GR 유비**: "절대공간 = 외부 무대" → "metric = 동역학 변수". SQT 에서 "anchor = 외부 단위계" → "anchor = 이론 구성요소".
- **L603 와 직접 연결**: measurement axiom 이 anchor 를 이론 내부로 끌어들인다.
- **L604 Bootstrap 와의 관계**: Bootstrap 는 K2(axiom) shift, Anchor-constitutive 는 K10(anchor) shift — *직교*. 양립 가능.
- **L601 substrate 와의 관계**: substrate 가 anchor 를 정의한다는 방향이면 양자가 자연 결합.

### Top-2: **C-K4a Emergent spacetime** (L601 substrate primary)
- **Newton→GR 유비**: "stage 위 사건" → "metric 자체가 동역학". SQT 에서는 "spacetime stage" → "substrate 가 spacetime 을 emerge".
- **L601 와 직접 일치**.
- **Bootstrap 와의 관계**: Bootstrap 는 axiom 차원 shift, emergent spacetime 은 ontology 차원 shift — 양립.
- **GR 4축 합산 가장 안정적** (모든 축 +/~ 만, − 없음).

### Top-3: **C-K7a 단일 substrate 에서 4-pillar 동시 도출** (L601 연장)
- **Newton→GR 유비**: 4개의 분리된 힘 → 1개의 geometry. SQT 에서 4-pillar(DM/DE/H0/Higgs) → 1 substrate.
- **L601 의 전면 야심형 버전**.
- **위험**: 4축 중 정확도(축 2) 가 가장 약함 — 단일 substrate 가 4 pillar 동시 데이터 정합성 보일지 미증명.
- **장점**: GR 의 "many forces → one geometry" 와 가장 닮은 *형태*.

### 후보별 L601/L602/L603/L604 와의 관계 요약

| Top | L601 substrate | L602 hidden | L603 anchor | L604 bootstrap |
|---|---|---|---|---|
| 1 anchor-constitutive | 결합 가능 | 무관 | **본체** | 직교 (양립) |
| 2 emergent spacetime | **본체** | 무관 | 결합 가능 | 직교 (양립) |
| 3 단일 substrate 4-pillar | **본체 확장** | 충돌 가능 | 결합 가능 | 직교 |

L602(hidden DOF) 는 K9 카테고리에 별도 — Top-3 와 직접 충돌하지 않으나, 단일 substrate 모델에서는 hidden DOF 양이 많으면 자가모순 위험.
