**Target**: SQT 가 quantum field theory 의 RG flow 와 어떻게 호환되는가? σ_0 가 scale-dependent (running)? n0 의 scale 종속성?

### 약점
1. Branch B 3-regime σ_0(z) 는 cosmological scale 에서 해 ; QFT scale (μ_RG) 에서는 ?
2. β-function 미정.
3. UV completion: trans-Planckian regime 에서 SQT 의 fate.
4. Asymptotic safety (Reuter) 와의 관계?
5. n0 = ρ_Pl/(4π) 가 Planck-scale UV fixed point 시사.
6. dimensionless coupling g = σ_0 (n0 c)^? 토이 계산.

### KILL
- K-RG1: g 가 1보다 크면 strong coupling, perturbative QFT 무효.
- K-RG2: β(g) 부호 — UV에서 free (asymptotic free) 또는 fixed point 인지.

### 실행
