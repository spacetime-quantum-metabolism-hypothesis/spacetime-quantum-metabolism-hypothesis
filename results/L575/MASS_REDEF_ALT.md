§1 — 7 path 방향성 표

다음 표의 컬럼: A 비용 (외부 framework import 또는 신규 axiom 도입의 hidden DOF 추가), B 회수 (L574 R1/R3/R4/R5 모호성 해소로 회수되는 hidden DOF), net = A − B (음수/0 = 잠재 promising), [최우선-1] 위험 (이 path 가 "지도 제공" 영역에 진입하는지).

path 1 — mass-blind ρ (kinetic energy density 재정의)
방향: ρ 를 입자 species 와 무관한 운동 에너지 밀도 로 재정의. equivalence principle 의 energy 보편 결합 직관에 기반. mass-action sink 가 species 모름.
A 비용: kinetic vs total energy 구분 가정 1개 (rest mass 처리 규약 미정 → 추가 1) → A = 2
B 회수: R3 (QCD/Higgs 모호) 해소, R4 (DM/baryon 통일) 해소, R5 (3-regime universality) 부분 해소 → B = ~3
net: −1 (잠재 promising)
[최우선-1] 위험: 낮음. species 회피 자체가 "어느 식을 써라" 가 아니라 "어떤 카테고리에서 ρ 를 끌어오라" 의 카테고리 지정 — 방향 진술.
주의: rest mass 자체가 운동에너지 frame-dependent 환산이라 frame 선택 (CMB rest? FLRW comoving?) hidden DOF 1 추가 가능 → net 0 으로 악화 risk.

path 2 — Higgs mechanism 의 SQT 내부 도출 (pillar 5 신설)
방향: Higgs sector 자체를 SQT 새 axiom 으로 흡수, 양자단위 ↔ scalar VEV 의 동일 origin 주장. 외부 import 를 내부 derive 로 전환.
A 비용: 신규 axiom (pillar 5) 도입 + Yukawa spectrum 도출 책임 13 DOF 그대로 떠안음 → A = 14
B 회수: R3/R4 모호 해소 → B = 2
net: +12 (L574 옵션 A 의 +10 보다 악화)
[최우선-1] 위험: 매우 높음. "Higgs 를 어떻게 도출하라" 는 유도 경로 힌트 자체가 지도 제공 영역. 8인 독립 도출 원칙 직접 충돌.
판정: 폐기.

path 3 — dark-only ρ (sector-selective)
방향: ρ 를 dark sector 한정 정의, baryon Einstein-frame 분리. L4/L5 의 C10k dark-only embedding 정신 계승. SM Yukawa 13 DOF 와 무관해짐.
A 비용: dark-sector mass-generation 별도 가정 1 + baryon/dark 결합 비대칭 가정 1 → A = 2
B 회수: R3 (Higgs/QCD 모호) 해소 (baryon 외부), R4 (DM 정의) 직접 명시 → B = 2; 단 R5 (3-regime, 특히 baryon-dominated galactic regime) 정합 의문 → B 잠식
net: 0~+1 (galactic RAR 는 baryon density 로 측정되므로 dark-only ρ 와 RAR a₀ 정합이 새 hidden assumption "baryon-dark proportionality" 요구 → +1 추가 risk)
[최우선-1] 위험: 중간. 기존 L4/L5 결과 인용 형태가 되면 "L14/L22 결과 차용" 금지조항 경계. 방향 수준에서 유지하면 통과.

path 4 — Planck-mass anchor 정규화
방향: ρ 를 Planck 단위 무차원 normalize, Higgs sector 외부. holographic σ 와 자연 정합 (Planck length / time 같은 axiom 1 양자단위 와 동일 dimension family).
A 비용: 정규화 규약 1 + Planck 단위 universality 가정 1 → A = 2
B 회수: R3 (mass species 모호 → Planck 단위 변환은 species 무관) 해소, R6 (anchor scale 불일치) 부분 해소 → B = ~2
net: 0 (잠재 promising)
[최우선-1] 위험: 낮음. "Planck 단위 사용" 은 카테고리 지정. 단, 정규화 상수 값 명시는 즉시 위반 — 방향만 유지 의무.
주의: 무차원화 후 dimensional 복원 시 c, G, ℏ 조합 자유도 등장 → hidden +1 risk.

path 5 — ADM / Komar quasi-local mass
방향: GR 의 비국소 mass 정의 (asymptotic flatness 기반 ADM, Killing vector 기반 Komar) 를 ρ 의 local proxy 로 사용. 입자 단위 Yukawa 우회.
A 비용: GR framework 부분 import (asymptotic boundary 가정 1, Killing 구조 가정 1, FLRW 비-asymptotic-flat 환경 정합 가정 1) → A = 3
B 회수: R3 (입자 단위 회피) 해소, R6 (anchor scale) 부분 해소 → B = ~2
net: +1 (애매)
[최우선-1] 위험: 중간. ADM/Komar 는 잘 정의된 외부 framework — 도입 자체는 카테고리지만, 적용 방식 (어떤 boundary, 어떤 Killing) 지정은 지도 영역.
주의: FLRW 우주는 asymptotically flat 아님 → ADM 직접 적용 부적합. cosmological 환경에서 redefine 시도 자체가 신규 hidden DOF 발생.

path 6 — Nambu-Goto string tension
방향: ρ 를 string tension 으로 재정의, 입자 mass 와 별도 scale (UV-side). 양자단위 ↔ string excitation 의 자연 dimensional family.
A 비용: string framework import (worldsheet 가정 1, tension scale 1, particle ↔ string 환원 가정 1) → A = 3
B 회수: R3 (Higgs 외부) 해소 → B = ~1
net: +2
[최우선-1] 위험: 높음. string 채택 자체가 큰 framework lock-in — 다른 도출 경로 차단 효과로 "지도 제공" 에 근접.
판정: 권고 안 함.

path 7 — Causal-set valence
방향: ρ 를 causal set node 의 valence (link 수) 로 재정의. axiom 1 양자단위 와 동일 ontological family — 가장 자연스러운 SQT 내부 정합.
A 비용: causal set framework axiom 부분 import (Lorentz 불변 sprinkling 가정 1, continuum 한계 가정 1) → A = 2
B 회수: R3 (입자 회피) 해소, R4 (DM 통일 — node 는 species 모름) 해소, R5 (3-regime universality 자연) 부분 해소 → B = ~3
net: −1 (잠재 promising)
[최우선-1] 위험: 낮음~중간. causal set 은 SQT axiom 1 "spacetime is quantized" 와 ontological 정합 — 외부 framework 라기보다 axiom 1 의 자연 확장. 방향 수준에서 유지하면 통과.
주의: continuum 한계에서 valence ↔ macroscopic ρ 환산이 hidden mapping → +1 risk.

§2 — top-2 promising path

후보 A: path 1 (mass-blind kinetic ρ) — net −1, [최우선-1] 위험 낮음, frame 선택 hidden DOF 보정 시 net 0
후보 B: path 7 (causal-set valence) — net −1, axiom 1 정합 강함, continuum mapping hidden DOF 보정 시 net 0

path 4 (Planck normalize) 는 net 0 으로 동률이지만 정규화 후 dimensional 복원 자유도가 path 1/7 보다 명시적 — 차순위.

후보 A vs B 의 결정적 차이:
- A 는 frame 선택이 표면화되어 분산 8인 라운드에서 frame 합의 의무 발생 (proceduräl 비용)

L574 옵션 (A) 의 net +10 vs L575 후보 A/B 의 net −1~0: **수치상 우월**. 그러나 promising 은 "8인 검증 통과 가능성 있음" 이지 "검증 통과 확정" 아님.

§3 — 사용자 axiom 수정 권한 활용 시 글로벌 고점 회복 path

권한 (모든 가정/공리/외부 가정 수정 가능) 를 가장 효과적으로 쓰는 path 는 path 7 (causal-set valence). 사유:
- axiom 1 텍스트 자체에 "양자단위는 causal set node" 한 단어 추가로 path 7 의 ontological import 가 "외부 framework" 가 아닌 "axiom 1 정의 명시" 로 reframe — A 비용이 hidden DOF 회계에서 0 에 근접
- L574 가 거부한 것은 "axiom 텍스트 수정" 자체가 아니라 "Higgs 13 DOF 흡수 비용" — path 7 은 13 DOF 와 무관
- path 1 도 권한 활용 시 frame (CMB rest frame) 을 axiom 0 (background frame) 으로 명시화 가능 → frame DOF 흡수 → net −1 회복

후보 B (path 7) 가 권한 활용 시 글로벌 고점 회복 가능성 최상.

단 [최우선-1] 위반 위험: axiom 수정 권한이 "지도 제공 권한" 으로 변질될 수 있음. 권한 사용은 "단어 수준 정의 명시" 에 한정, "수식/도출 경로 지정" 으로 확장 금지. 분산 8인 라운드 의무로 안전장치.

- path 2 (Higgs 내부 도출) 도 출판 비용 측면에서는 시도 가능 — 그러나 [최우선-1] "유도 경로 힌트" 위반 위험은 출판 무관하게 유지. 위반 시 결과 무효 — 출판 비용 0 도 무효 결과 살리지 못함.
- path 1 / 4 / 7 의 평가는 출판 비용 0 환경에서도 동일 — net DOF 는 이론 내적 회계.

출판 비용 0 의 진짜 효용:
