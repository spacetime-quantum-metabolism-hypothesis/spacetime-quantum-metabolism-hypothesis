# L198 — Physical interpretation of smooth σ(ρ)

## Smooth σ(ρ) 가 자연스러운가?

### Pro-smooth (more natural)

1. **물리 시스템 일반적**: 자연계 매끄러운 의존성 다수 (전기 저항도, 점성도, etc.)
2. **Ostrogradsky 안전**: discrete jumps 가 미분 불가능 → 후속 해석 어려움
3. **연속 매질** 표준: σ_0 가 연속 값 가지는 것이 더 일반
4. **Phase transition manifest**: 만약 진정 phase transition 있으면 critical point 근처 *부드러운 transition* 이 자연

### Con-smooth (less natural)

1. **Branch B regime 구조** 가 *empirically 발견된* 패턴 (T17/T20/T22 σ_0 difference)
2. **3 distinct regimes** 이 데이터 명료
3. **Phase transition** 가 가능성 제시 (응집체 비유)

### 결정적 차이

```
[Smooth σ(ρ)]
- σ continuously varies
- No 'phase' boundaries
- Mass scaling natural
- 'modify gravity smoothly'

[Branch B 3-regime]
- 3 distinct phases
- Boundary-defined regions
- 'phase transitions in spacetime'
- More radical physical claim
```

### 학계 reception

Smooth 는 *more conservative*, 학계 더 호의적:
- "Continuous deviation from GR" — many existing modified gravity papers
- "Standard f(R), MOND interpolation" — 표준

Branch B 는 *more radical*, 학계 *more skeptical*:
- "discrete spacetime regimes?" — unconventional
- "phase transitions in σ_0?" — needs strong evidence
