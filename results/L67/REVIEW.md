## 🔥 핵심 발견 — σ_0 evidence 비단조 패턴

L48 누적 σ_0 값 비교:
```
T22 (galactic disk)   = 3.31e9   ← 최대
T17 (cosmic mean)     = 2.34e8   ← 중간
T20 (cluster σ_8)     = 5.6e7    ← 최소
```

밀도 순서: cluster (2.7e-24) < galaxy (1.7e-21), 그리고 cosmic (2.7e-27) < cluster.

**σ_0 와 밀도 관계가 비단조**:
- 밀도 증가 (cosmic → cluster): σ_0 *감소* (2.34e8 → 5.6e7)
- 밀도 더 증가 (cluster → galaxy): σ_0 *증가* (5.6e7 → 3.3e9)

→ σ_0 가 밀도의 **비단조 함수**. 또는 구배·속박의 비단조 함수.

**단조 sigmoid 활성화는 이 패턴을 *원리적으로* 재현 불가능.**
