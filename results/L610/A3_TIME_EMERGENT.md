## §1. A3 + Time emergent 결합 의미 (방향)

### 1.1 적출 대상 commonsense
- 기존 commonsense: "Γ₀ 는 시간에 따라 변화하는 emission rate" — t 라는 외부 parameter 가 *암묵적으로 절대시간* 의 자격으로 이론에 진입.
- L527 Path-α 가 Γ₀ 시간의존성을 추가했으나, t parameter 자체의 *존재* 는 의문에 부치지 않았다.
- L610 paradigm shift 의 *방향*: t parameter 자체를 commonsense 환상으로 적출한다. Γ₀ 는 *시간 함수* 가 아니라 *boundary state* — 시공간의 가장자리에서 정의되는 정적 condition.

### 1.2 GR analogue 의 위계적 강도
- GR 의 commonsense 전복: "Newton 의 절대시간" → "worldline 별 proper time".
- SQT 가 본 paradigm 에서 시도하는 commonsense 전복: "Γ₀(t) 라는 함수" → "Γ₀ = boundary state, 시간은 상호작용 횟수 / count 로 emergent".
- 위계 측면에서: GR 이 *절대* 시간을 적출했다면, SQT 는 *parameter* 시간 자체를 적출하는 위치 — 한 단계 더 깊은 commonsense 층.

### 1.3 Time emergent 의 원천 후보 (방향만)
- 상호작용 카운트 channel
- boundary 와 bulk 사이의 정합 channel
- entropy / complexity flow channel
- 8인 토의에서 자율 도출. 본 문서 사전 지목 금지.

## §7. 회의적 압박

1. **외부 framework import 위험**: "boundary state" 와 "emergent count" 는 causet, loop quantum gravity, Wheeler-DeWitt 등 기존 framework 에서 차용한 것이 아닌가? 차용이라면 *L610 paradigm 의 독립성* 주장이 약화된다. 8인 토의에서 자율 도출 형태가 외부 import 없이 유지되는지 검증 필요.
2. **Path-α 박탈 사유의 우회 형태 회귀**: K-Z 단일 보조 의존이 *재이름* 으로 (예: "single emergent-count anchor 의존") 되돌아올 위험. 박탈 사유는 *형식* 이 아니라 *구조* — paradigm 변경에도 동일 구조 잔존하면 동일하게 박탈.
3. **Postdiction 시간 순서 (R3) 위반**: 1/(2π) 와 a₀ 결과를 *알고 난 뒤* paradigm shift 를 도출하고 있다. 본 paradigm 이 그 결과를 자연 재현하지 못하면 ad hoc rescue 가 된다. R3 위반 의심을 *명시적으로* 안고 진행.
4. **검증 불가능성 위험**: time emergent 가 너무 추상적이면 falsifiable 예측을 생성하지 못할 수 있다. 8인 토의에서 K (kill) 기준 사전 명시 필수.
5. **L549 박탈 재심 의 자기정당화 위험**: paradigm 을 바꾸면 박탈을 회피한다는 논리는 *공통적으로 적용 가능* — 따라서 회피 자체로 paradigm 의 우월성을 주장하면 안 된다. 회피의 *물리적 기반* 이 필요.
