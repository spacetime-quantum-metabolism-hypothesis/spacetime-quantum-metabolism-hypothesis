# L353 NEXT_STEP — 후속 세션 지시

## 결정 사항 (L353 산출)

- c 도출 가능성: **부분 가능 (B 등급)**.
- 부호 + 자릿수 = 이론 도출. Prefactor = scheme tag 동반 보고.
- b/c → σ_cluster 매핑 = 차원적으로 닫히되 매핑 form 의 uniqueness 미확정.

## L354 (다음 세션) 지시

1. **c 의 부호 결정 단독 세션**: 8인 팀, 차원/대칭/1-loop 정합성만으로 c 부호를 좁힌다. 수식 없는 방향만 공급.
2. **scheme 선택 명시 세션**: c 의 prefactor 를 정의할 때 어떤 scheme 를 SQMH 의 "natural" scheme 로 채택할지 합의. 채택 근거는 SQMH 공리 (L0/L1) 와의 정합성으로만.
3. **saddle stability 검증**: b 부호 + c 부호 조합이 saddle FP 의 stability matrix 에서 모순 없는지 4인 코드리뷰 팀이 자율 분담으로 수치 점검.
4. **σ_cluster 매핑 uniqueness 검증**: 차원 + 대칭 두 축에서 매핑 form 이 unique 한지 재확인. unique 하지 않으면 ansatz 명시.

## 금지 사항 (다음 세션)

- L353 의 결론 (B 등급) 을 "L354 에서는 A 로 끌어올린다" 류 목표 설정 금지. 데이터/이론이 허용하는 만큼만.

- c 의 수치를 외부 phenomenology (σ_cluster 측정값) 에서 거꾸로 fit 하여 "도출했다" 고 보고 금지. fit 결과는 calibration 라벨 명시.
