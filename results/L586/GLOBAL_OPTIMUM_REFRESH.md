## §2 글로벌 고점 정의 갱신 (출판 시도 금지 반영)

### 2.1 갱신 전 (L568 SYNTHESIS_479 시점)
- 외부 acceptance (PRD Letter 진입) 가 글로벌 고점의 외적 신호.
- portfolio 가치 + 실험 falsifiability + 회의적 통과를 모두 평가.

### 2.2 갱신 후 (L586 — 출판 무관)
- **외적 acceptance 무관**. 따라서 PRD Letter 진입 조건 (Q17 OR Q13+Q14) 은 *내부 의사결정 신호* 로만 사용. 제출 행위 자체는 금지.
- 글로벌 고점 = (i) priori 도출 회복 1건 이상 + (ii) 회의적 검증 통과 + (iii) Hidden DOF 9–14 구조 유지 (L582 종결 후 정착값) 의 동시 충족.
- 본 정의 하에서 현재 priori 회복 0건이며 활성 후보 4건이 모두 미통과. 따라서 글로벌 고점은 **현시점 미달**.
- 이론적 완성도 축은 L582 Mass redef 종결로 *축소*된 상태이며, L585 5th path 가 결과 자체를 회수하지 못하면 추가 축소 위험.
