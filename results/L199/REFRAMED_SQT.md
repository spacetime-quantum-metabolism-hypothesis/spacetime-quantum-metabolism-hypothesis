## Reformulated Section 3 outline

### Section 3: SQT phenomenology and regime structure

#### 3.1 Need for σ_0(env)

Cosmic, cluster, galactic data require different effective σ_0 values
[L48-L67 history]. We need parameterization for σ_0(ρ_m).

#### 3.2 Branch B 3-regime ansatz (Primary)

Phenomenological 3-regime: σ_0 takes 3 distinct values with sharp
boundaries.
- Cosmic: 10^8.37
- Cluster: 10^7.75
- Galactic: 10^9.56

Motivation: distinct physical regimes (phase transition possibility,
RG fixed points). Statistical preference over alternatives in joint
fit (anchors + SPARC).

#### 3.3 Smooth σ(ρ) alternative (Equally Valid)

Continuous σ_0(ρ) ansatz:
σ(log ρ) = a + b·log ρ + c·(log ρ)²

Provides equivalent fit to 3 anchor points. Slight preference for Branch B
in joint anchor+SPARC fit.

#### 3.4 Predictions independent of regime structure

9 of 14 SQT predictions (P2, P3, P4, P5, P6, P7, P8, P12, P14) are
*axiom-derived* and independent of regime structure choice. Only 5
predictions (P1, P9, P10, P11, P13) require Branch B specifically.

#### 3.5 Distinguishing future tests

The 5 Branch B-specific predictions can be tested at intermediate
density regimes (dSph, void galaxies, NS interior) and provide
*decisive* tests of regime structure.

#### 3.6 Theoretical foundations (current limits)

Both Branch B and smooth ansatz are phenomenological. Underlying
microscopic mechanisms (Landau-Ginzburg, RG flow with multiple fixed
points) provide plausibility but are not first-principles derivations.
