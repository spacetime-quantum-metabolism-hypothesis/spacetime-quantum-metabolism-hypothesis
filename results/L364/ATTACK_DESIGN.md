# L364 ATTACK_DESIGN — Causal Set Theory as SQT n-field micro pillar

날짜: 2026-05-01
주제: Sorkin Causal Set Theory (CST) 가 SQT n-field discrete spacetime quantum 의 micro pillar 가 될 수 있는가?
독립 세션: L46~L363 결과를 모방/이식하지 않고, CST 자체 구조에서 출발해 n-field 와의 매핑 가능성만 평가.

---

## 1. 정직 한국어 한 줄

CST 의 이산 점과 SQT 의 n-field 격자값을 동일시하려는 직접 매핑은, 인과 구조(반사슬/사슬) 와 스칼라 밀도(연속 n) 의 차원·의미 차이 때문에 단순 일대일 대응으로는 micro pillar 로 채택할 수 없으며, 매개층(coarse-graining) 없이는 정합 불가하다.

---

## 2. 공격 대상 (claim under attack)

CLAIM-A: "CST 의 element 하나 = n-field 단위 격자값"
CLAIM-B: "CST sprinkling density ρ ↔ n(x) 직접 동일"
CLAIM-C: "CST 의 BD action 변동 → n-field 동역학 직접 도출"

---

## 3. 공격 방법 (방향만, 수식 금지)

### 공격축 1 — 차원/의미 분리
- CST element 는 사건(event), 차원 없음. n-field 는 SQT 내 metabolism 밀도 (차원 있음).
- 공격: "둘 사이 단위계 변환이 가능한지" 만 기호적으로 점검. 수식 도출은 팀 자율.

### 공격축 2 — 인과 구조 보존
- CST 의 핵심 자산은 partial order (선후 관계). n-field 는 스칼라이므로 순서 정보 손실.
- 공격: "n(x) 만으로 partial order 를 재구성할 수 있는가?" 카운터 예시 탐색.

### 공격축 3 — Lorentz invariance
- CST 는 Poisson sprinkling 으로 LI 보존. n-field 가 균질 격자라면 LI 깨짐.
- 공격: "n-field 가 Poisson 분포여야 하는지, 균질 격자여도 되는지" 강제 조건 도출.

### 공격축 4 — Coarse-graining 필요성
- 직접 매핑이 실패하면, 중간층(예: n = 국소 element 밀도의 평균) 을 거쳐야 하는지 검증.
- 공격: "매개층이 들어가는 순간 micro pillar 가 아니라 meso pillar 로 격하" 라는 함정.

### 공격축 5 — Action / dynamics
- CST BD action 은 인과 사슬 카운팅. n-field 동역학은 metabolism flux.
- 공격: "두 동역학이 같은 물리량을 다른 형식으로 적분하는지, 아니면 별개 자유도인지" 분리 판정.

---

## 4. 팀 구성 (역할 사전 지정 금지)

- 6인 자유 토의팀: CST 측 3명 + SQT n-field 측 3명. 역할 배정 없음. 매핑 시도 → 자발적 분업으로 차원/순서/LI/coarse-graining/action 검토.
- 매핑 결론은 만장일치 또는 6/6 표기. 5/6 이하면 "매핑 불성립" 보고.

---

## 5. 합격 조건 / Kill 조건

- PASS: 공격축 1~5 모두 통과 + coarse-graining 없이 직접 매핑이 가능한 구체 사례 제시.
- KILL-soft: 1~5 중 하나라도 매개층 필요 판정 → micro pillar 자격 박탈, meso 후보로 강등.
- KILL-hard: 인과 순서 정보가 n-field 에 환원 불가능 → CST 와 SQT 는 보완 관계이지 pillar 관계 아님.

---

## 6. 과적합 방지

- CST 의 BD action 파라미터를 n-field 동역학에 끼워 맞추는 시도 금지. 자유도 추가 시 AICc 패널티 명시.
- 매핑이 "거의 된다" 수준이면 KILL-soft 로 처리. 회색 영역을 정합으로 보고 금지.
