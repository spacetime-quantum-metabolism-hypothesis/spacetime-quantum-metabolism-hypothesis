## R1 — causal-set framework 외부 import 여부

회의자: framework 회계 검열관.

Sorkin 의 causal set 프로그램 (1987~) 은 독자 axiom set 을 가진 외부 양자중력 framework. 핵심 가정: (i) Lorentz 불변 Poisson sprinkling, (ii) causality = partial order, (iii) continuum 한계로 GR 회복, (iv) BDG action (Benincasa-Dowker) 의 비국소성. 이 4 가정 중 (i) 과 (ii) 는 SQT axiom 1 ("spacetime is quantized") 에 *자동 포함되지 않음*. axiom 1 은 단지 "양자단위 존재" 만 진술 — 양자단위들 사이의 *order 구조* 는 axiom 1 외부.

즉 path 7 채택 = sprinkling 가정 + causal 구조 가정 import. L575 §1 은 A 비용 = 2 (sprinkling + continuum 한계) 로 회계했지만 실제로는 (a) sprinkling 의 Lorentz 불변 특이성 (Poisson 외 다른 분포는 Lorentz 깨짐) 1, (b) partial order = causal 구조 1, (c) continuum 한계 mapping 1, (d) BDG-style action 또는 대안 dynamics 1, 최소 **A = 4**. L575 회계 과소평가.

**판정 R1**: causal set 은 외부 framework. axiom 1 의 "자연 확장" 어휘는 회계 면제 사유가 아니다. [최우선-1] "외부 import 시 hidden DOF +1" 이상 위반 가능성 농후.

## R2 — axiom 1 자연 확장 vs 변경

회의자: axiom 텍스트 변경 검열관.

axiom 1 (paper §2.1, L0) 본문: "scalar number density n(x,t) of spacetime quanta with individual rest mass μ." 이는 *scalar field* 형식의 number density — point set 위에 정의된 *연속 함수*. causal set 의 valence 는 *discrete graph* 위에 정의된 *정수값 노드 속성*. 두 ontology 는 동일하지 않다:
- scalar n(x,t): 위치 (x,t) 의 함수, 미분 가능 (continuity equation 2.1 에서 ∂_t n + ∇·(nv) 사용)
- valence v_i: 노드 i 의 정수, 위치 정의 자체가 sprinkling 의 통계적 implementation 에 의존

L575 의 "axiom 1 의 ontological 자연 확장" 주장은 *redefinition* 을 *clarification* 으로 reframing 하는 어휘 트릭. 실제로는 axiom 1 의 기반 수학구조 (scalar field on manifold) 를 (graph valence) 로 *교체*. 이는 단어 수준 정의 명시가 아니라 **axiom 1 본문 수정** 이며, [최우선-1] "수식/유도 경로 사전 고정" 영역에 진입.

continuity equation (2.1) 의 ∇·(nv) 은 graph 위에서 의미가 모호 — graph Laplacian 으로 교체할지 continuum 한계 후 사용할지 자체가 dynamics 선택 = 유도 경로 지정.

**판정 R2**: "자연 확장" = 자기기만. axiom 1 본문 수정에 해당.

## R3 — Higgs sector 13 DOF 처리

회의자: zero-parameter slogan 검열관.

L574 가 옵션 (A) 를 net +10 으로 거부한 핵심 사유는 Higgs+Yukawa 13 DOF 흡수. path 7 의 valence ρ 는 SM 입자 단위와 무관 — 표면적으로는 Higgs sector 회피 성공. 그러나:

- a₀ derivation, RAR 정합 검증은 SPARC photometric mass (M_*/L 기반, stellar Yukawa-mass) 를 입력으로 사용. valence ρ 가 어떻게 SPARC stellar mass 와 mapping 되는지 미정 — 이 mapping 자체가 silent Higgs sector 재import.
- BBN η_b 는 baryon number density × proton mass — proton mass 는 99% QCD binding + 1% Yukawa. valence ↔ proton mass 의 mapping 또한 hidden.
- Cassini PPN 의 Sun mass 도 동일.

valence 단위 ρ 가 도입되어도 *관측 anchor 와의 결합부* 에서 SM 13 DOF 가 hidden mapping 으로 재진입. paper §6 의 13 hidden DOF disclosure 의무는 그대로. path 7 은 13 DOF 를 "회피" 하는 게 아니라 "한 단계 깊이 숨김".

**판정 R3**: Higgs disclosure 의무 잔존. path 7 은 13 DOF 문제를 풀지 않고 mapping 층으로 전가.

## R4 — 3-regime universality 가정

회의자: regime universality 검열관.

L574 R5: cosmic / cluster / galactic 3-regime 단일 σ 적용에는 "ρ 의 종류 무관, total energy density 만이 sink rate 결정" universality 가정 필수. path 7 valence ρ 는 노드 속성이므로 species-blind — universality 자동 충족 주장 가능.

그러나:
- galactic regime: baryon-dominated. valence-density 가 baryon mass density 와 비례한다는 가정 = mass-blindness 의 *재구현*, 회피 아님.
- cluster regime: DM-dominated. baryon 과 DM 이 동일 valence 결합?
- cosmic regime: Λ-dominated. Λ 는 valence 에서 어떻게 나오는지 — vacuum node 의 self-valence?

valence universality 는 R4 (L574) 의 mass-blindness 와 동일한 가정의 reformulation. 회피가 아니라 **assumption 의 다른 옷**. L575 회계가 "R5 부분 해소 +1 회수" 로 본 것은 과대 평가.

**판정 R4**: universality 가정 잔존. valence reframing 은 가정을 옮길 뿐 제거하지 않음.

## R6 — valence 측정 가능성

회의자: 관측 가능성 검열관.

valence v_i = 노드 i 의 link 수. continuum 한계 ρ_v(x) 로 변환 후 macroscopic mass density 와 매칭. 그러나:
- 단일 노드 valence 는 Planck scale (~10⁻³⁵ m) 에서만 정의 — 직접 측정 불가
- continuum ρ_v 는 *정의상* macroscopic ρ 와 비례하도록 설계되므로 *독립 관측* 불가능 (tautology)
- valence distribution 의 *분산* 또는 *higher moment* 가 별도 관측에 영향 준다는 prediction 부재

valence 는 측정 불가능한 *부속 변수* — net DOF -1 회수 주장은 형식적이며, 측정 불가능한 변수로 회수된 DOF 는 falsifiability 측면에서 의미 없음. K-test (관측 prediction) 통과 채널 0개.

**판정 R6**: valence 측정 불가. 회수된 DOF 는 페이퍼상 숫자에 불과. [자동 박탈 사유 후보 1].

## R7 — 4-pillar 정합 검증

회의자: pillar 충돌 검열관.

- **SK (Sakharov)** / Holographic σ: σ = 4πG·t_P 는 G 와 Planck time 의 결합 — valence 정의가 Planck scale 의 *graph* 면 G 의 metric 정의 자체가 emergent. continuum 한계 GR 회복은 BDG action 가정 필요 — 이는 σ = 4πG·t_P 의 *유도* 가 아닌 *부과* 로 변질.
