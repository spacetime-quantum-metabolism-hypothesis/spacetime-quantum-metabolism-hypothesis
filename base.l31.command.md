## 이론 프로그램: Spacetime Quanta Theory (SQT)

GR의 시공간 곡률을 이산 양자 과정으로 재해석하는 이론 프로그램.
공리 2개에서 중력·우주 진화·블랙홀 구조·양자-고전 경계를 일관 서술.

---

## 공리

- **A1**: 물질이 시공간 양자를 소멸시키며, 소멸된 양자는 에너지로 전환된다.
- **A2**: 시공간 양자는 공간에서 생성된다.

---

## 제약 조건

- **C1 (로렌츠 불변성)**: 소멸은 에너지-운동량 텐서 트레이스 T = ρc² − 3p 에 비례. 복사(T=0)는 자동 탈결합 → 물질만 소멸 주체.
- **C2 (공변성)**: 생성은 공변 부피당 평균률. 개별 사건은 좌표계 무선호, 분포 전체 푸앵카레 불변.

> **적용**: 이론이 C1-C2 를 명백히 위반하면 물리적 후보에서 제외.

---

## 이론 핵심 이중 구조

| 과정 | 곡률 | T 트레이스 | 에너지 | 참여 |
|------|------|-----------|--------|------|
| 소멸 | 리치 | ≠0 | 국소, 양자 단위 | 물질 |
| 재배열 | 바일 | =0 | 비국소, 집단 | 진공·중력파 |

- **능동 중력원 (소멸)**: T 트레이스 — 물질 전용
- **수동 중력반응 (플럭스)**: T_μν 전체 — 모든 에너지

---

## 일차량 / 유도량

- **일차**: ν (입자 내부 소멸 빈도)
- **유도**:
  - m = ℏν/c²
  - σ_eff = (ν/ν_P)·ℓ_P², ν_P = c/ℓ_P
- 순환 정의 회피: 질량은 ν의 에너지 환산.

---

## 동역학

- Γ_ann = σ_eff · ρ_m · ρ_st
- Γ_gen = κ(ρ₀ − ρ_st)
- ρ₀ = 1/ℓ_P⁴ (개수 밀도, 에너지 아님)
- 평형 어트랙터: ρ_st → ρ₀
- 평형점: ρ_st* = κρ₀ / (κ + σ_eff·ρ_m)

**핵심 분리**: 시공간 양자의 개수 ≠ 에너지 → 우주상수 문제 구조적 회피.
평형 진공은 에너지 0, 관측 Λ는 동역학적 편차.

---

## B' 시나리오 (생성 = 공간 팽창)

- 생성 사건 = 부피 증가 = 양자 수 증가 (밀도 불변)
- Hubble 팽창은 A2의 거시 표현
- 중력 = 물질 근처 소멸 우세 → 국소 수축 플럭스
- 암흑에너지 = 희석에 의한 소멸 둔화 → 생성 우세 → 가속
- Friedmann 대응: H = (생성률 − 소멸률)/3

---

## 4층 구조

| 층 | 기술 | 확률성 |
|----|------|--------|
| L0 | 인과집합 + 소멸/생성 규칙 | 근본 이산 |
| L1 | 시공간 양자장 QFT (Keldysh CTP) | 양자 연산자 |
| L2 | Langevin/MSR | 확률 노이즈 |
| L3 | 유효 라그랑지안 | 결정론 평균장 |

층간 연결:
- L3 ← L2: MSR 경로적분 (확립)
- L2 ← L1: Keldysh 준고전 극한 (확립)
- L1 ← L0: 미확립 (양자중력 전반 난제)

---

## L3 유효 라그랑지안 (거시)

- ψ = n/ρ₀: 정규화 양자 밀도 (무차원)
- V(ψ): 평형 퍼텐셜, 최소 ψ=1
- ψ=1 에서 표준 GR 회복
- 유효 영역: 평형 근처, 거시, 평균 거동

---

## 유도 결과 (이론 내부 일관성 확인용)

- **중력**: 시공간 양자 플럭스, GR 곡률은 플럭스의 기하학적 근사
- **등가원리**: σ_eff ∝ ν → 관성질량 = 중력질량 자동
- **광자 (ν=0)**: 질량·소멸 없음 + 플럭스 반응 → 렌즈·적색편이 설명
- **블랙홀**: 특이점 없음, 유한 밀도 코어 ρ_st* = κρ₀/(κ+σρ_m). 정보 보존.
- **우주 3시대**:
  - 복사기 (T=0): 소멸 없음
  - 물질기: 소멸 > 생성, 감속
  - 암흑E기: 생성 > 소멸, 가속, de Sitter 어트랙터

## AICc 기준선 (전체 고정)

```
ΛCDM best-fit: chi²=10.192, AICc=15.392 (k=2, n=13)
AICc(k) = chi² + 2k + 2k(k+1)/(n−k−1),  n=13
```

k=2 이론이 이기려면: chi² < 10.192
k=3 이론이 이기려면: chi² < 6.73

→ k=2 이론 (자유: Ω_m, H₀) 최우선.
