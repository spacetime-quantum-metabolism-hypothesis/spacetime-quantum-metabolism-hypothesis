## Run 1: MCMC 설계 및 예비 분석

**핵심 발견**:

1. **H_ref 의존성 — MCMC 판정 선행 조건**
   B=9.065 재현 여부는 H_ref 설정에 조건부. H_ref=H₀ 임의 설정이면 판정이 이론 예측력을 측정하지 않음. "H_ref=H₀ 가정 하에" 조건부 판정임을 명시해야 함.

2. **자기일관 ODE가 MCMC 선행 필수 조건**
   ω_DE(H)의 H-의존성으로 인해 H(z) 계산이 implicit ODE. ODE 수치해 없이 MCMC 불가.

3. **Fisher matrix 예비 추정**
   B 변화에 대한 D_M(z) 민감도 ~0.3-1%. DESI DR2 정밀도 내에 구분 가능하나 tight하지 않음.

4. **AICc 자유화 패널티**
   A, B 자유화 시 +4 패널티. 이론값이 최적이라면 ΔAIC(고정 vs 자유) ≈ +4 → 고정이 오히려 통계적으로 유리. 이것이 역설적으로 이론에 긍정적 신호.

5. **L19 B_free=8.76 해석**
   이론값 9.065와 3.4% 차이. Posterior uncertainty 미확인 → 판정 보류. 실제 MCMC 필수.

**MCMC 코드 구조 수립**: emcee/dynesty + DESI DR2 full covariance matrix + 자기일관 ODE solver. 4인 코드리뷰 계획 완료.

**Run 1 등급: C+** (설계 완료, 미실행. H_ref 없이 판정 불가)

## Run 2: H_ref 고정 시도 — 이론 내부 고정 불가 확정

**시도된 모든 경로 및 결과**:

| 경로 | 결과 |
|------|------|
| AdS₂ 곡률 스케일 | 실패 — 무차원 Casimir, 스케일 미제공 |
| SL(2,R) 불변량 | 실패 — 에너지 스케일 변환에 외부 상수 필요 |
| Swampland 추측 | 부분 — 차수만 추정, 정확한 값 불가 |
| 음향 지평선 r_s | 실패 — 바리온 물리 의존, 이론 내부 아님 |
| 물질-DE 등분점 H_eq | 실패 — Ω_m, Ω_Λ 입력 필요, 순환 |
| Large diffeomorphism | 미결 — 위상 비자명 변환 가능성 |

**핵심 발견**:

1. **이론 내부 절대 스케일 고정 불가 — 구조적 특성**
   단순한 계산 부족이 아님. 순수 이론에서 절대 에너지 스케일을 뽑으려면 항상 외부 스케일(M_P, Λ, r_s, ...)이 필요함.

2. **상대적 구조는 완벽히 고정됨 (새 발견)**
   B=2π/ln2이면 w=-1 고정점들이 {H_ref, 2H_ref, 4H_ref, ...} — 정확히 2배(octave) 간격.
   **H₊/H₋=2 비율은 H_ref 무관한 이론 예측.** 이것이 관측 가능한 H_ref-독립 예측.

3. **H_ref 미고정 시 여전히 가능한 예측**
   - 진동 고정점 비율: H₊/H₋=2ⁿ
   - EOS 진동 패턴 형태 (B 고정에서)
   - A_obs/A_theory ≈ 1 (H_ref 무관)
   - 우주 역사에서 w=-1 교차 redshift의 비율

4. **실용적 대안: H_ref를 MCMC 자유 파라미터로 포함**
   관측이 H_ref를 결정. 단, 파라미터 3개(A, B, H_ref)로 AICc 패널티 증가.

**등급: D (확정)** — H_ref 이론 내부 고정 불가. "EE2는 이 점에서 예측 이론이 아니라 관측 결정 파라미터를 가진 모델"

## Run 3: P3 판정 — Z₂ 게이지/글로벌

**8인 팀 투표 결과: Z₂ 글로벌 5표, 게이지 0표, 판정 불가 3표**

**글로벌 방향 논증 4개 (독립 수렴)**:

1. **CPT 반유니타리성 (수학적)**
   게이지 대칭의 generator는 유니타리해야 함. CPT는 반유니타리 연산자 Θ → Hilbert space ray를 바꿀 수 있음 → CPT가 Z₂를 구현하는 방식은 게이지보다 글로벌에 가까움.

2. **S³ 스핀 구조 유일성 (위상수학적)**
   dS₄의 공간 단면 S³는 simply connected → 스핀 구조 유일. "두 스핀 구조" 문제는 실제로 스피너 위상의 전체 부호 선택(global Z₂)으로 환원. 내부 대칭 = 글로벌.

3. **A=2·e^{-π}의 "2"와 간섭항 계수 일치 (현상론적)**
   Z₂ 글로벌 → 선형합 → |M_+ + M_-|² → 간섭항 계수 2.
   L19 실측: A_free=0.088 ≈ A_theory=0.0864 (Δchi²=0.003, 이론값=최적값).
   이 "2" 인수가 수학적으로 정확히 일치.

4. **CP 위반 관측 (실험적)**
   입자-반입자 간섭이 실제 관측됨 → 두 상태가 물리적으로 구분 가능 → 게이지 동치 아님.

**남은 약점**:
- CPT ↔ dS₄ antipodal Z₂ 동일시의 엄밀 수학적 증명 미완
- CKM CP 위반과 dS₄ antipodal Z₂의 직접 구조적 연결 미완성

**등급: A₂ → B+** (글로벌 우세, 확정이 아님. 엄밀 증명 완성 시 A 격상 가능)

## Run 4: Euclidean→Lorentzian — Borel 구성적 도출

**핵심 성과: cos(B·ln H) 구성적 도출 성공**

**도출 경로**:

```
1. Euclidean dS (S⁴) 경로 적분 Z_E(g)의 Borel 변환:
   B[Z](s) ~ C · (s - s_*)^{iB-1}  (s_*=24π² 근방 분기점)

2. Borel 역변환:
   f(g) ≃ C · e^{-s_*/g} · g^{iB} · Γ(iB)
   g^{iB} = cos(B·ln g) + i·sin(B·ln g)

3. Lorentzian Wick 회전 g → ig_L:
   Borel 경로 [0,∞) → [0,i∞) 로 회전
   (ig_L)^{iB} = e^{-Bπ/2} · e^{iB·ln g_L}

4. 결과:
   Re[f] ∝ e^{-Bπ/2} · cos(B·ln g_L + const)
   g_L = H/H₀ 대응 → cos(B·ln(H/H₀)) 확립
```

**τ 계수 통일**:
- τ = 2πi/H (JT중력 = Gibbons-Hawking Wick 회전): B=2π/ln2의 2π 인수가 자연스럽게 도입
- τ = i/H (딜라톤 WKB): 2π 인수 누락 → 근사 수준으로 재분류
- **τ = 2πi/H 선택이 이론 내부 일관성에서 우선** (경로 A, B 합치)

**수치 검증 (L19)**:
- A_free=0.088 ≈ A_theory=0.0864 (Δchi²=0.003) ✓
- B_obs=8.765 vs B_theory=9.065: 3.4% 차이 (H_ref 불확실성 범위 내)

**미결**:
- A=2·e^{-π}의 e^{-π} 인수 완전 Borel 도출 미완
- 분기점이 s^{iB-1} 형태가 되는 동역학적 이유: 이산 스케일 불변성 assertion 수준

**등급: C → C+** (cos 구성적 도출 성공. A의 완전 도출 미완으로 B 미달)

## Run 5: 자기일관 ODE + 논문 초안 판정

**Part A: H(z) 해석적 추정**

- **최대 이탈 위치**: z≈0.3 (H=√2·H₀), 크기 ~2-3%
  → DESI BAO 측정 오차(~1-2%)와 경계 수준. 관측 구별 가능 신호.
- **진동 1주기**: H₀ → 2H₀ (z≈0→0.6). DESI DR2 관측 범위에 정확히 걸림.
- **z=0 초기조건**: cos(0)=1 → ω_DE=-1 → ΛCDM과 완전 일치.

**⚠️ 신규 치명 발견: 팬텀 횡단 문제**
```
ω_DE,min = OL0·(1 + 2A) = -1·(1 + 2×0.0864) = -1.1728
```
ω_DE < -1 구간 존재 → 팬텀 영역 진입.
단일 스칼라 k-essence로 팬텀 불가. Galileon, 이중 스칼라, 또는 CPT 이중항 구성 필요.
**이것이 이론의 추가 치명 약점으로 등록.**

**Part B: Bayesian 증거**

EE2의 k=0 구조:
- Occam 패널티 = 0
- 실제 Bayes factor: 2ln B ≥ ΔAICc = 10.6 → Jeffreys "Decisive evidence" 영역
- AICc보다 EE2에 더 유리한 구조

**Part C: 논문 초안 판정**

## Run 6: 팬텀 횡단 해결 — CPT 이중항 스칼라 구성

**집중 주제**: ω_DE,min = -1.17 < -1 문제. CPT 이중항과 팬텀 횡단의 이론적 연결.

### 8인 팀 토의 핵심

**[T1: 이론물리] 문제 정확 진단**
- ω_DE,min = OL0·(1 + 2A) = -1.1729 (cos=-1일 때 최솟값)
- H/H₀ = √2 (= exp(π/B))에서 최대 팬텀 이탈. ΛCDM 근사에서 z ≈ 0.61
- 단일 k-essence: K_X > 0 (유령 없는 조건) => w ≥ -1 => 팬텀 구현 원리적 불가

**[T2: 장이론] Horndeski/Galileon 경로**
- Horndeski 이론 (G2 + G3·□φ + G4·R + ...): 브레이딩(braiding) 항 G3(φ,X)·□φ가 유효 w < -1 허용
- cs² > 0 유지 가능 (유령-없는 팬텀)
- EE2 cos(B·ln H) 구조와 직접 연결: **미확립**

**[T3: CPT 구조 분석]**
- Run 3 확립: Z_total = Z[α] + Z[β] = 2 Re[Z[α]]
- 경로 적분 수준의 CPT(스핀 구조 교환)는 실제 스칼라장 CPT 변환이 아님
- CPT 이중항이 자동으로 Quintom 이중 스칼라를 생성하지 않음: **구조적 공백**

**[T4: Quintom 이중항 분석]**
- Quintom = phi₊(quintessence) + phi₋(phantom) 조합 (Feng, Wang, Zhang 2005)
- L = +(1/2)(∂φ₊)² - (1/2)(∂φ₋)² - V(φ₊, φ₋)
- w_eff = (p₊ + p₋)/(ρ₊ + ρ₋) < -1 가능
- CPT 이중항과의 대응: phi₊ = Z[α] 스칼라, phi₋ = CPT(Z[α]) 스칼라 제안 가능
- 문제: phi₋는 양자 불안정 (음의 운동 에너지, UV completion 없음)

**[T5: 수리물리] 신규 구조 발견 — K(X) = X·cos(B·ln X)**
- CPT 이중항 Re[Z[α]] 구조에서 복소 k-essence 실수부:
  K(X) = Re[X^{1+iB}] = X·cos(B·ln X)
- X를 H/H₀에 대응시키면 EE2의 cos(B·ln(H/H₀)) 구조와 직접 대응!
- **신규 구조적 발견**: CPT Re[Z] 구조 → 유효 k-essence 라그랑지안의 자연적 형태

**[T6: 안정성 검사] K(X) = X·cos(B·ln X) 유령 분석**
- K_X = cos(B·ln X) - B·sin(B·ln X)
- B ≈ 9.065에서 X 범위 [0.01, 3.0] 전체에서 K_X < 0 구간 존재
- cs² < 0 불안정 구간 발생
- **결론: K(X) = X·cos(B·ln X)는 B~9에서 유령 불가피. 직접 사용 불가.**

**[T7: Horndeski 대안] 브레이딩 구조**
- L = X - V(φ) + (c₃/Λ³)·X·□φ

## Run 7: A = 2e^{-π}의 e^{-π} 인수 완전 Borel 도출

e^{-Bπ/2} = 6.55×10⁻⁷  (Run 4 Borel Wick 회전 결과)

### 8인 팀 토의 핵심

**[P1: 위상 적분] Γ(iB)의 위상 분석**
- Γ(iB) 계산: |Γ(iB)| = 5.45×10⁻⁷, arg(Γ(iB)) = -2.444 rad = -0.778π
- Stirling 근사: Γ(iB) ~ √(2π/B)·exp(-Bπ/2)·exp(i·위상)
- Γ(iB)의 위상은 -0.778π. e^{-π} 인수를 직접 생성하지 않음.

**[P2: Borel 변환 재분석] Run 4 결과 검증**
- Lorentzian Wick 회전: g → ig_L
- (ig_L)^{iB} = exp(-B·ln(ig_L)) = exp(-B·ln g_L)·exp(-i·Bπ/2)
- 실수부: exp(-Bπ/2)·cos(B·ln g_L)
- 이것은 Run 4 결과와 일치. e^{-π} 생성 불가.
- **Borel Wick 회전 경로에서 exp(-π)를 얻으려면 B_eff = 2 필요. 이론의 B = 9.065 >> 2 → 원리적 실패.**

**[P3: RP⁴ 위상 불변량 경로] S_cl = π와 연결**
- 제안: A = 2·exp(-S_cl)이고 S_cl = π이면 A = 2·exp(-π) ✓
- S_cl = π의 제안된 기원:
  - RP⁴ = S⁴/Z₂의 기본군 π₁(RP⁴) = Z₂ → 비자명 루프 작용 = π
  - dS₄/Z₂ 반대점 동일화에서 경로 길이
  - JT 중력: AdS₂ 경계 작용 S_bdy = φ_r·χ_disk = π

**[P4: 논증 구조 평가]**

  구조적으로 자연스러움
- 미완성: S_cl = π 자체의 독립 정량적 유도 없음
  - RP⁴ 경로: "비자명 루프 작용 = π" 계산이 assertion 수준
  - JT 중력 경로: φ_r = π는 외부 정규화 가능성

**[P5-P6: Γ(iB) 정밀 분석 + B_eff 요구사항]**

**[P8: 세 경로 최종 평가]**

## Run 8: P3 엄밀 증명 — 스핀 구조 Z₂ 판정 심화

**집중 주제**: Run 3에서 글로벌 5/8 투표 확립 후 미완성 항목 심화. CPT ↔ dS₄ antipodal Z₂ 동일시 수학적 엄밀화.

### 8인 팀 토의 핵심

**[Q1: 스칼라 수준 재확인] Θ = J 확인**
- CPT 연산자 Θ (반유니타리, 반선형)
- dS₄ antipodal 사상 J: x → -x (de Sitter 쌍곡면에서)
- 스칼라장 수준: Θ[φ(x)] = φ*(Θx) = φ*(-x*) = J[φ]
- **결론: 스칼라 수준에서 Θ = J 확정. 논란 없음.**

**[Q2: 스피너 수준 분석] 위상 모호성**
- 스피너 변환: Θ[ψ(x)] = γ⁰γ²K·ψ(J·x) (Wick 회전 포함)
- J의 스피너 표현: J* = exp(iπs)·J_scalar (스핀-s에서 위상 exp(iπs) 추가)
- 반정수 스핀 (s=1/2): exp(iπ·1/2) = i → 위상 인자 i 모호성 발생
- **미결: 이 i 위상이 Z[β]=Z[α]* 관계에 영향을 주는가?**

**[Q3: 정수 스핀 보조 장 분석]**
- s=0 (스칼라), s=1 (벡터): exp(iπs) = ±1 → 위상 ±1
- Z[β] = ±Z[α]* → A₂ = 2Re[Z[α]] 또는 A₂ = 0
- s=1 케이스: A₂ = 2Re[Z[α]] = ±2|Z[α]| (부호 정해짐 시)
- 관측 AICc = 10.8: A₂ > 0 관측 → s=1 위상 +1 선택 → 글로벌 Z₂ 논리 유효

**[Q4: 반정수 스핀 (페르미온) 분석]**
- s=1/2: Θ² = -1 (반정수 스핀 Kramers 이중항)
- CPT로 연결된 두 스피너: ψ_+와 ψ_- = Θψ_+
- Θ² = -1 → Kramers 이중항 → 두 상태가 에너지 동일 (에너지 겹침)
- 분배함수: Z_total = Z_+ + Z_- = 2Z_+ → A₂ = 2 (Kramers 이중항에서 자동!)
- **신규 발견: 페르미온의 Kramers 이중항이 A₂ = 2의 독립적 추가 근거 제공**

**[Q5: 관측적 제약 연결]**

- **실험이 이미 글로벌을 지지하는 간접 증거**

**[Q6-Q7: 약점 목록 작성]**

- Ramanujan π 공식 계수 (1103, 26390): B와 수론적 관계 없음
- Hardy-Ramanujan 1729과 B×190.74: 우연적 근사
- (θ₃/θ₄)² = √2 = 2^{1/2}: ln2 연결 불연속 (ln√2 ≠ ln2)
- Dedekind η(2i)/η(i): ln2와 무관 (비율 -0.186)
- E₆(i) = 0 확인 (τ=i 모듈러 특수성), ln2 없음
- CM 이론 (k=1/√2, 렘니스케이트 케이스): A 기원 심화, B 연결 없음

**추가 확인**:
- A = 2·nome(τ=i) = 2e^{-π}: 완전 확정 (τ=i 타원 nome)
- k = (θ₂/θ₃)² = 정확히 1/√2: 렘니스케이트 특수값 확인

**결론: A와 B를 동시에 결정하는 단일 수학 구조 없음**
- A의 기원: 타원함수론 (복소 분석, 모듈러 형식, τ=i)
- B의 기원: 정보이론 (이진 엔트로피, ln2 = S_min)
- 이 두 세계를 잇는 단일 구조: **미발견**

**등급: F** (단일 구조 탐색 완전 실패)
