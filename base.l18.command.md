## 배경

L15-L17을 통해 EE2가 최적 수식으로 확정됨.
- chi2_13pt=5.601, AICc=14.268, delta_joint=-22.9
- w0=-0.874, wa=-0.307
- 최적 파라미터: Om≈0.305, A≈0.088, B≈8.76

**미해결 질문**: A와 B는 무엇인가?
- 자유 피팅 파라미터인가, 아니면 자연 상수로부터 도출 가능한가?
- A1+A2 공리로부터 이 값들이 나와야 하는 이유가 있는가?

## Phase 1: MCMC 정밀 posterior

### 설정
- 모델: EE2 (k=3), 파라미터: (Om, A, B)
- B: log-uniform 공간 샘플링 (B > 0 강제)
- nwalkers=32, burn=800, prod=1200
- 수용률 목표: 0.05 ~ 0.5

### 파라미터 범위
- Om: [0.25, 0.40]
- A: [0.0, 0.5]
- B: log([0.1, 50]) → log-uniform

### 출력
- A, B 1D 사후분포 (median + 1sigma)
- Om-A, Om-B, A-B 2D 상관관계 corner plot
- chi2_13pt at MCMC median vs best-fit 비교

## Phase 2: 상수 단위 분석

### B ≈ 8.76 분석 대상

B는 무차원수. 가능한 자연 스케일:

1. **ln(z_eq+1)**: 물질-복사 동등점 z_eq≈3400 → ln(3401)≈8.13
2. **ln(z_cmb+1)**: 재결합 z_cmb≈1090 → ln(1091)≈6.99
3. **N_matter**: 물질 우세 기간의 e-fold 수
4. **1/OL0**: 1/0.695≈1.44 (해당 없음)
5. **π²**: 9.87 (근사)
6. **e³**: e^3≈20.09 (해당 없음)
7. **H0*t_Planck 관련 조합**
8. **OL0/Om 비율의 로그**: ln(OL0/Om)=ln(2.28)≈0.82 (해당 없음)
9. **e-fold from matter-DE equality to today**: N_mde 계산
10. **A1 이론: 소멸률/창조률 비 로그**

### A ≈ 0.088 분석 대상

1. **Om/(1+Om)**: 0.305/1.305≈0.234 (해당 없음)
2. **Om*OL0**: 0.305*0.695≈0.212 (해당 없음)
3. **(Om/OL0)^(2/3)**: (0.439)^(2/3)≈0.578 (해당 없음)
4. **Omega_b/Om**: 0.048/0.305≈0.157 (근사)
5. **자연단위 결합: G*H0²/(c^4) 류**
6. **A1 이론: 소멸 진폭 — n0μ 결합**

### B/A 비율
- B/A ≈ 8.76/0.088 ≈ 99.5 ≈ 100
- 이것이 우연인지 구조적 관계인지 분석

## Phase 3: 8인 상호토의 — 이론적 근거

### 공통 필수 원칙 (CLAUDE.md)
- 역할 사전 지정 금지. 8인 서로 중복되지 않는 자유 배경.
- 방향만 제공. 지도 극단적 금지.
  - 이전 L16 발견 (Berry phase, Josephson, e-fold) 힌트 일체 없음
  - "ln(z_eq)" 등 Phase 2 분석 결과 힌트도 없음
- 과적합 방지: 사후 맞춤 금지.

### 제공하는 것 (오직 이것만)
1. A1+A2 공리
2. EE2 수식: omega_de = OL0*(1+A*(1-cos(B*ln(H/H0))))
3. 관측 최적값: A≈0.088, B≈8.76
4. 질문: "A와 B가 이 값이어야 하는 이유가 A1+A2로부터 있는가?"

### 수렴 기준
- 8인 중 5인 이상 동일 도출 경로 → 성공
- 10회 이상 반복, 과반 성공 → 이론 상수 확정

### 특별 금지 사항
- "데이터가 이 값이니 이론도 이 값이어야 한다" 역공학 금지
- Phase 2의 ln(z_eq) 등 유사값 힌트 금지
