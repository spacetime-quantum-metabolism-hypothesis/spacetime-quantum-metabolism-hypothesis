## 핵심 발견: L19의 수학적 오류

### CRITICAL ERROR — w₂(RP⁴) = 0

L19는 다음을 주장했다:
```
Z(RP⁴) = (-1)^{∫w₂²} = e^{iπ}
```
이 공식은 **수학적으로 틀렸다.**

**계산 (Whitney product formula)**:
```
w(RP⁴) = (1+a)^5 mod 2
= 1 + C(5,1)a + C(5,2)a² + C(5,3)a³ + C(5,4)a⁴
= 1 + a + 10a² + 10a³ + 5a⁴
≡ 1 + a + 0·a² + 0·a³ + a⁴  (mod 2)
```
따라서 **w₂(RP⁴) = 0** (mod 2).

결론: `(-1)^{∫w₂²} = (-1)^0 = +1`, **not e^{iπ}**.

### 올바른 TFT: Dijkgraaf-Witten Z₂ 게이지 이론

**정확한 TFT**: 4D Dijkgraaf-Witten Z₂ 게이지 이론 with non-trivial 4-cocycle

```
α ∈ H⁴(BZ₂; R/Z) = Z₂
Z(RP⁴) = e^{2πi⟨α,[RP⁴]⟩} = e^{2πi × 1/2} = e^{iπ}  ✓
```

이유: RP⁴ = S⁴/Z₂에서 Z₂ principal bundle의 최소 홀로노미가 
⟨α,[RP⁴]⟩ = 1/2 (mod 1)을 결정.

**S_cl = π는 여전히 올바르다** — 단, 도출 공식이 바뀌었다.

**장벽 1 — Wick rotation 방향 역전**:
표준 결과: Euclidean에서 a(τ) = H⁻¹ cos(Hτ) → Lorentzian: a(t) = H⁻¹ cosh(Ht)
cos가 cosh가 된다. EE2는 Lorentzian에서 cos가 필요하므로 반대 방향. 극복 불가.

**장벽 2 — 시간적 단절**:
S⁴/RP⁴ instanton은 우주 초기(Planck 스케일)에서 정의.
EE2 공식은 오늘날 H_today에 의존.
두 사이를 연결하는 동적 메커니즘: 없음.

**장벽 3 — 이산 스케일 불변성의 기원 없음**:
B = 2π/ln2 → H → 2H마다 cos가 한 주기를 도는 구조.
S⁴/RP⁴의 어떤 표준 계산에서도 이 구조가 나타나지 않음.

**장벽 4 — A, B, S_cl의 Euclidean 기원 미확립**:
- S_cl = π: de Sitter에서 S_E = π/GH² (H-의존). 상수 π로 고정되는 이유 없음.
- A = 2e^{-π}: "2"의 조합론적 기원 없음.
- B = 2π/ln2: RP⁴ = S⁴/Z₂에서 ln2가 나오는 표준 경로 없음.

검토한 후보 (모두 실패):
- 복소 saddle 간섭: cos(B·ln H) 형태 도출 불가

- EFT RG running: 완전 추측, 표준 경로 아님
