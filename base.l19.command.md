## 배경

L18에서 확정된 것:
- EE2: omega_de = OL0*(1+A*(1-cos(B*ln(H/H0))))
- B = 2π/ln(2) = 9.0644 (A등급, 4경로 수렴)
- A = 2·e^(-π) = 0.0863 (B+등급)

**미결 과제**:
1. **A₂ = 2 vs √2** — 이론적으로 해결 가능한가? 아니면 DESI full/Euclid 필요?
2. **S_inst = π UV-완비 도출** — 이것이 해결되면 A가 A등급으로 승격
3. 양자 간섭 위상차 — 우선순위 낮음

## Phase 2: A₂ = 2 vs √2 이론 해결 (8인, 10회)

### 핵심 긴장
- 위상적 논증: A₂ = 2 (π₄(S⁴) = Z₂, instanton+anti-instanton 계수)
- Coleman 1-루프: A₂ = √2 (행렬식 보정 (S/2π)^(1/2) = 1/√2 추가됨)

실제 두 계산의 관계:
- Coleman: A = (S_inst/2π)^(1/2) × 2 × e^(-S_inst)
  = (π/2π)^(1/2) × 2 × e^(-π) = √2 · e^(-π) = 0.0611
- 위상적: A = 2 × e^(-π) = 0.0863
- 차이: (S_inst/2π)^(1/2) = 1/√2 인자

### 질문: 이 인자가 EE2 진폭에 들어가는가?

### 제공하는 것
1. A1+A2 공리
2. EE2 수식
3. 관측값 A = 0.089 ± 0.037
4. 이론 후보 두 값: 0.0863 (2e^-π), 0.0611 (√2·e^-π)
5. 질문: "A₂를 결정하는 물리적 원리는 무엇인가?"

## Phase 3: A 고정 시뮬레이션

### 목적
현재 13pt BAO 데이터로 두 이론값을 구별할 수 있는가?

### 방법
- EE2 수식에서 A를 고정하고 (Om, B)만 최적화
- 케이스 1: A = 0.0863 (= 2e^-π)
- 케이스 2: A = 0.0611 (= √2·e^-π)
- 케이스 0: A 자유 (현재 최적: A=0.088, chi2=5.601)
- chi2 비교 → 현재 데이터의 판별력 정량화

### 기준
- Δchi2 < 1.0: 현재 데이터로 구별 불가 → DESI full/Euclid 필요
- Δchi2 > 4.0: 하나가 유의하게 선호됨
