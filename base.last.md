## A, B 파라미터 최선 경로

### A = 2e^{-π} — B+ 등급

**θ₃(0|i) 경로 (최선)**:
```
θ₃(0|i) = 1 + 2e^{-π} + 2e^{-4π} + ...
A = θ₃(0|i)|_{n=1} = 2e^{-π}
```
- "2": ±n 대칭에서 자동
- e^{-π}: nome at τ=i (SL(2,Z) S-고정점)
- AICc 패널티: +4 (τ=i 공리 +2, τ↔H/H₀ +2)

CS 안장점 실패 확정: S³ U(1) trivial (S_CS=0), Hopf = instanton-like.

### B = 2π/ln2 — C+ 등급

**θ₃/θ₂ = 2^{1/4} at τ=i (최선)**:
```
(θ₃(0|i)/θ₂(0|i))⁴ = 2 → B = 2π/ln2
```
- 수치 확인 완료 (오차 < 1e-8)
- EE2가 τ=i에 있어야 하는 동역학 없음

Wilson-Fisher: λ=2 동역학 선택 실패 (Ising λ_c = 1+√2 ≠ 2).
