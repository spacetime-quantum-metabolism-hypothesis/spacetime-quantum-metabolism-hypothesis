# L55 — 공리에서 현상론까지 통합 감사 (Audit)

> 작성: 2026-04-26
> 목적: 시공간 양자 대사 가설(SQT/SQMH)을 공리 → ODE → 도출 관계식 → 현상론 예측 → 관측 비교의 다섯 단계로 분해해 *각 단계에서 도입된 추가 가정과 패치*를 추적하고, **개선 가능 지점**을 작업 가능한 형태로 식별한다.
> 본 감사는 "마지막 통합 시도"가 아니다. 이론을 살리거나 죽이려는 시도가 아니라, *지금 이 순간 정직하게 어디에 구멍이 있는지*를 보는 작업이다.
> 8인팀 모드(P1~P5 + C1~C4) 적용. C팀 전원 강력 비판 모드.

## 0. 결론 요약 (먼저 읽을 부분)

- 공리 6개 자체에는 *수식적 모호성*이 있다 (a1과 a6의 결합/분리, a4-a5의 순환). 그러나 정의 추가로 보강 가능한 수준이다.
- 공리 → ODE 단계에서 **흐름 속도장 v(x,t), Λ_eff, ε**의 세 가지 비공리적 요소가 사일런트하게 추가되어 있다.
- 도출 관계식 단계에서 **τ_q의 이중 정의 (Planck 시간 vs Hubble 시간)** 가 60 dex 충돌을 만든다. 이것이 가장 큰 내부 모순이다.
- 현상론 단계에서 **σ_0(k), σ_0(z), friction_factor 계수 0.1, β, h_1**은 모두 axiom-level 도출이 없는 사후 패치다.
- L48~L54 시뮬은 "단일 σ_0 4 검증 동시 만족 불가"를 강하게 보여주며, σ_0(k) (L53), σ_0(z) (L54) 둘 다 χ² > 50으로 4 검증 영역을 동시에 못 잡는다.
- 이론은 "구조적 반증"보다 "구조적 미완성"에 가깝다. 공리는 살아있지만 공리에서 ODE까지의 연결 다리가 흔들린다.

**정직 별점 갱신**: ★★☆☆☆ (이전 L53 직후 강등 유지. L54 σ_0(z) 사망으로 변동 없음.)

## 1. 단계 1 — 공리 a1~a6 자체 점검

### 1.1 현재 상태 (정직)

```
a1: 양자 + 물질 → 소멸 (메커니즘)
a2: 양자 에너지 = 흡수된 에너지 (보존)
a3: 평균 우주공간 균일 생성 (등방성)
a4: 공간 = 양자 패턴 (β-해석)
a5: 물질 = 안정 양자 패턴 (실체)
a6: 패턴 유지율 ∝ 패턴 에너지 (선형 흡수)
```

### 1.2 발견된 문제

**(A) a1과 a6의 부분 중복/불완전 분리**
- a1은 "소멸이 있다"는 *정성적* 진술. a6은 "소멸률이 패턴 에너지에 선형 비례"라는 *정량적* 진술.
- 그러나 a1만으로는 2체 반응 (∝ n·ρ_m) 형태가 강제되지 않는다 — 1체 반응 (∝ n)이거나 임의의 비선형 (∝ n^a·ρ_m^b)이어도 a1을 만족한다.
- 따라서 ODE의 "σ_0·n·ρ_m" 항은 *a6에서 부분적으로* 도출되며, 추가로 "패턴 에너지 = ρ_m·c²"이라는 가정을 더 필요로 한다 (= a5에서 암묵).
- **이는 도출 가능한 형태의 모호성이지, 명시된 공리는 아니다.**

**(B) a3의 "균일"이 시간 불변성을 포함하는지 미명시**
- "평균 우주공간 균일 생성"은 *공간적* 등방성/균일성. *시간적* 불변성은 별도 진술.
- L54가 σ_0(z) 모델 V를 *허용*해도 a3에 충돌하지 않음 — 왜냐면 a3는 σ_0가 아니라 Γ_0의 공간 균일성에 관한 진술.
- 이는 "σ_0가 시간 의존하면 어디 공리에 충돌하는가?"라는 질문을 *원리적으로 미정* 상태로 둔다.
- **C2(차원수치) 의견**: 공리에 "σ_0는 본질 상수"가 명시되어야 한다. 그렇지 않으면 σ_0(t), σ_0(k)는 모두 "공리 외부 패치"이고, 이론은 무한한 자유도를 비밀리에 보유한다.

**(C) a4-a5의 순환**
- a4: "공간 = 양자 패턴" — 공간 자체가 패턴.
- a5: "물질 = 안정 양자 패턴" — 물질도 패턴.
- a1: "양자 + 물질 → 소멸" — 패턴이 패턴을 소멸시킴.
- 결과: "공간 패턴이 물질 패턴을 소멸 → 공간을 흡수" 형태로 읽히지만, 공간 자체가 패턴이면 "공간이 자기 자신을 흡수"의 자기참조 구조가 생긴다.
- 이는 BEC 응축체로 해석할 때(base.md §VI) 해소되지만, *공리 수준에서는 미해소*.
- **C1(자기일관) 의견**: 공리 수준에서 "양자(quantum)"와 "패턴(pattern)"의 관계를 명시하는 추가 공리(a0?)가 필요하다.

**(D) 누락된 공리들**
1. *τ_q의 정체*: 매질 고유 시간 (단위 [s])가 어디서 오는지 공리에 없음. 도출 관계식 G = σ_0/(4π·τ_q)의 우변 분모에 들어가는 τ_q는 본질적으로 추가 공리 없이 결정 불가.
2. *Λ_eff의 기원*: 수정 Friedmann의 Λ_eff/3 항은 공리에서 도출되지 않음. base.md §IV는 V(φ) 라그랑지안에 ρ_Λc² 상수를 *손으로* 넣는다.
3. *ε(양자 에너지)의 정체*: ε는 개별 양자의 에너지인데, "ε = 3H_0²Ω_Λc²/(8πG·n_∞)" (L48) 식은 *Λ_eff와 일치시키도록 사후 결정*된다. 즉, ε는 자유 파라미터이며 a2의 "흡수 에너지"와 직접 동일시 불가.

### 1.3 개선 가능 지점 (작업 가능 형태)

| 번호 | 항목 | 작업 형태 |
|---|---|---|
| **S1-1** | a6의 정확화 | "패턴 유지율" 대신 "단위 시간당 단위 패턴 에너지 흡수율은 양자 수밀도와 패턴 에너지 밀도에 *각각 독립적으로 일차*다"라고 다시 쓰기. 이러면 σ_0·n·ρ_m 형태 강제. |

### 1.4 개선이 본 이론의 다른 부분에 미치는 영향

- **S1-2 채택 시 (가장 강력)**: σ_0(k), σ_0(z) 패치가 모두 *공리 위반*이 됨 → L48~L54 사망 인정 강제. 이론은 "단일 σ_0 SQMH"로 한정되어 *반증된 상태*로 정직하게 기록.
- **S1-2 거부 시 (약한 이론)**: σ_0(k), σ_0(z)는 *허용*되지만, 함수 형태에 대한 추가 자유도 패널티(AICc) 명시 필요.
- **S1-4 채택 시**: τ_q의 이중 정의 문제(§3 참조)가 *공리 수준에서* 해소된다. 단, τ_q = 1/(3H_0) 자기일관 조건은 여전히 별도 도출 필요.

### 1.5 8인팀 의견

- **P1(이론)**: a6이 a1을 강하게 함의하므로 a1은 *별도 공리가 아닐 수 있다*. 5개 공리로 축약 검토 필요.
- **P2(수리)**: a4-a5 순환은 BEC 형식화에서 해소되지만, *공리 수준 자기일관*을 위해 a0 필요.
- **P3(실험)**: 공리에 "σ_0 = 본질 상수" 강화하면 L48 즉시 사망. 거부하면 σ_0 자유도 변형 패널티 *반드시* 부과.
- **P4(우주론)**: Λ_eff와 ε의 비공리적 도입이 핵심 약점. base.md §IV 라그랑지안 형식이 V(φ) 항을 *손으로* 넣는 점 정직 인정.
- **P5(응집)**: a4-a5 순환은 GFT BEC에서 자연 해소 — 응축체 위 여기 패턴이 응축체 자신을 *국소* 변형. 그러나 이것은 공리가 아닌 모델.
- **C1(자기일관, 강력 비판)**: "공리 6개"라고 부르지만 *진짜 자기일관 공리*는 a2, a3, a6의 셋. 나머지는 정의 또는 메커니즘 진술. 공리 카운트 정직성 미달.
- **C2(차원수치, 강력 비판)**: τ_q [s], ε [J], μ [kg], n_0 [m⁻³]가 공리 어디에도 명시 정의 없음. *모두 도출 관계식에서 사후 등장*. 이는 "공리 → 도출"이 아니라 "도출 → 사후 공리화"의 역방향.
- **C3(관측, 강력 비판)**: 공리 6개로 *어떤 관측량을 직접 예측*할 수 있는가? (없다. ODE를 거쳐야 한다.) 따라서 공리는 *예측력이 없는* 진술이고, 모든 예측력은 ODE와 추가 가정에서 나온다. 공리의 가치는 *제약*뿐.
- **C4(자유도, 강력 비판)**: σ_0를 본질 상수로 고정하지 않는 한, σ_0(x, t, k)의 *모든 함수 자유도*가 사실상 자유 파라미터. "0개의 자유 파라미터"라는 base.md §1.3 주장은 *σ_0를 함수 자유도로 풀면 즉시 거짓*.

## 2. 단계 2 — 공리 → ODE 도출 점검

### 2.1 현재 상태 (정직)

핵심 ODE (L48):
```
dn/dt   = Γ_0 - 3Hn - σ_0·n·ρ_m
dρ_m/dt = -3Hρ_m + σ_0·n·ρ_m·ε/c²
H²      = (8πG/3)(ρ_m + n·ε/c²) + Λ_eff/3
```

### 2.2 공리 → ODE 도출에서 추가된 가정

| 항 | 공리에서? | 추가 가정 | 비고 |
|---|---|---|---|
| `dn/dt` LHS | a1 (메커니즘) + 정의 | 시간 미분이라는 *연속체 가정* | discrete 인과집합에서 연속 한계 진입 |
| `Γ_0` (생성) | a3 (균일 생성) | 상수 (시공간 독립) | 명시 |
| `-3Hn` (희석) | (없음) | FRW 균질 우주 가정 | 공리에 *우주 모델 없음* |
| `-σ_0·n·ρ_m` (소멸) | a6 (선형) | 2체 반응 형태 | a1+a5 결합 |
| `dρ_m/dt` LHS | (없음) | 물질 밀도 시간 진화 | 공리에 *물질의 시공간 분포 진술 없음* |
| `+σ_0·n·ρ_m·ε/c²` | a2 (보존) | 흡수 에너지 = 패턴 에너지 게인 | ε의 정체 미정 (S1-5 참조) |
| `H² = ...` (Friedmann) | (없음) | GR + FRW | 공리는 GR을 함의하지 않음 |
| `Λ_eff/3` | (없음) | 우주상수 (V(φ)에서) | base.md §IV 라그랑지안에서 *손으로* |

### 2.3 사후 도입된 패치 (이론이 데이터를 만나며 추가됨)

| 패치 | 도입 시점 | 형태 | 공리 도출? |
|---|---|---|---|
| **σ_0(k)** | L50, L52 | `σ_0(k) = (σ_IR + σ_UV(k/k_t)²)/(1+(k/k_t)²)` | ❌ 없음 |
| **σ_0(z)** | L54 | `σ_0(z) = σ_now + (σ_late-σ_now)·0.5(1+tanh((z-z_t)/Δz))` | ❌ 없음 |
| **friction_factor(z)** | L52, L54 | `1 + 0.1·log10(σ(z)/σ_ref)` | ❌ 없음 |
| **β** (a_0 비례) | L49 | `a_0 = β·c/τ_q`, β ≈ 0.035 | ❌ 없음 (피팅) |
| **h_1** (V11 환경) | L49 | `a_0_local = a_0·(1 + h_1·δn/n_∞)` | ❌ 없음 (현상론) |
| **σ_0(global) → σ_0_T17, σ_0_T20, σ_0_T22** | L48~L49 | 4개 σ_0 값 | ❌ 모순 (단일 공리 위반) |

### 2.4 개선 가능 지점

| 번호 | 항목 | 작업 형태 |
|---|---|---|
| **S2-1** | FRW 가정 명시 | ODE의 -3Hn, dρ_m/dt 항이 *FRW + GR*에서 오는 점을 별도 보조정리로 분리. 일반 메트릭에서는 N(x,t) 보존류 형태로 다시 쓰기. |
| **S2-2** | Λ_eff 도출 | Λ_eff = (Γ_0 - σ_0·n_0·ρ_m,0)·ε·8πG/(3c²) 형태로 *대사 잔차*에서 직접 도출 (base.md §1.4 가정 1과 일치). 라그랑지안 V(φ) 자유 항 제거. |
| **S2-3** | ε 도출 또는 자유도 표시 | ε를 자유 파라미터로 *명시 인정* OR 양자 BEC 미시론에서 ε = ℏω_q (양자 응축 모드 에너지)로 도출. 후자는 base.md §VI에서 부분 시도. |
| **S2-4** | σ_0(k), σ_0(z) 패치 *공리 위반* 표시 | S1-2 채택 시 자동 — 이 둘은 공리 위반이므로 제거. *대신* 공리 보강(S1-4, S1-5)으로 동등 자유도 흡수 시도. |
| **S2-5** | β, h_1을 ODE에서 직접 도출 | β = a_0·τ_q/c는 정의에서 자동. 그러나 a_0 자체는 ODE의 비선형 정상상태에서 *도출되어야* 한다. base.md §III.1의 v(r) = σM/(4πr²) 흐름은 deep-MOND 한계와 직접 연결되지 않음. 이 다리 누락. |
| **S2-6** | 4개 σ_0 가치 통일 또는 분해 인정 | T17(2.34e8), T17_sc(1.17e8), T20(5.6e7), T22(3.31e9). 이를 단일 σ_0로 묶지 못하면 *4개 다른 σ를 4개 다른 영역에 적용한다*고 정직 인정 (= 효과적 IR-UV 흐름이지만 단일 함수형 부재). |

### 2.5 영향

- **S2-2 채택**: V(φ) 라그랑지안의 우주상수 항 사라짐 → coupled QE 부분집합 주장 약해짐. 그러나 *공리 자기일관성 향상*. 트레이드오프.
- **S2-4 채택 + S1-2**: L48~L54 패치 모두 무효화. SQMH의 단일 σ_0 형태는 공식적으로 4 검증 동시 통과 불가 → 정직 사망 인정.
- **S2-5 채택**: SPARC 회전곡선 도출의 미시 다리(BEC 흐름 → MOND 가속도 한계)가 *증명 필요 항목*에 추가됨.

### 2.6 8인팀 의견

- **P2(수리)**: dρ_m/dt에 +σ_0·n·ρ_m·ε/c² 항이 들어가는 것은 *질량-에너지 동등성 + a2*에서 자동. 그러나 ρ_m이 *질량 밀도*인지 *에너지 밀도*인지 모호. SI 차원 [kg/m³] 채택 시 ε/c² 인수가 필수.
- **P4(우주론)**: H² 식의 Λ_eff/3 항이 V(φ) 라그랑지안과 결합 → 라그랑지안 형식이 ODE 형식과 *일관되게 정합되지 않음*. base.md §4.1의 V(φ) = ρ_Λc² + (1/2)m_φ²c²(φ-φ_0)² 중 첫 항이 Λ_eff. 그러나 Λ_eff는 ODE에서 *대사 잔차*에서 자동 도출 가능 — 이중 카운트 위험.
- **P5(응집)**: σ_0(k) Lorentzian은 *BEC 응축체 음향 분산*과 형태적으로 유사 (롱-파장 IR vs 단-파장 UV 분기). 공리 도출은 안 되지만 *자연스러운 미시론적 후보*. 단, 이것이 σ_0(k)의 *물리적 정당화*가 되려면 GFT 응축체에서 직접 도출 필요.
- **C1(강력 비판)**: σ_0(k) Lorentzian 함수형은 *데이터에 맞춰 골랐다*는 자백을 피할 수 없다. "BEC 응축체의 자연스러운 형태"는 *사후 합리화*에 가깝다. CCC.
- **C2(강력 비판)**: ε의 정체 미정은 차원-수치 자기일관 결정타. ε [J]는 ODE에서 *자유 파라미터*인데, base.md §1.3은 "자유 파라미터 0개"라 주장. 모순.
- **C3(강력 비판)**: 4개 σ_0 값은 1.4 dex 차이 — 이는 "한 자리 정도 차이" 수준이 아니라 *25배 ~ 1000배* 차이. 단일 σ_0로 묶을 수 있다는 주장은 *경험적으로 거짓*.
- **C4(강력 비판)**: friction_factor = 1 + 0.1·log10(σ/σ_ref)의 계수 0.1은 *완전 임의*. 이론에서 어떤 도출도 없으며, 데이터 매칭을 위한 손-튜닝. 자유도 +1 패널티 명시 필요.

## 3. 단계 3 — ODE → 도출 관계식 점검

### 3.1 현재 상태 (정직)

```
G = σ_0/(4π·τ_q)        # L48 명시 (시나리오 A)
τ_q = 1/(3H_0)           # 자기일관 조건 (L48)
n_∞ = Γ_0·τ_q            # 정상상태
ε = 3H_0²·Ω_Λ·c²/(8πG·n_∞) # Λ_eff 일치 사후 결정
a_0 = β·c/τ_q            # MOND 가속도 (L49)
```

또한 base.md (§3.3)에는:
```
G = n_0·μ·σ²/(4π)        # 포텐셜 흐름 매칭
σ = 4πG·t_P              # SI 차원 일관 (Planck 시간!)
n_0·μ = ρ_P/(4π) ≈ 4.1e95 kg/m³
```

### 3.2 가장 큰 모순 — τ_q의 이중 정의

직접 수치 계산:
- base.md SI 정의: σ = 4πG·t_P = 4π × 6.674e-11 × 5.39e-44 ≈ **4.52×10⁻⁵³** m³/(kg·s)
- L48 자기일관 정의: σ_0 = 4πG·τ_q (τ_q = 1/(3H_0) ≈ 1.39e17 s) ≈ **1.17×10⁸** m³/(kg·s)
- **두 값의 비**: 1.17e8 / 4.52e-53 ≈ **2.6×10⁶⁰** (= 60 데시멀 자릿수 차이)

이는 base.md의 σ와 L48의 σ_0가 *서로 다른 물리량*임을 의미한다. 같은 기호 σ를 두 다른 의미로 쓰는 *기호 충돌*.

| 표기 | base.md (Lamb 1932 매칭) | L48 (자기일관 시나리오 A) |
|---|---|---|
| 정의 | σ = 4πG·**t_P** (Planck 시간) | σ_0 = 4πG·**τ_q** (Hubble 시간) |
| 값 | ~4.5e-53 m³/(kg·s) | ~1.17e8 m³/(kg·s) |
| 도출 | 포텐셜 흐름 vs 뉴턴 매칭 | τ_q·3H_0 = 1 자기일관 조건 |
| 함의 | *미시* (Planck) 척도 흡수 | *우주론* (Hubble) 척도 흡수 |

### 3.3 어느 쪽이 맞는가?

| 검증 | base.md σ (Planck) | L48 σ_0 (Hubble) |
|---|---|---|
| 차원 [m³ kg⁻¹ s⁻¹] | ✓ | ✓ |
| 뉴턴 중력 도출 | ✓ (Lamb 매칭) | ❌ (스케일 너무 큼) |
| H_0 tension 해결 영역 | ❌ (σ 너무 작음) | ✓ (σ_0 ≈ 2.34e8) |
| 자기일관 τ_q·3H_0=1 | ❌ (τ = t_P) | ✓ |
| SPARC β = a_0·τ_q/c | ❌ (β >> 1, 비물리) | ✓ (β=0.035) |

→ base.md σ와 L48 σ_0는 *논리적으로 양립 불가능*. 이것이 이론 내부 가장 큰 미해결 문제.

### 3.4 다른 차원·자기일관 점검

**(A) n_0·μ = ρ_P/(4π) (base.md §3.3)** — *Planck-σ 사용 시* 자기일관. *Hubble-σ_0 사용 시*에는 n_0·μ가 다른 값을 가짐 (도출 미수행).

**(B) ε의 정체** — ODE에서 ε는 자유 파라미터처럼 행동하지만, L48에서는 ε = 3H_0²Ω_Λc²/(8πG·n_∞)로 *Λ_eff와 일치시키도록* 사후 결정. 이는 ε가 *독립* 매개변수가 아니라 *Λ_eff에 묶인* 종속변수임을 의미. 그러나 base.md §VI는 ε를 GFT 응축체 모드 에너지(ε ~ ℏ/τ_q)로 *원리적* 도출 시도 — 두 도출이 일치하는지 미검증.

**(C) a_0 = β·c/τ_q에서 β** — β ≈ 0.035 (L49 SPARC). 이것은 *피팅 결과*이지 도출이 아님. 이론적 β의 기원이 미정. (base.md는 a_0 ~ c·H_0으로만 기술 — 계수 정확도 ~30% 부정확.)

**(D) Γ_0 = 3H·n_∞** — 정상상태 도출. 자기일관 조건. 수치 도출 가능: n_∞ = Γ_0/(3H_0) → Γ_0를 자유 파라미터로 둘 때 n_∞도 자유. base.md §6.2의 Γ_0 = 5λ_5|σ_0|^4 GFT 도출은 미결정 매개변수에 의존.

### 3.7 8인팀 의견

- **P1(이론)**: τ_q 이중 정의는 본 이론의 *원조 분기*에서 발생. base.md는 미시(Planck) 응축체 모델, L48은 거시(Hubble) 자기일관 모델. 둘 다 일관성이 있지만 *서로 다른 이론*. 통합 시도 시 어느 쪽이든 약화.
- **P2(수리)**: Lamb 1932 포텐셜 흐름 매칭은 *비압축성* 가정 하 결과. 대사적 흐름은 *생성-소멸* 항이 있으므로 비압축성 위반. 이는 base.md §3.3의 도출이 *근사*임을 의미.
- **P3(실험)**: 어느 σ를 채택해도 L48~L54 결과는 동일하게 적용 (σ_0가 자유 파라미터로 들어가므로). 따라서 S3-1은 이론 내부 일관성 문제이고, 관측 비교는 별도.
- **P4(우주론)**: τ_q = 1/(3H_0)은 *현재 우주*에서 자기일관. 그러나 과거 우주(z→∞)에서는 H(z)→∞ → τ_q→0. 이는 σ_0 = 4πG·τ_q가 *시간 의존*이 됨을 의미 → σ_0(z) 패치(L54)가 *공리 외부*가 아니라 *τ_q 시간 의존*에서 자동 도출될 가능성. 이는 새로운 도출 경로.
- **P5(응집)**: μ = m_P 자연 선택 강력 후보. n_0 = ρ_P/(4π·m_P) ≈ 1.9e103 m⁻³. 이는 GFT BEC 응축체의 *기준 밀도*와 일치 가능성.
- **C1(강력 비판)**: τ_q 이중 정의는 *논문 제출 즉시 reject*급 모순. 8인팀 합의 도달 전 외부 발표 금지.
- **C2(강력 비판)**: σ의 60 dex 충돌은 단순 단위 혼동이 아닌 *구조적 모순*. base.md와 L48 사이 어떤 다리도 부재.
- **C3(강력 비판)**: τ_q = 1/(3H_0)은 *오늘만* 성립. 과거-미래에서 τ_q가 어떻게 변하는가에 대한 공리 부재. 시간 의존이면 G도 시간 의존 → Ġ/G ≠ 0 → LLR 위반 가능. 정량 검증 필요.
- **C4(강력 비판)**: "0개 자유 파라미터" 주장은 *현재 이론에서 거짓*. ε, β, h_1, σ_0(k 함수 형태), σ_0(z 함수 형태) 등 최소 5개 자유도 잠재. AICc 패널티 5+ 인정 필요.
