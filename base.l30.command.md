## 공리 (base.md II, VII 추가)

### 핵심 공리 (A1-A2, 기존)

- **A1**: 물질이 시공간 양자를 소멸, 빈 공간이 생성.
- **A2**: A1로부터 양자-고전 경계가 유도됨.

### 생성-소멸 비대칭 구조 (base.md II)

- **A3**: 생성은 공간적으로 균일(위치 무관). 소멸은 물질 밀도에만 국소적으로 의존. 이 비대칭은 드시터 대칭, 열역학 제2법칙, 양자장론, 일반 공변성 네 원리가 동시에 요구하는 필연적 구조.
- **A4**: 순률(생성 − 소멸)의 부호가 세 영역을 결정한다: 순 생성(빈 공간 → 팽창/암흑에너지), 순 소멸(물질 근처 → 수축/중력), 균형(경계면).

### 물리 법칙 일관성 조건 (base.md VII)

L30의 모든 이론은 아래 조건들을 위반해서는 안 된다.

- **C1** 등가 원리: 관성질량 = 중력질량. 동일한 소멸 계수에서 양쪽 모두 도출되어야 함.
- **C2** 운동량 보존: 비등방 유입의 순 운동량 = 중력력.
- **C3** 각운동량 보존: 회전 질량체의 비등방 소멸 → Lense-Thirring 효과.

> **적용**: 이론이 C1-C3 중 하나를 명백히 위반하면 물리적 후보에서 제외.
> **제거됨 (2026-04-13)**: 구 C3 홀로그래픽 원리 — A1-A4 부피 역학과 도메인 이질.
> **제거됨 (2026-04-13)**: 구 C2 CPT 대칭 — 물질/반물질 소멸 동등성은 ALPHA-g 실험 일치지만, A1에서 필연적으로 요구되는 것이 아닌 외부 실험 제약. 이론 도출 공간 확대를 위해 제거.

## AICc 기준선 (전체 고정)

```
ΛCDM best-fit: chi²=10.192, AICc=15.392 (k=2, n=13)
AICc(k) = chi² + 2k + 2k(k+1)/(n−k−1),  n=13
```

k=2 이론이 이기려면: chi² < 10.192
k=3 이론이 이기려면: chi² < 6.73

→ k=2 이론 (자유: Ω_m, H₀) 최우선.
   추가 상수는 반드시 이론에서 유도. 수치 피팅 시 k에 산입.
