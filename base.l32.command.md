## ■ 왜 이 실험을 하는가?

2024년 DESI(암흑에너지 탐사 실험)의 BAO(바리온 음향 진동) 관측 결과가 표준 우주론 ΛCDM의 예측과 2~3σ 긴장을 보임.
→ 암흑에너지가 단순 상수(Λ)가 아닐 가능성.

질문: "시공간 양자 이론(SQT)이 이 데이터를 ΛCDM보다 더 잘 설명하는가?"

---

## ■ SQT 이론 구조

### 공리 2개

- **A1**: 물질은 시공간 양자(SQ)를 소멸시킨다. 소멸된 SQ는 결합 에너지 형태로 전환.
- **A2**: 빈 공간에서 SQ가 생성된다. 생성 = 공간 팽창의 미시적 원인.

### 제약 3개

- **C1 (로렌츠 불변성)**: 소멸은 T^μ_μ에 비례. 복사는 T^μ_μ=0이므로 자동 탈결합. 물질만 소멸 주체.
- **C2 (공변성)**: 생성은 좌표계 무선호, 공변 부피당 평균률.
- **C3 (에너지-운동량 보존)**: 총 T^μν_total 국소 보존. 물질·SQ·상호작용 에너지 교환 허용, 총량 닫힘.

### 장 변수 정의

ψ(x) = |Φ_cl(x)|²/ρ_max (고전 안장점, 무차원)
- ψ=1: 평형 진공
- ψ=0 접근: 블랙홀 코어 극한 (도달 불가)

### 동역학

- 소멸률 = σ · ρ_matter · ρ_SQ
- 생성률 = κ · (ρ_max - ρ_SQ)
- 평형점: ρ_SQ* = κ·ρ_max / (κ + σ·ρ_matter)
- ψ*(z) = 1 / (1 + α·(1+z)³),  α = Ω_m / Ω_DE (z=0 유도, 피팅 매개변수 아님)

### B' 시나리오

- Hubble 팽창 = 생성률이 소멸률을 초과하는 양
- 물질 근처: 소멸 > 생성 → 중력 (수축)
- 빈 공간: 생성 > 소멸 → 암흑에너지 (팽창)

## ■ L32이 테스트하는 것

SQT 공리에서 암흑에너지 밀도 ρ_DE(z)를 여러 방식으로 유도하여 총 30개 이론 생성.

### 모든 이론의 공통 구조

E²(z) = Ω_r·(1+z)⁴ + Ω_m·(1+z)³ + ρ_DE(z)

### ρ_DE(z) 유도 방식에 따라 4개 그룹

#### FA (8개) — 소멸 진폭 변형

ρ_DE = Ω_Λ0 · (1 + f · Γ_norm(z))

f는 물리적 근거로 고정 (피팅 금지):

| f 값 | 물리 근거 |
|------|-----------|
| 0.5 | 단순 반비례 |
| 1/3 | 복사 상태방정식 w=1/3 대응 |
| 1/π | 구면 입체각 기하 |
| 1/e | 지수 감쇠 자연 스케일 |
| Ω_m | 물질 분율 자체결합 |
| Ω_Λ0 | 암흑E 분율 자체결합 |
| Ω_m/Ω_Λ0 | 비율 결합 |
| 1-Ω_m | 잔여 분율 결합 |

#### GB (8개) — 생성-균형 시나리오

ψ 평형 구조 변형:
- sqrt(ψ): 장 진폭 관점
- log(ψ): 격자 기체 엔트로피 구조
- B' 직접: H = (생성-소멸)/3 명시 적용
- 복사 보정: 초기 우주 복사기 연속성 확보

#### VG (8개) — V(ψ) 긴즈버그-란다우 퍼텐셜

V(ψ) = (1/2)μ²·(ψ-1)² + (γ/6)·(ψ-1)³

μ², γ는 이론 내부 자연값으로 고정 (K91 회피).

#### MC (6개) — 다채널 결합

리치(소멸) + 바일(재배열) 이중 채널 + 여러 물리 스케일 중첩.

## ■ 승패 판정 기준 (완전 사전 고정)

### 데이터

- DESI DR2 BAO: 13개 측정 포인트
- 공분산: 13×13 full 행렬 (대각 사용 금지 → K92)

### 자유 파라미터

- r_s = 147.09 Mpc 고정 (Planck 2018)
- 추가 상수는 이론 유도값만 허용

### 통계량

AICc = χ² + 2k + 2k·(k+1)/(n-k-1),  n=13, k=2

### 기준선 (불변)

ΛCDM best-fit: χ²=10.192, AICc=15.392

### 판정 등급

| 등급 | 조건 | 의미 |
|------|------|------|
| Q90 PASS | ΔAICc < 0 | ΛCDM보다 우수 |
| Q91 STRONG PASS | ΔAICc < −2 | 통계적 유의 우세 |
| Q92 GAME-CHANGER | ΔAICc < −4 AND wa < −0.5 | 이론 큰 진전 |

## ■ 암흑에너지 상태방정식 추출

각 이론 피팅 후:

w(z) = -1 - (1/3)·d ln ρ_DE / d ln(1+z)

CPL 매개화: w(z) ≈ w₀ + wa·z/(1+z)

z ∈ [0, 2] 최소자승으로 w₀, wa 산출.
K93 판정은 이 wa 값 기준.

---

## ■ 최적화 방법

1. multi-start Nelder-Mead (20개 시작점)
2. differential_evolution (popsize=30, maxiter=1000)
3. 두 방법 중 더 낮은 χ² 채택

수렴 체크:
- 두 방법 결과 Δχ² < 0.01 → 전역 최소 확신
- Δχ² ≥ 0.01 → 추가 multi-start 50회 재확인
