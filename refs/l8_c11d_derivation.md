## 배경

C11D: Pure disformal quintessence (A'=0), CLW exponential potential.

CLW 자율계 (Copeland-Liddle-Wands 1998):
```
x' = -3x + (√6/2) λ y² + (3/2) x [2x² + (1−w_m)(1−x²−y²)]
y' = -(√6/2) λ x y + (3/2) y [2x² + (1−w_m)(1−x²−y²)]
```
x = φ̇/(√6 H M_P), y = √(V/(3H² M_P²)), ' = d/d ln a
λ = 0.8872 (L5 posterior mean)

SQMH 균일 배경:
```
dn̄/dt + 3H n̄ = Γ₀ − σ n̄ ρ_m        ... (SQMH-BG)
```

## 8인 병렬 상호토의

8인팀이 각자 독립된 각도(수학/물리/수치/스케일/위상/변환/반례/통합 접근)에서 동시에 공략을 시작하고, 발견 즉시 공유하며 상호토의 후 합의에 도달했다.

---

### [물리 접근] 변수 대응 후보 탐색

후보 3 (핵심): n̄ ∝ exp(−λφ/M_P)
```
d(exp(-λφ/M_P))/dt = -√6 λ x H · exp(-λφ/M_P)
→ dn̄/dt = -√6 λ x H n̄
```
SQMH-BG 대입:
```
(-√6 λ x H + 3H) n̄ = Γ₀ − σ n̄ ρ_m
→ σ ρ_m = H(3 − √6 λ x) (조건)
```
**발견**: σ ρ_m = H(3 − √6 λ x) 조건이 성립하면 SQMH-BG와 동형.

---

### [수학 접근] 동형 조건 엄밀화

σ_eff 역산 공식:
```
σ_eff(a) = [Γ₀/n̄(a) + (√6λx(a) − 3)H(a)] / ρ_{m,0} a⁻³
```
Q32 판정 기준:
- σ_eff(a) ≈ const (z 독립) AND
- σ_eff ≈ 4πG t_P (10% 이내)

### [스케일 접근] 결정적 발견 — 61차수 갭

스케일 접근 팀이 즉시 공유한 핵심 발견:

CLW 역학적 스케일: σ_need = H₀/ρ_{m,0} ~ **8.23×10⁸ m³/(kg·s)**
SQMH 플랑크 스케일: σ_SQMH = 4.52×10⁻⁵³ m³/(kg·s)

**갭 = 61차수** → 변수 치환으로 극복 불가능.

**긴급 상호토의**: 전체 팀 공유. 수학/물리 팀이 후보 모두 검토해도 스케일 갭은 근본적임을 확인.

---

### [수치 접근] CLW 궤도에서 σ_eff 계산

thawing 특성 (λ = 0.8872 < √3): x ≈ 0, y 지배.

x ≈ 0 (thawing) 한계:
```
σ_eff ≈ [Γ₀/n̄ − 3H] / ρ_m
```
3H 항이 지배 → **σ_eff < 0 전체 구간**.

후보 n̄ ∝ exp(+λφ) 부호 반전 시도:
→ σ_eff < 0 구간 18% 양수로 개선되나 여전히 K32.

### [변환 접근] 단위 및 차원 분석

- M_P = √(ħc/G) = 2.176×10⁻⁸ kg
- σ_SQMH = 4πG t_P = 4.52×10⁻⁵³ m³ kg⁻¹ s⁻¹
- σ_need = H₀/ρ_{m,0}: Hubble-matter 스케일 (우주론 배경)

차원은 동일(m³kg⁻¹s⁻¹)하나, 플랑크 스케일(G·t_P)과 우주론 스케일(H₀/ρ_m)의 갭 61차수는 단위 변환과 무관.

### [위상 접근] attractor 궤도 분석

λ = 0.8872 < √3: 트래커 없음. thawing quintessence (φ 오늘 막 구르기 시작).
attractor 위에서 x/y = const이면 σ_eff 상수화 가능성 있으나,
thawing 궤도에서 x ≈ 0 → σ_eff 부호 문제 해소 불가.

### [반례 접근] 동형 한계 비판

비판 1: "σ_eff(a) 상수 아님 → 동형 아님" — σ의 상수성은 SQMH 공리.
비판 2: "thawing에서 σ_eff < 0 → SQMH 부호 위반" — 소멸이 아닌 생성이므로 물리 부정합.
비판 3: "A'=0 pure disformal은 배경에서 minimal coupling(ZKB 2013) — disformal 역할 없음" — 역유도는 CLW ODE 자체와 SQMH의 동형 문제.

**토의 결론**: 어떤 비판도 반박 불가. K32 확정.
