# refs/l8_c28_derivation.md — L8-R: C28 역유도 (8인 병렬 상호토의)

> 작성일: 2026-04-11.
> **방법**: 서로 중복되지 않은 8인팀, 수단과 방법을 가리지 않고 동시에 병렬 및 상호토의를 통해 유도. 결과 취합 후 최종 판정.
> 목표: Maggiore-Mancarella RR 보조장 ↔ SQMH 연속방정식 동형 탐색. Q33 판정.
> C28는 독립 이론. 성공 기준 수준 2 (구조 동형).

---

## 배경

C28: Maggiore-Mancarella RR 비국소 중력 (Phys.Rev.D 88, 021302, 2013)

작용:
```
S = M_P²/2 ∫ d⁴x √-g [R − m²/6 · R □⁻² R]
```

보조장 도입:
```
U = □⁻¹ R  →  □U = R       ... (U-eq)
V = □⁻¹ U  →  □V = U       ... (V-eq)
```

FRW 균일 우주에서:
```
Ü + 3HU̇ = R = −6(Ḣ + 2H²)     ... (U-bg)
V̈ + 3HV̇ = U                    ... (V-bg)
```
γ₀ = 0.0015 (L5 posterior mean), m² = γ₀ H₀²

---

## 8인 병렬 상호토의

8인팀이 각자 독립된 각도에서 동시에 공략을 시작하고, 발견 즉시 공유하며 상호토의 후 합의에 도달했다.

---

### [구조 접근] V-eq과 SQMH-BG 비교

P ≡ V̇ 로 정의하면 V-eq를 1차 ODE로 환원:
```
Ṗ + 3HP = U        (P-eq)
```
SQMH-BG:
```
dn̄/dt + 3H n̄ = Γ₀ − σ n̄ ρ_m
```
**구조 동형 발견**: P ↔ n̄, U ↔ (Γ₀ − σ P ρ_m).

dP/dlna + (3 + Ḣ/H²)P = U 형태. SQMH 연속방정식과 **동일한 미분 구조**.

---

### [수학 접근] U source의 동형 조건

완전 동형 요구: U(t) = Γ₀ − σ P(t) ρ_m(t)

U 방정식: □U = R = −6(Ḣ + 2H²) → U는 R의 인과적 적분, a에 의존.

약화된 Q33 기준:
- U(a) = A + B × P(a) × ρ_m(a) 피팅 잔차 < 20%?

완전 동형은 U가 상수(Γ₀)가 아닌 동적 필드이므로 불가. 수치 피팅으로 판정.

---

### [수치 접근] U(a)=−12.41, OmDE_RR < 0 발견

단순화 ODE (Dirian 2015 full 방정식 없이) 수치 적분 결과:

| 항목 | 값 |
|------|----|
| U(a=1) | **−12.41** (음수) |
| E²_RR(a=1) | **0.31** (OmDE_RR < 0) |

**발견**: U(a=1) = −12.41 음수. 소스항이 n̄을 생성이 아닌 흡수 → SQMH 물리와 반대 부호.

OmDE_RR < 0: 단순화 ODE에서 정상 에너지 밀도 재현 불가. Dirian 2015 완전 방정식(U·V 교차항) 없이는 해결 안 됨.

---

### [스케일 접근] σ_eff 피팅 잔차

U(a)를 (Γ₀_eff − σ_eff P(a) ρ_m(a)) 형태로 피팅:

→ 잔차 = **100%** (피팅 의미 없음)

U(a)와 (Γ₀ − σ P ρ_m)이 동일 스케일이 아님. 에너지 스케일 불일치.

---

### [변환 접근] 비국소성과 동형의 의미

RR 비국소 메모리 커널(□⁻¹)과 SQMH 소비 누적 효과의 개념적 대응 탐색:
- V̇ ↔ n̄ 대응, U ↔ source: 구조적 유사성은 흥미로움.
- 단, 소스 U가 시간 의존적이고 음수 → 물리적 대응 파탄.

Nersisyan+2016, Belgacem+2017: "RR and quintessence are phenomenologically similar but structurally different." 기존 문헌도 한계 인정.

---

### [반례 접근] C28-SQMH 동형의 비판

비판 1: "RR은 수정 중력, SQMH는 DE 유체 — 프레임 자체 다름."
→ 호너 등가 원리로 연결 가능하나 U < 0이 치명적.

비판 2: "U가 상수 Γ₀가 아닌 동적 필드 — 완전 동형 불가."
→ 약화된 기준(20% 잔차)도 실제로는 100% 잔차로 불통과.

비판 3: "L6-T3 합의: C28은 독립 이론. SQMH와 현상론 일치만 주장 가능."
→ 구조 동형 탐색은 허용이나 에너지 스케일 불일치로 Q33 달성 불가.

---

### [위상 접근] dP/dlna 구조 동형성 확인

```
dP/dlna + (3 + Ḣ/H²)P = U
```
이 구조는 SQMH 연속방정식과 **정확히 동일한 미분 구조**. 위상적 동형성은 성립.

그러나 소스항 U(a=1) = −12.41 (음수), OmDE_RR < 0 → 물리적 내용에서 파탄.

---

### [통합 접근] 최종 수치 결과 및 판정

| 항목 | 값 |
|------|----|
| U(a=1) | **−12.41** |
| OmDE_RR (단순화 ODE) | **< 0** |
| dP/dlna + (3+Ḣ/H²)P = U 구조 동형 | 성립 (수학적) |
| σ_eff 피팅 잔차 | **100%** |
| Q33 기준 (잔차 < 20%) | **FAIL** |

## 최종 합의

8인 병렬 토의 후 합의:

| 항목 | 결과 |
|------|------|
| U(a=1) | **−12.41** (음수, SQMH 반대 부호) |
| OmDE_RR | **< 0** (단순화 ODE 부족) |
| 구조 동형 (미분 구조) | 성립 — 단, 에너지 스케일 불일치 |
| 잔차 | **100%** |
| Q33 | **FAIL** |
| K33 | **TRIGGERED** |

**판정**: RR-SQMH 구조 동형은 수학적으로 흥미롭지만 에너지 스케일이 일치하지 않음. 단순화 ODE 부족으로 OmDE < 0. Q33 FAIL, K33 TRIGGERED. C28은 독립 이론 유지 (§8 "동형 탐색 실패" 기재, 구조 동형성은 각주 수준 언급 가능).
