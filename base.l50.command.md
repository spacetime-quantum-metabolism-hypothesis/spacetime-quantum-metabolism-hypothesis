
## ODE 시스템 (L48 동일, σ_0→σ_IR)

```
dn/da    = [Γ_0 - 3H·n - σ_IR·n·ρ_m] / (a·H)
dρ_m/da  = [-3H·ρ_m + σ_IR·n·ρ_m·ε/c²] / (a·H)
H²       = (8πG/3)·(ρ_m + n·ε/c²)
```

H는 ODE 변수 아님 — 매 단계 직접 계산 (P4 지적 반영).

## 도출 관계식

```
τ_q   = σ_IR / (4π·G_N)
n_∞   = Γ_0 · τ_q
ε     = 3H_0²·Ω_Λ·c² / (8πG·n_∞)
자기일관: τ_q · 3H_0 = 1
```
