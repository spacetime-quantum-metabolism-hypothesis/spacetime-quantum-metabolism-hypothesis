## 모델

### σ_0(k) 함수 (L50과 동일)
```
σ_0(k) = (σ_IR + σ_UV·(k/k_t)²) / (1 + (k/k_t)²)
σ_UV = 3.31×10⁹ (L49 고정)
k_8 = 0.13 Mpc⁻¹ (σ_8 스케일)
```

### 구조 성장 ODE (a 미분)
```
δ'' + friction_factor·(3/a + H'/H)·δ' - (3Ω_m)/(2a⁵E²)·δ = 0
E² = Ω_m/a³ + Ω_Λ
H'/H = dH/da / H = -(3Ω_m)/(2a⁴E²)
```

### SQT 마찰 보정
```
σ_eff = σ_0(k=0.13)
friction_factor = 1.0 + α·log10(σ_eff/σ_ref)   [α=0.1, σ_ref=1.17×10⁸]
σ_eff > σ_ref → 마찰 강화 → σ_8 감소 (KiDS 방향)
σ_eff = σ_ref → friction_factor = 1 (ΛCDM 한계)
```

### σ_8 계산
```
D+(a), D_LCDM(a): 동일 초기조건에서 ODE 풀이
σ_8_SQT = σ_8_Planck × (D_SQT(a=1) / D_LCDM(a=1))
```
